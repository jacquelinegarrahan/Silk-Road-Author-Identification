Not sure I want to know what you intend to do with it...

Potassium cyanide is a basic salt, so it will react pretty strongly with an acid, although the byproduct is hydrogen cyanide, which is pretty bad for you (in very small amounts you'll get a pretty nasty headache) and extremely flamable.

to have absolute confirmation that it is in fact potassium cyanide is a pretty in-depth process which you don't likely have the equipment for.  You can buy a testing kit online, but they're kind of expensive
