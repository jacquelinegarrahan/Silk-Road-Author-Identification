Title: Re: Money Sent Hasn't Been Deposited to My Account Here
Post by: ConcernedCitizen on April 20, 2013, 04:24 pm
There is no such thing as a "deposit code" in bitcoin. You can only send bitcoins from one bitcoin address (looks like the number 1 followed by random characters) to another. There is built in error checking so it is very unlikely a mistyped address was accepted. If you copy/pasted the wrong address then unless the recipient sends it back, you are outta luck. As was stated above, bitcoin transactions take a while to get "confirmed," and the Silk Road only accepts transactions once they have received at least six confirmations, which takes an average of one hour. Sometimes the transaction can get stuck in the network for a while before it is confirmed, so give it 24h before you fret.
