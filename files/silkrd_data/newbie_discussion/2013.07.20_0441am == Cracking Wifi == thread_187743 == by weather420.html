Post by: Tessellated on July 20, 2013, 07:52 pm
I realize I was being a bit vauge. Backtrack 5 is a linux based operating system that you can boot on most computers from a USB stick. It has most of the popular hacker tools pre-installed and allows you to skip the tedious setup process of hundreds of little tools.

You can download a copy of this OS, flash it to a USB stick and boot your computer from it. From there you can hack wifi, bluetooth, crack passwords,  setup botnets and install rootkits. All in a handy package.

You still need to know how to use the tools, you just don't need to install all of them. There are countless tutorials for basic hacks that use backtrack as their environment. It is very nice to follow a tutorial and know that the same command will work the same way on your system.
