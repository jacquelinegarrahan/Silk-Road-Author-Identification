Post by: Yoda on September 21, 2013, 04:10 pm
Hit up the security forum, lots to read there. 

Your mac address is like a unique serial number on your network card.   So if you did some illegal shit, you obviously wouldn't want that number associated with said illegal shit.

VM... Virtual machine.  It allows you to run an OS within your OS... so like if you're using the OS in the VM to do crazy shit, and downloaded a virus, it would be contained within that VM OS(hopefully).  Which you can then toss out and reload a fresh one.
