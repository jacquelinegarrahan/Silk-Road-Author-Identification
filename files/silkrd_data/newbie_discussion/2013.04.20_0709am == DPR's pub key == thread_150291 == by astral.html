For the benefit of others, DPR's key is populated in clearnet keyservers.  Didn't find it before because the key's ID gives no indication that it's DPR's.  One way to find is to search for <staff@silkroadmarket.org>.  If you are confident that you're connected to a trustworthy key server and get only one hit from that search, you can be confident that's the real one (it has no expiry).

A well-respected keyserver that is kept up to date is http://www.stinkfoot.org:11371/ .  (Reference http://www.rossde.com/PGP/pgp_keyserv.html#pubserv ).
