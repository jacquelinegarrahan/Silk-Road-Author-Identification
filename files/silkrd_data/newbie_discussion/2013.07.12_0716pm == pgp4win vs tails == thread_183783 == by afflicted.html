Title: Re: pgp4win vs tails
Post by: kahn on July 12, 2013, 08:57 pm
Two entirely different animals..... 8)
PGP (either pgp4win or pgp4usb) are simply encryption programs. They are both adequate for SR. For a concise guide to using PGP go to the Wikipedia tutorial for Gpg4win. After struggling for a week I found this and it offered simple direct answers. Above all practice using it w/someone you can trust.
TAILS, on the other hand, is a complete system for stealth use of the TOR network. If you have concerns that someone else may see your use of a specific computer, or if you're using someones computer (ie. not your own computer, at work, at a commercial "web cafe", or your parents computer) definitely use TAILS. It is designed for use of TOR (or anything else) and to leave no trace.
BTW, once you've setup PGP, you should load the PUBLIC KEY you intend to use for SR communication on the USB stick you're using for TAILS so you'll have it if you decide to use TAILS.
There is a very useful tutorial for use of TAILS on USB @ http://dkn255hz262ypmii.onion/index.php?topic=114141.0

Hope this helps.... ;) ;) ;)
