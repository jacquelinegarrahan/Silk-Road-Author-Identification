Hi all,

I and the staff at Atlantis would take some time to address peoples concerns regarding the recent Atlantis phishing site. As this is a new account, we could only create the thread here.

We were notified about the site approximately 4 days ago and upon verification we updated our register/about/login pages to advise uses to verify the onion URL they are using before logging in. We sent out a message to all users advising them of the phishing site and to make sure they use our onion link (without using a third party provider like onion.to or onion.sh.). We also updated our auto-withdrawal code for vendors so that any changes require access require the users security code. All other functions of the site (purchasing items/withdrawing and transferring funds require it). Luckily, the phishing site only gathered passwords and so far we've only had one user report that their funds were stolen (since they used the same password for their security code).

In relation to the phishing site, the site wasn't run by us at all - it was a transparent proxy site which logged user passwords but shared the same characteristics as ours (server time etc). It works in the same way that onion.to and onion.sh services work. You can read about the initial accusations here: http://dkn255hz262ypmii.onion/index.php?topic=183922.0. If our sole purpose was to make money nefariously, we could just run off with all user funds and pull a pirateat40. Obviously this is not the case and never will be!

We genuinely care about our users, what we are trying to accomplish and our image. It would crush our reputation if we pulled such a stunt. Our goals are to build a community based on liberation, to promote cryptocurrency and to offer a true free market whilst making an aesthetic medium to do so.

Thanks,
The Atlantis Team.
