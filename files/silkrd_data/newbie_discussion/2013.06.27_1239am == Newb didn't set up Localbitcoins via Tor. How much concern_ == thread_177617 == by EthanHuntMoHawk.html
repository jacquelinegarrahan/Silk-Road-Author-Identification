Title: Re: Newb didn't set up Localbitcoins via Tor. How much concern?
Post by: GrimWaldo on June 27, 2013, 01:03 am
BitCoins are not illegal, so unless you're purchasing a lot of BTC it shouldn't be an issue. I purchase mine over ClearNet from a very above-ground source (straight from my bank account).

If you're in the USA, using Tor for everything (and not just DeepNet) is OK... but remember, your ISP knows that you're using Tor if you don't use Bridges/VPN. Use TAILS to be safe.

It's been mentioned that some online sites may log and record people that connect to their website via Tor (banks, USPS, .gov, etc), so you may want to use ClearNet for ClearNet things, and Tor for DeepNet things. That's what I do.
