Title: Re: how should i format my order
Post by: boaclon224 on August 05, 2013, 12:30 pm
Go to the USPS website (presuming it's a US order) and find the page for how addresses should be formatted. Some vendors link to it from their vendor pages. Then just put your address in the address box (encrypted, obviously); leave out the "please send to:" and the product, so the vendor can just put the address straight onto a label with no hassle.
