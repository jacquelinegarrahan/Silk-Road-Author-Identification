I don't use a mac, but the general gist of most gpg programs is
1.  Write an unencrypted message to encrypt
2.  Import the recipients public key into your gpg program
3.  Copy the unencrypted message into the gpg program
4.  (optional)  Add your own public key to the end of the message, so the recipient can message back securely.
5.  Encrypt the message, select the recipient's key to encrypt with
6.  Copy the -----BEGIN PGP MESSAGE-----  text into your email/whatever client and beam it off
My apologies for the vagueness
