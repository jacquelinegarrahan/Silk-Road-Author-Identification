Title: Re: GPG Keychain Access Question
Post by: flatlineconstruct on September 21, 2013, 04:26 am
This is easy. Just highlight the key you want to get and choose File>Export... and choose ASCII as the format. You will get a file that is (keyid).asc. The contents of that file will be text version of the key.

Be aware that you don't need to send your key to a vendor unless you are exchanging info. If you're just buying, you only need to import the vendor's key and use that to encrypt your address/message.
