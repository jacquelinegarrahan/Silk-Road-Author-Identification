I have put together some applications that you can use (with simple instructions included in each zip file), to connect and communicate with each other via IRC.
Please read the readme files, they are there for a reason.
Also these applications have been designed to be portable. So create a truecrypt volume and extract the zip into it or use USB flash based media (micro sd cards are easy to hide).
I have only tested these on 10.8 ML and windows 7.
If there is a problem I'll get right on it and re-release.
For the newbies, the signatures are for the zip files, so that you can be sure that they haven't been tampered with en-route to you.
Last thing, its a chat. Please be polite and nice. If you are feeling generous donate btc to SR user "thepot" and it will go towards server funds (or so I'm told).
