Quote from: ManMist on September 21, 2013, 09:14 pm
How do I open an encrypted message on my clipboard when I have the public key to open it in my Key Manager (of GNU Privacy Assistant)

Typically if someone has sent you an encrypted message, it is encrypted with your public key, and you would use your secret key to decrypt it.  If you can't find the tool to perform this operation on the clipboard, you can paste the contents into a plain text file. Then select it in the main window of GPA and you should be able to decrypt it.
