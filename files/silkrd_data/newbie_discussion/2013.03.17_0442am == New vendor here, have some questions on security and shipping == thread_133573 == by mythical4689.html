Title: Re: New vendor here, have some questions on security and shipping
Post by: QuiteArbitrary on March 17, 2013, 05:15 am
If you're doing pills, vacuum sealing and then placing that package in an xray proof baggie (google them) is pretty standard procedure. Concealing the pills in a product box is not necessary for retail distribution. A letter containing a few blank pieces of paper and the goods will do. As far as the return address goes, post a real one as a fake one is a red flag in the system. Do not use a return address that can ever be linked back to you. I suggest making a few orders from a separate buying account from the other vendors in the market to see how its done firsthand.
