Grats :) Once you get the hang of it you will realize how easy it really is :P

- Download & install gpg/pgp. Don't forget to set up the shortcuts in your keyboard
- Generate your own private key
- Save vendor public key in a text file and import it
- Write a message in your text editor of choice
- Encrypt it with earlier imported vendor key
- Profit.

Good luck to the rest ;)
