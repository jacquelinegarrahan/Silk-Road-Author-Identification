I think it should very on your location... I can only speak for the US. Living in a large city and an apartment building with cluster-mailboxs I see no issue using a fake name as the post-person doesnt know anyone on a personal level, this might only become an issue if its a larger package and you arent around upon delivery and then have to go pick it up at the post-office.
