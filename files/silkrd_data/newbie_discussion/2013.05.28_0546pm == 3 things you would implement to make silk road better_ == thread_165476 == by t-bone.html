Title: Re: 3 things you would implement to make silk road better?
Post by: BlackIris on May 28, 2013, 06:56 pm
1. I would change the domestic filter a little (or make another similar option). Let's face it: if you are in EU, with the only exception of the UK maybe, you don't want to list only your country's vendors (especially for those countries where there are almost surely no vendors), but all those inside the EU (since it would work mostly as domestic for the USA in this way).

2. Probably a revised feedback system. The 1-5 is not working so well, because anyway anything below 5/5 is the same as negative. At this point it would be probably better something similar to BMR with positive, neutral and negative only.

3. Possibility to check money value of BTC also when you have BTC as default and possibility of checking the value also in EU and not only dollars (depending on your country, naturally).
