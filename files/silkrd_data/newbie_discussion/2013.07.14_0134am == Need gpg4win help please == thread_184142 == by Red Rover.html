Title: Re: Need gpg4win help please
Post by: Croskin on July 14, 2013, 02:33 am
If you've already downloaded gpg4win then go into that folder and open up GPA.  Look up a tutorial if you need one, but it is pretty easy to use.  Make a new notepad file, put a vendor's key in it, save it, click import in GPA, choose the file, open the clipboard, type your message, click encrypt, choose the vendor key that you just imported.  Good luck :)
