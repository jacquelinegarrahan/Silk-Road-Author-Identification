Title: Re: Isn't Tor enough?
Post by: ilovelucyshow on April 14, 2013, 06:42 am
To my knowledge, spying on traffic within the TOR network itself is not possible. My understanding is that the primary vulnerability of the TOR network is at the so called "exit nodes". The node operator could sniff unencrypted packets at the exit point. This is where HTTPS comes in. If the connection is via HTTPS, the packets are still encrypted after leaving the exit point.

Assuming a valid and properly configured https connection, I think it's "secure enough".

Are there specific threads addressing other vulnerabilities? If so, perhaps you could share the links here.
