Title: Re: How did I do?
Post by: proxanne on May 23, 2013, 04:28 am
Simply, PGP is an encryption protocol.  It is useful for securing sensitive information you send a vendor (i.e. shipping information)
You need a program to make an encryption.  I use pgp4usb.  Its a portable program so it requires no installation.  I don't even have it on my hard drive (I run it from a usb).

The program includes a text editor and the ability to encrypt that text.

When you run the program, you will set up two keys.  One private key, and one public.  Share your public key with people you want to send encrypted messages to.  Never share your private key.  That's all client side.
Likewise, to receive encrypted messages you need the sender's public key.

Do you follow?
