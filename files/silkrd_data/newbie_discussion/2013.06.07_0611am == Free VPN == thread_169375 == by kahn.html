If you want to use a free, open source, easy to set up VPN for all your net traffic, (http://softether.org/ ). Incredible number of volunteers using their systems to facilitate a free world wide VPN. This is a bunch of Japanese students that have set this up. I don't know when or if they will start trying to charge for it. Once you've got the software DL'd, what can they do? I expect they will, at some point, charge for the service but if the software is still in use and the volunteers continue to act as servers, all they can do is discontinue support.

DL the VPN Gate ( http://www.vpngate.net) to access an easily usable list of volunteer IP addresses. The number of available volunteer IP addresses, world wide, is astounding. No problem using w/Tails or TOR. You're traffic will look like it's coming from France, Japan, India, Slovenia, etc, etc, etc. Korea doesn't like you to look at porn...

Just FYI. If you're paying for a "private" VPN, read the fine print. Do your own research.
