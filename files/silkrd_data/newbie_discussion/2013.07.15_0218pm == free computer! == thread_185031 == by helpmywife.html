Hey mate, almost got the right idea. Tails is a complete operating system, no need to install linux prior to tails. Burn the iso file to a DVD and then run tails from that disc. Make sure your booting option is set to boot from DVD rom. Once in tails, insert your USB and select "Tails USB installer" in the "System Tools" menu. After it's cloned, restart your computer, set your boot sequence to USB, take out the CD and you're done!
