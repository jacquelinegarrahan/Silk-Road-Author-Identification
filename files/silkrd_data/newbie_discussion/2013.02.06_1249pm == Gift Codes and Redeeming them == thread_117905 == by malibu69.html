Hi guys

Thankyou for the advice.  I have since found out how to use a gift code.

You must place an order with the vendor either custom order or of the vendors page and then utilize the gift code as a
form of payment.

I originally thought you could purchase a gift code and pass the details to the vendor so the vendor could redeem the value of the
gift code, this is not the case.  It must be used in the purchase of an item.

Hopefully this will help others as it is not on the FAQ page of SR or information supplied to buyers.

Cheers everyone and thanks again.
