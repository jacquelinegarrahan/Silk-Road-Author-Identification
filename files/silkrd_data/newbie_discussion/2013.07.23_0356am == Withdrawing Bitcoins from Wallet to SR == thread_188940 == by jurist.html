Title: Re: Withdrawing Bitcoins from Wallet to SR
Post by: SoAlone on July 23, 2013, 04:54 am
Yes your SR Wallet address is simply that, an address.

You can deposit or withdraw using that same number.

Also if you get a new wallet address#, which the SR program allows you to do, the old address still will work. I've heard that it's good to use a new one each time, as it is only a matter of time before blockchain gets subpoenaed. :o

Look into tumbling your bitcoins, as an added layer of security.
