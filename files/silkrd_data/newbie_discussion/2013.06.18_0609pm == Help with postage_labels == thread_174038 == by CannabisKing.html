Post by: mcguire39 on June 18, 2013, 06:15 pm
Hi, I'm not an expert and not a seller, but some things I've seen in the forums say right to always use a printed/typed address label (not handwritten) and to make sure it is neatly and squarely applied to the box (not crooked). There was a thread that said to be careful what printer you use, since some (mainly color) printers spit out an identifiable code in light yellow. So the printer should have been purchased anonymously not near where you actually are.

For postage, I've heard of people using stamps for packages light enough to drop in a blue box. Someone mentioned buying prepaid priority mail envelopes and boxes, but I don't know that I've ever seen those. The 400 label seems to be a popular tracking choice.

Someone mentioned using open wifi hotspots to do the priority mail shipping right from usps dot com using prepaid visa cards I think they said. Of course whatever method is used it must be top notch stealth.
