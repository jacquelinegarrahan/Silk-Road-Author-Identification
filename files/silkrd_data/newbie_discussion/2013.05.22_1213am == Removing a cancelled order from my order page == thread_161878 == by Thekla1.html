Hi Thekla1,

The order was cancelled after the item was put 'in transit', so you are still able to leave feedback for it. If everything was resolved satisfactorily I would advise leaving a 5/5 along with a short one or two line description of what occurred. If you do nothing, the ability to edit the feedback and the order itself will be removed from your feedback page after 3 months.
