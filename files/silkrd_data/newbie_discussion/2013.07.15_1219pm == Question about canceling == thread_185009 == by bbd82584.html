Title: Re: Question about canceling
Post by: mcguire39 on July 15, 2013, 02:40 pm
The vendor can cancel the order. The vendor is under no obligation to accept any order. But if the vendor doesn't cancel the order I think it takes four days before the buyer cancel order button is available. I think that's to give the vendor enough time to mark it in transit if s/he accepts the order since SR does not have any state between 'processing' and 'in transit'. I noticed Atl. does have a state between those two so that you can cancel the order at any time before the vendor accepts the order (though it does ask you to enter a reason for cancelling, probably out of courtesy to the vendor).
