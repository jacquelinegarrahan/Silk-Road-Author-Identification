Title: SimpleGPGEncrypt, a simple frontend for gpgtools on MacOS X
Post by: ximp on February 14, 2013, 02:07 am
Hi,

SimpleGPGEncrypt is a simple frontend for gpgtools on MacOS X

(clearnet) http://code.google.com/p/simplegpgencrypt/downloads/

It only provides the most basic functionality of encrypting a cleartext messages with a public GPG key, but is much easier to use than having to remember gpg command line.

Just enter your address, copy the vendor's public GPG key, press encrypt, and you're done!

Note that the imported public key is automatically removed from gpg so it leaves no traces.
