Title: Re: 2 quick questions about Tails
Post by: Treasurer on July 29, 2013, 12:23 am
On the top right of the screen when using TAILS you should see an option to select "OpenPGP Encryption Applet." If you open that and select "manage keys" the program opens and shows all your keys, under the "My personal keys" or "other keys" tabs.

To import a new key, click "file" then "import." Select the key and import it. If you don't have a file, but just the text of the public key, then make the file. Just make a new text file and paste the text in, then save it as a .asc file. Then import. Should work fine.
