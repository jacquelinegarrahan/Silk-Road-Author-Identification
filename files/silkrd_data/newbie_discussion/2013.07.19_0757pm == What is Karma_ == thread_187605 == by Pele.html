Title: Re: What is Karma?
Post by: PinnacleGoods on July 19, 2013, 08:05 pm
From the link below (clearnet warning!)
http://wiki.simplemachines.org/smf/Features_and_Options#Karma

Karma is a feature that shows the popularity of a certain member. To make it available, the administrator must activate it in core features. If members are allowed, they can choose to raise (Applaud) or lower (Smite) a person's karma. This is how a member's popularity is calculated.

Admins can set all kinds of different values and labels to use with Karma, but generally speaking it's just a way to show who's popular/appreciated.
