It's tough to be patient but that's how it is. Assuming you picked a good vendor and the vendor's page doesn't say s/he is on vacation etc you should be fine. So right build yourself up a bit of a stash so that you're not needing to be overly worried about a few extra days processing. I've had orders go into shipped within a couple hours, sometimes it takes a few days. I think some vendors even if they actually ship it the day you ordered it wait until the next day to mark it in transit.
