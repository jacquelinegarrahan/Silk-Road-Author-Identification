Title: Anoncoin [anonymous crypto currency]
Post by: notbeingseen on October 01, 2013, 07:23 pm
Hi. I thought you guys might be interested in this project. It's still in early development, but aims to be a completely anonymous currency through i2p, TOR, zerocoin, and possibly coinjoin as well. The main dev took part in creating i2p (invisible internet protocol), and they are in constant talks with the guys behind zerocoin to help implement it.

They will offer several anonymous services, among them web hosting and an anonymous exchange (I2P & Tor). Their latest project is an anonymous cloud storage service (currently in alpha testing), payable by ANC, BTC, and LTC. But ANC buyers get bonuses. You can read more about their planned services on their website (the cloud service isn't mentioned there yet, but talked about in their bitcointalk thread and on IRC).

ANC is mostly traded on cryptsy, although it's not mentioned as an exchange in the first post on bitcointalk; it is where you'll find it.

If you want to you can run Anoncoin only via TOR as well:
Quote
1. Install TOR https://www.torproject.org/download/download-easy.html.en

2. Put this in your config file (anoncoin.conf):
onlynet=tor
tor=127.0.0.1:9050
addnode=m6xwmit3jovw34r2.onion
addnode=aqmdzilaqayzjb6o.onion
addnode=sf2px26xdsqphkef.onion
addnode=wwniwxrykmk2wtcw.onion

On windows, simply type "%appdata%" in Run, then navigate into Anoncoin folder and create the file if it does not exist.
MARK: create it with notepad and select all files (in save dialog), and make sure it don't add a trailing .txt so the file will be named anoncoin.conf.txt

On linux (Terminal):
cd .anoncoin
touch anoncoin.conf
open anoncoin.conf
