Title: Re: Need some advice! Help a ninja out!
Post by: RS7FI8ZRkm on June 08, 2013, 07:02 pm
you don't want to raise any suspicion to the package/envelope, so using the correct name is wise, but not necessary.
you can test by just mailing you're self a letter with the fake name see if it turns up ::)..

also receiving mail with drugs isn't illegal unless you're knowingly doing so.. worst case, deny, deny,.. deny!.. 

make sure the person shipping, takes some short of measure of smell proofing. with some sort of MMB like mylar packing..

tell him wear gloves wipe everything down with rubbing alcohol..  always use a legit looking return address..  an make that package look like any other parcel there :)

hope that helps some.. read through the shipping threads you'll find more then you need there.

Title: Re: Need some advice! Help a ninja out!
Post by: BlackIris on June 08, 2013, 07:24 pm
Use your real name and address, it is much better. Don't use a P.O. box or a false name, it is just a waste. Deniability is actually worse with a P.O. box and with a false name you risk of the package not being sent to you.

You want to blend the package as much as possible with usual mail you receive. The less complicated the better, the more normal the better.

Naturally if then you plan on becoming a reseller the thing will change, but in that case you need to do completely different things.
