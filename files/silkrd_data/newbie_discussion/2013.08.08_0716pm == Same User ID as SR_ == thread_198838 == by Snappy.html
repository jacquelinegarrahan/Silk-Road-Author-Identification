Noob here, testing the waters. Question is, do most of you on the forums here use the same user ID as your SR account?  Seems like most of the vendors do in order to promote business and respond to feedback, but what about the buyers?

Thoughts?

Title: Re: Same User ID as SR?
Post by: postrex on August 09, 2013, 12:03 am
Use different passwords just to be safe, but you should never have any problems purely from using the same nickname on SR and the SR forums.  If one site is compromised, it would be fairly safe to assume the other is as well, so just keep that in mind, and limit your exposure the same as you would anyway.

And of course, don't use a nickname that links to your real life.
