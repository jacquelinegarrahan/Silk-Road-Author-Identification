Easiest way to maintain maximum privacy and security is to download TAILS and burn it to a DVD or USB drive. Its a linux distro designed to run from memory and clears all traces when the media is removed. It comes pre-configured with TOR and a secure browser, etc.  You just put in the DVD or USB, restart and boot from it.
