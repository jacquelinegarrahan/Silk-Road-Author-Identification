Post by: TideHat on March 25, 2013, 03:25 pm
I've been in the same situation, but never ended up taking that many! That is quite a lot, obviously you have a huge tolerance. Anyways, i also felt pretty shitty the following days after i started taking around 250mg a night to sleep, groggy and like a zombie sometimes. If you need to i may suggest tapering down instead of stopping completely because it will be very hard to fall asleep if you've beenn using it for a month then stop.

But it shouldn't last more than a few days ~ week at max. I don't have any scientific proof or anything, but just from experience. DPH has some long term effects if you use high doses very often, it can be bad for you. So try to take less, or take a break for a while. Some natural supplements that promote sleep you can try would be Melatonin or Valerian root (or both) they help me relax and fall asleep. good luck!
