Title: Re: Browsing from .to
Post by: scout on May 15, 2013, 04:22 am
If you aren't using Tor, then browsing using the .to proxy is incredibly risky.  .to is a proxy letting you access Tor sites without the use of Tor.  Whoever runs that proxy has access to your IP and other information.  The .to site itself issues the following warning:

Quote from: onion.to
By using this service instead of connecting directly to the Tor network you are trading off security and anonymity for convenience. Know that we can identify the IP of visitors using this service (but chooses not to). We strongly recommended to access Tor Hidden Services directly using the Tor Browser Bundle to increase your security and anonymity.

You should be using the Tor Browser Bundle itself and not messing with this .to proxy.
