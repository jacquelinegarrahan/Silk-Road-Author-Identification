Title: Re: Vidalia Control Panel
Post by: Nakorx on July 18, 2013, 07:39 pm
Unless you know what you are doing 100%, it is best to leave the control settings the way they are. And by Vidalia control settings, I assume you mean just that, the "settings" menu on the Vidalia Control panel.

As to changing identity, I have been using Tor for about 15 months and have never had to change identity. Not even when I have been ultra-paranoid.
However, there are some really useful tips that it can be used for:
1) When you get the Tor message "such and such has timed out, try again later"
2) Or a variation, like this "there was problem with the server at so and so and the system has not been able to load your data before it was reset.

Now many of these problems can be solved just as the page says, just hit "try again".
But there are a few that behave differently, and if you look carefully, you will notice that the little circle, representing data traffic, is missing, and you will instantly get the message "there is a problem loading this page". (By the way, these might not be exactly the correct messages, as I am doing this from memory, but they are close enough) and without getting technical, it is because Tor has marked these pages as being inaccessible for the time being, which could be hours!

Now this is where the "New Identity" comes in useful. If you select the New Identity under these circumstances, you will fool Tor into thinking you are a new user and have never visited this site before and it will usually work.

So next time you get the "problem page will not load" give it a try.

Title: Re: Vidalia Control Panel
Post by: PlanB on July 19, 2013, 09:55 am
As many others on this page have said, the majority of the time you won't need to mess around with any of the button's on the control panel of Vidalia. However if you did want to mess then here's my understanding of what some of the buttons do if used.

• Stop Tor - Completely stops the connection between other networks basically stopping you from surfing.
• Setup Relaying - Means you can set yourself up to be one of the middleman networks to help peoples connections to become scrambled and untraceable, just like for what happens to you when your surfing....Someone is being that scramble relay for you.
• View Network - View network allows you to see currently the journey your connection is taking whilst it goes through the interwebs, so you can see what countries your connection goes through to be scrambled, so my connection is going through USA & Norway.
• Use New Identity - This resets which path and people your connection travels through to be scrambled, I usually press this when Tor is having loading problems, therefore my connection changes and goes through a new middleman giving me a working relay.

All the other smaller buttons are mostly self explanatory like the bandwidth graph for example. Hopefully this has been of some help to you, happy surfing chap!
