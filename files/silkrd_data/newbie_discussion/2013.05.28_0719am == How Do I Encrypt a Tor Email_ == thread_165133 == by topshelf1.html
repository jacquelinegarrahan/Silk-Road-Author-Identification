Someone correct me if I'm wrong but encrypting emails are the same as encrypting your address for an order.  Some pgp programs just have an addon that lets you decrypt/encrypt from inside an email program so it can be confusing.

When all is said and done when your encrypted an email message your just encrypting the body text of the email, so anyone that intercepts can't see your message.

So you would need to get the recipients public key, then import it into your pgp program.  Type out your message, use your pgp program to encrypt with the recipients public key, then copy and paste the encrypted block of text into your new email message then send it.

I think there are tutorials on the forums about using gpg4win if not I'm sure there are youtube videos.
