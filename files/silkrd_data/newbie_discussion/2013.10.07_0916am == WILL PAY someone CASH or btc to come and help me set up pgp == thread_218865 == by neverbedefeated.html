DOWNLOAD/INSTALL/GETTING YOUR PUBLIC KEY:

1. If you're using windows download this:
http://www.gpg4win.org/download.html (the first link size is 29megabytes)

2. Install the program and open "GPA"

3. At the top next to File and Edit click "Keys" and then click "New Key"

4. Fill in all your information, your name is the name you want to show up (ie walkaboot) and the rest is optional

5. Go back to the GNU privacy assistant and under the "Key Manager" window you'll see your name. Right-click>export key

6. save the .txt file somewhere. Open it and you're PUBLIC key will be in there. This is the one you give out to everyone.



IMPORTING KEYS:

1. Create a folder and name it something you will remember.

2. Inside of that folder create a .txt file

3. In this .txt you will paste the PUBLIC key of someone you wish to send an encrypted message too.

4. Open "GPA" and in the GNU privacy assistant click the "Import Key" button.

5. Open the folder you created and select the .txt file



HOW TO ENCRYPT/DECRYPT MESSAGES:

1. Open "GPA" and in the GNU privacy assistant click the "Clipboard" button.

2. TO DECRYPT: Paste the encrypted message in the clipboard and click "Decrypt," then select your personal key

3. TO ENCRYPT: Write your message in the clipboard and then click "Encrypt," then select the key of the person who sent you the message.
