Quote from: CaptianKirk on September 05, 2013, 10:05 am
COULD YOU SHARE HOW YOU ACCOMPLISHED THIS. IVE BEEN STUCK WITH THE SAME PROBLEM

In your original PGP program you need to find the option to export both of your keys--your public and private.  Usually when you casually export a key you're only exporting the public key which you publish so people can encrypt messages to you.  Exporting your private key as well usually takes a few extra steps because of the security risk involved in having your private key floating around in clear text.

After you find the option to export your entire key-pair save it and use the Tails PGP key manager to import your key pair.  You can test to see if it works by opening the text editor and encrypting and then decrypting a test message to yourself.
