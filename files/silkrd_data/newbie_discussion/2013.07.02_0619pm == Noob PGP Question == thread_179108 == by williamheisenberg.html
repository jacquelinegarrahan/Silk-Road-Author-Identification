Hello all, I have a simple question.  When encrypting one's address to receive one's order do we send our public pgp key as well?  I know that one is suppose to send the encrypted pgp message but I'm wondering about the public key.  Seems like I need more practice with this.
Title: Re: Noob PGP Question
Post by: asafa on July 02, 2013, 06:31 pm
No. You only need the recipents public key to encrypt the message, he is able to decrypt it by using his private key. Your public key is the one you can share with people, so that they are able to encrypt messages for you, which you can in turn decrypt using your private key.
