Title: Re: Tor status page concern
Post by: spectrum on August 12, 2013, 02:25 am
Your Tor client creates new circuits every 10 minutes, so your IP address should change every 10 minutes, although theoretically a new circuit could terminate at the same exit node.

The check page can sometimes be wrong. If it fails to query the database of exit nodes, then it will say you aren't using Tor, but if you are browsing hidden services, you are obviously using Tor, because there's no other way to access them.
