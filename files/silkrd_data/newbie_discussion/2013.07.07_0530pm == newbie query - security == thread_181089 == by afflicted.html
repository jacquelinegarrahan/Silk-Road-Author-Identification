Title: Re: newbie query - security
Post by: mcguire39 on July 07, 2013, 05:39 pm
The VPN, if configured so that traffic goes through the VPN then to the tor entry node, helps obfuscate from your ISP and other snoopers that your IP address is accessing Tor.

PGP in the SR context is normally used when placing an order to encrypt your name and address, so that only the vendor can access that information.

Tails is an image that helps you to use Tor on pretty much any computer without leaving any trace later that Tor was used on it. Basically you put it on a USB stick or possibly DVD and boot from that.

Kleopatra is for managing keys and doing PGP encryption, but it's not nearly as easy to use as something like GPG4USB.

gpg4usb is compatible with Kleopatra I'm guessing .........

You don't need both; they both support PGP encryption/decryption and signing. GPG4USB is easier to use because it has a built in text editor and allows operations directly from the clipboard (copy/paste). Kleopatra wants to just work with files already saved on the computer.
