It's different. It is not the same as using DMT after LSD.

Smoking DMT after taking harmala alkaloids slows down dramatically the experience (and naturally also making it longer just for this) also making it stronger (you need much less to obtain the same results, 20mg is like smoking 50 for example) but it has not a different flavor as LSD + DMT.

DMT + harmala is used principally if you have difficulty "bringing on earth" (so to speak) the experience(s) you have with DMT usually. Personally I've used the combination many times but now I use it no more since I've not problem doing what I have to do with using DMT alone, but especially at beginning it was somewhat useful, yes, to interact better with the experiences I had (with DMT you have somewhat to learn how to step up at the same speed of it while you are there and it takes a little of time - and plateau, some of them very fearful if you are not accustomed to certain planes as the ones of hell/underworld - to do so fully).
