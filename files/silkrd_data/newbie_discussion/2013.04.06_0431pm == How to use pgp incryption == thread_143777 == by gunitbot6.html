Title: Re: How to use pgp incryption
Post by: tree on April 06, 2013, 04:42 pm
Download gpg4usb, it's the easiest to use IMO. To encrypt messages, copy and paste the public key of someone to a text file on notepad (if you're on windows), then click on import key on gpg4usb, select that file, and the key is imported. Next, type your message on the main screen, select the one you wish to encrypt it to on sidebar on the right, click encrypt. There's also a tutorial on the gpg4usb website, and there are tons of tutorials on here, just check out the stickied posts in the security subforum.
