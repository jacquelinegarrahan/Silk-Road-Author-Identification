Whilst I'm sure this key has been replaced by now, it serves as a valuable reminder of what *not* to do.  Ever.

1)  Do not use a PGP tool run from an external website (e.g. IGolder).
2)  Do not use any PGP-like software based on the Bouncy Castle libraries (e.g. Portable PGP, IGolder, etc.).
3)  Do not use a PGP key of a size lower than 2048-bit strength.
4)  Do not post your private key anywhere, only post the public key.

The post above broke every single one of those rules.

Get GNU Privacy Assistant it is the graphic front end to GNU PGP.  It's really easy to use to creat a new key just click new key.  To import a key just click import and to export your public key just click export.  It order to encrypt something just click on clipboard then type your message into it and click encrypt and select the key you would like to encrypt with.  To decrypt paste the message into the clipboard and click decrypt and select the key you would like to decrypt with.
