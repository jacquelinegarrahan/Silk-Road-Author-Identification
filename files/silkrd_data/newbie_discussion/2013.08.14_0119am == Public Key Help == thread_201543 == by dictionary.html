Title: Re: Public Key Help
Post by: Hitch on August 14, 2013, 01:24 am

   Hi :) Your public key is given to individuals you want to converse with. You use their public key to encrypt messages that they can decrypt.

   Private key - never give out, used to encrypt messages that others can identify comes only from you.
   Public key - give out to those who you want to be able to encrypt messages to you.

   When you encrypt a message you're encrypting it with your private key and the intended recipients public key, so it becomes a message that only the two of you can decrypt.
