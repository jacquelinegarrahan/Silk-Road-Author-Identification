Title: Home Country Setting on Silk Road

This thread has two purposes, the first some thoughts on the current Home Country listings and the second is just a list of what countries are currently listed by the Home Country Setting on Silk Road for future reference. For those not familiar with the Home Country Setting it's just a setting for using the Silk Road website which is used to show all domestic vendors in your region if you filter it that way.

When I list all countries in the world personally I generally go with the definition in this article: http://geography.about.com/cs/countries/a/numbercountries.htm - in other words 196 countries. However Silk Road which seems based on postal codes as on http://en.wikipedia.org/wiki/List_of_postal_codes has 243 listings after excluding "Select new" and "undeclared", which is an additional 47 countries in comparison. I cross-checked them to each other and found 58 discrepancies.

First a small notice in my previous post I mentioned 243 listings in the Postal Codes Wikipedia Article, this is apparently an incorrect number as the postal system is based on a standard called ISO 3166-1 which currently has 249 regions. Proof for this can be seen either at https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2#Officially_assigned_code_elements or http://www.iso.org/iso/country_codes/iso_3166_code_lists/country_names_and_code_elements.htm

It's interesting how web sites use so many variations on this list and don't keep it up to date (Silk Road included).
