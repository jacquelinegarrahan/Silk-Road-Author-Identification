Yeah I finally got it to work, ill go over some of the holes in my knowledge here so if anybody else is unsure, hopefully it will make things easier for you.

First of all, for mac users, make sure when adding the services, the GPG ones are checked under Text, and not just Files and Folders (in the Services preferences). This allows you to highlight text then right click and go to services to either encrypt or decrypt. This works on Chrome and Text editors, doesnt seem to work with Tor.

You dont need to create any files at all. Simply create a PGP key, which allows you to give your public key to others (so they can send you encrypted messages). Once you have created the sec/pub key, right click it in the main GPG Keychain Access window and click copy. This allows you to paste your public key anywhere :)

When someone sends you a message, copy the text and paste into a text editor. Highlight all of the text and right click then Services and then Decrypt. You should then see the plain text of the encrypted message from the sender.

If you want to send a message to someone via their public PGP, then copy their public key to a text editor, highlight it all and then right click and then go to Services and import key.

Once you have done this, you can create an encrypted message that only the holder of the public key can decrypt. Open a text editor and type your desired message, highlight it all (starting to see a pattern?) right click and Services, then Encrypt. You will be asked which key you want to encrypt it with and then be presented with the encrypted message. Copy this and paste anywhere you want as only the holder of the secret key to match the public will be able to decrypt.

Hope this has cleared a few questions you may have. If you want to test it out, my public key is at the bottom of the first post of this thread, feel free to PM me a message, and dont forget to put your public key if you want me to send you an encrypted message for you.

If any of the information is incorrect in this thread, please correct it :)

Yep you got it, so for OSX all you do is:

1) Download and Install GPGTools
2) Go into System Preferences > Keyboard > Services and make sure all the openPGP boxes are ticked (really important and the part that confuses most if not done)
3) Copy and paste the desired Vendor PGP into a text file, open GPGTools and import that text file.
4) Open TextEditor, type your message / address, highlight it, right click (or old OSX equivalent) and select services > Encrypt
5) Select your vendor key from the list
6) Copy and paste the new generated PGP back.
