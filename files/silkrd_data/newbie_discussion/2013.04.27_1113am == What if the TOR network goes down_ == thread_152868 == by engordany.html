This is why Yubikeys and USB backups are so important. If you have valuable information--such as hash accounts--then you should be responsible by making a copy of them offline and AFK.

If you are worried about BTC, then just reflect on how many miners there are, and that new blocks continue to be mined. That is all the security you really need for BTC to survive.

As for TOR, I believe there are too many contributors and dedicated users that it would come to a complete halt. Maybe temporary hiccups, but nothing absolute. Furthermore, if TOR were seriously threatened, then the next generation of liberty would certainly follow, as thunder is heard frmo a flash of lightning.
