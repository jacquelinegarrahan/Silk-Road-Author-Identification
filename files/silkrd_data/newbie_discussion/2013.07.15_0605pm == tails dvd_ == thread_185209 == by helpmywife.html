Post by: Chernovy on July 15, 2013, 07:19 pm
I think i understood that....

When you download tails it comes in DVD ISO format.  You don't need to burn the DVD before you put on the USB stick though. That's just the documented way to do it.  DVD-R, DVD-RW, DVD+R, DVD+RW will work just as long as your writer accepts the format.

Here are you options:

 1. Burn the ISO to a DVD. Boot from the DVD and install to the USB.
 2. Use UNetBootIn(http://unetbootin.sourceforge.net/) to move the DVD to a USB.  Haven't tried this with tails but since tails is Ubuntu based it would probably work.
 3. Use a Virtual Machine like VMWare or Virtualbox to boot from the ISO and install it to the USB.

Title: Re: tails dvd?
Post by: Chernovy on July 15, 2013, 09:34 pm
There are applications that will let you download and burn simultaneously... but i wouldn't recommend it.

Tails is a LiveDVD which means you can run it from the DVD or Install it. From the instructions i've seen on the Tails website it can installed to a USB Stick.

The reason why the instructions say 2 USB sticks is because they are not burning to a DVD.  The first USB Stick is just the DVD in a USB format. The second DVD is where you are installing tails permanently.

A truecrypt volume can be any size.  The 4GB limit is based on the File System.  For example Windows 98 used FAT32 as it's file system and the largest file was 4GB.  Windows XP and on used NTFS which can have  a much larger single file..

If tails uses FAT32 then 4GB will be the limit.  Most bootable USB sticks are FAT32.  Only way around it is if you are familiar with Linux
