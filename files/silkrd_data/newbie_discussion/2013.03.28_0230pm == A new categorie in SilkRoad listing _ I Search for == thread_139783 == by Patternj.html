Post by: Codesr on March 28, 2013, 02:49 pm
The reason it's difficult to find things with the search feature on SR, is because of the way it's programmed. It always returns a ton of results that are not related to your terms, because it doesn't limit the results to things that contain the words you searched for. It's using a mysql like search, or something like that, so if you search for a term like "red" you get results that contain terms like "fred", "color", and "blue". Instead of using mysql like, it should search all titles and descriptions for the exact individual terms in the search individually and then combine the results. A lot of people would like a sort by price feature, but first the search has to be improved so that it functions like a proper search.
