Post by: DoctorFate on May 15, 2013, 06:49 pm
Yeah I would try looking up tutorials for "using PGP on a MAC" 

I found a few but I'm not sure if there are free pgp programs for the mac...

You know how pgp works?  You get a public key, put it in your pgp program, then use your pgp program to encrypt some sensitive information, like your name, with the public key you got.  Then you send that encrypted block of information to the owner of the public key to decrypt and get the information.

Another option could be to use a virtual machine like virtualbox (https://www.virtualbox.org/) and run tails (https://tails.boum.org/), which has all the pgp and tor stuff you need.

Or you can just use the virtualbox  to run ubuntu or windows...

Of course it's better to actually setup a dual boot on your computer to run one of these OS's but if you don't know how to do that a virtual machine is a less secure option.

The new pgp club thread is a good place to practice if you want to be sure your can encrypt your personal info:
http://dkn255hz262ypmii.onion/index.php?topic=107219.0
