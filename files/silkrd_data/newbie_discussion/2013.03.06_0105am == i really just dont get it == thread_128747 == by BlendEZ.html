Title: Re: i really just dont get it
Post by: tramonym on March 07, 2013, 06:20 am
you have public and private key (called key pair)
other person has public and private key (=keypair)

public key can be send to others (private key not!!)

in order to send encrypted message you will have to encrypt the message with other person's public key. this makes sure that only the person which has the private key of the keypair can decrypt / read it.

same way around.

hope this helped
