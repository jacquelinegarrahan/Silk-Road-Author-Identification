Title: Re: Avatars?
Post by: dotgoat on August 16, 2013, 11:39 pm
Yep, it's traveling over the same anonymized connection that everything else is traveling over.  What you want to make sure of is it doesn't have any metadata info.  For example if it was a picture taken with an iphone it will have the exact geolocation coordinates embedded in exif.  There's probably utilities out there specifically for this but I like XnView as a general purpose image viewer and it allows to see, modify, and completely remove all exif/iptc/whatever other metadata there is.

If you just grabbed a pic off the internet then it's not really an issue.
