Quote from: BL93X on February 09, 2013, 11:20 pm
Hi all, I used an old resident's name on a package but it's now several days late. Am i super screwed? Would cops come knocking on my door? Should I clean computer + clean house?

Packages can be late for any number of reasons. Were you previously getting mail for this person at your place? If you were they probably don't have forwarding so the package is most likely late for another reason.

In future though, it is bad practice to use an existing resident's name on your packages. Not only is there an increased chance of having the item intercepted by someone (post office, previous resident, etc) but you are also amplifying repercussions if the package is intercepted.
