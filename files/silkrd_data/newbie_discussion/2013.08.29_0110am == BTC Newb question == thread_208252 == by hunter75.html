Title: Re: BTC Newb question
Post by: frank-butcher24 on August 29, 2013, 01:39 am
Yes but the application will download the entire blockchain, so you'll need a 16GB or 32GB one to be safe.

Create a large Truecrypt encrypted file on the USB drive, and then unpack the bitcoin software into a folder called "bitcoin".

Also put the Tor Browser Bundle onto it, I suppose.

Before running it for the first time, create a folder called "Data" or whatever, then create a shortcut in the same location as the software, which points to bitcoin-qt.exe -datadir=X:\bitcoin\data

The ensure you always mount the truecrypt volume as X:, and run the software by clicking on the shortcut. Make sure you're running Tor or it won't be able to make any connections. The Bitcoin software comes with the Tor (proxy) option enabled by default.
