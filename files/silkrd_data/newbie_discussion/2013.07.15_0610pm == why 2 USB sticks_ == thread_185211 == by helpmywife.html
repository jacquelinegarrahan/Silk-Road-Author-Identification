I am in no way an expert in this, but as i understand it, you can do i with only 1 usb. You just have to make a boot dvd first, and then you can use this, to set up a tails usb with persistent volume.

As i understand it, the reason why you need 2 usb drives if you dont want to use a boot dvd first, is that you have to create the tails usb, boot it up, and then (when you are in tails), you can make a new tails usb with persistent volume.

So as i understand it, there is no problem in creating a tails usb, directly, but the persistent volume can only be created as soon as you are booted into tails/linux, and then it is not possible to make a persistent volume on the usb you are running from (because it is in use).

As stated i am not sure if this is correct, so if i am mistaken, please feel free to correct me.
