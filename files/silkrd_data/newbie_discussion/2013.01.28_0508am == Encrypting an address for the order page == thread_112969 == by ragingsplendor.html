Correct. You encrypt your sensitive info using their Public Key, found on a vendor's Home page, with Gpg4usb or Gpg4win. Some vendors want all correspondence Encrypted, especially if talking about bulk orders, but generally speaking, Encrypting your sensitive info and pasting in the Address box on the order page is spot on.  :) :)
