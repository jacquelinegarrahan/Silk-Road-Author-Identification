Post by: EverAfterGlow on September 03, 2013, 08:17 am
My advice.

Use PGP and only use vendors who use PGP, because it shows they are somewhat cautious. Dumb/careless vendors can compromise their buyers security in a number of ways.
 
Always do as much research as possible on any vendor that you plan on using. Forums, Feedback, and then watch over time. DON'T spontaneously order just because something looks good.

If it looks too good to be true it probably is a scam.
 
Connect to Tor networks using connections that have no attatchment to you.

Use a computer that doesn't contain your personal info...

Smaller buys are more likely to escape detection.

Domestic is a lot safer.

Don't use TOR to browse clearnet, only SR.

Don't open any downloaded files while using Tor.

Don't start vending unless you want to be paranoid for the rest of your life. :D
