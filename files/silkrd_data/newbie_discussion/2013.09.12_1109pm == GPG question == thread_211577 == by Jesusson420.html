@Jesusson420
hey dude,

once you install GPGTools a small window appears titled Install GPGTools.
there are 5 pages to scroll through (seen on the left). Introduction, Destination Select, Installation Type, Installation and Summary.

when you get to Installation Type a list appears with boxes that need to be checked.
the second Type in the list is GPGTools Macgpg2. which i think you're missing.
check all of these boxes and continue.

this link shows screen shots of the process.

http://thelastbastille.wordpress.com/2013/01/18/how-to-configure-use-pgp-encryption-for-email-mac-osx-instructions/
