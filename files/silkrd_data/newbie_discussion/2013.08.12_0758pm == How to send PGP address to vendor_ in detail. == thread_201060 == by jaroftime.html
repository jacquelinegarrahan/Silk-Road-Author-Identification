Title: Re: How to send PGP address to vendor? in detail.
Post by: kahn on August 12, 2013, 08:59 pm
Hope this helps...

1)Use notepad (or other) type address (and any message you want to send) and save.
2)Encrypt saved address (and any message you want to send) and save.
3)Copy encrypted address and paste onto box at bottom of order screen.
4)If want to send your PGP PUBLIC KEY (for Gods sake DO NOT send PRIVATE KEY to ANYONE) copy and paste PUBLIC Key below your encrypted address with a clear text message in between that says "Below is public key". Don't encrypt public key.

If you have any problem PM and I'll help if I can.
