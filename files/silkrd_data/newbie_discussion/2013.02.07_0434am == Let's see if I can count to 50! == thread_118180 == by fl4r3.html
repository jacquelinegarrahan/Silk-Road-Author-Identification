It's the only number that we know of that has this property.

31 is a lucky number.  If you start with 1 and cross out every second number up to 100, and then start again taking the next number that's remaining, which would be 3, so then you cross off every third number of the remaining numbers.  If you keep repeating this process (the next number would be 7, so you'd cross off every 7th number of the remaining ones), 31 is the highest number under 100 that's left.
