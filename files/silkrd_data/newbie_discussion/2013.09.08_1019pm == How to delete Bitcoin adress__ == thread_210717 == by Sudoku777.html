Title: Re: How to delete Bitcoin adress??
Post by: The Bubble Dreamer on September 08, 2013, 10:54 pm
There is no such thing as deleting a bitcoin address.

Bitcoin addresses start existing on the network when first seen in a valid transaction. The person with the matching private key can transfer bitcoins from that address. It is even possible to generate a valid bitcoin address without knowing the private key (nobody would be able to transfer bitcoins from that address).

If the private key for a certain bitcoin address is lost, the address becomes unusable (though bitcoins can still be transferred to it). This happens all the time and there is a significant amount of bitcoins that has been 'lost', but there is no explicit 'deletion' of the address. Remember there is no central authority for bitcoin addresses.

I guess this doesn't fully answer your question - I don't really know what you observed in the blockchain, but it was not a 'deleted' address.
