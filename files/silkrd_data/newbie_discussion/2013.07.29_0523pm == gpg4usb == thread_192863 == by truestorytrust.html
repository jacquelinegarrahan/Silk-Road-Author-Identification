Post by: Nakorx on July 29, 2013, 09:48 pm
No this is wrong. First you need to import their public key onto your keyring.
Load up your gpg4usb for whatever OS you are using, e.g. if you are using Windows then click on "Start_Windows.exe".
Then you copy their public key, which you will find on their profile, by holding down left mouse key and highlighting it and using the "copy" command to get it onto the clipboard.
From gpg4usb click on the "Import key" and then click on "clipboard".

You should see a box come up saying "Key Import Details" and then you should also see at the top "considered 1" and "Public imported" or similar. (Doing this all from memory, so may not be 100% correct, but close enough)
Then you will see the name of the vendor, email address, status and Fingerprint.
The vendors name will also appear on the "Encrypt For" panel on RHS screen.

Now type the name and address you want to encrypt in the main panel. Then tick the box beside the vendors name in the "Encrypt For" panel.
Click on the "encrypt" button and you should see a PGP message which looks like a PGP key.
Next use the "Select All"  button to highlight the message, and then click on the "Copy" button to get in onto the clipboard.
Finally, paste this into the address box in the shopping cart, and you are done.

Make sure when you highlight the vendors PGP public key that you start with the 6 ------ at the very beginning and the last line which looks as follows:
-----END PGP PUBLIC KEY BLOCK-----
If you don't copy these it will not work.
There is a tutorial that goes with gpg4usb, tells you all this although not so clear maybe.
Hope this helps.
