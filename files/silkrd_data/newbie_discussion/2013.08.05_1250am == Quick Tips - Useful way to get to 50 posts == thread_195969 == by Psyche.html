I'll kick it off.

Use the OTR protocol for a more secure and convinient real time chat platform than PGP.

It uses a combination of the diffie-hellman key exchange, AES, and SHA-1. You can chat over most services such as google talk or AIM but I don't trust those services not to be keeping those logged for future decryption when quantum computers come to fruition, you need to find a widely used XMPP server with a privacy policy which has "No logs" in it.

You can use it on any operating system including mobile ones.

For Windows: Install pidgin and pidgin-otr off of this clearnet website, obviously visit it with caution: http://www.cypherpunks.ca/otr/

For Ubuntu Linux and similar distros: (clearnet again):http://www.cypherpunks.ca/otr/ubuntu-install/otr-setup.html

On IOS you can use chatsecure which is open source and on android phones you can use gibberbot.

It comes natively installed on Tails.

I like it because it is very easy and user-friendly and effectively combats the "Why Johnny can't encrypt" problem of encryption's inconviniences surpassing it's usefulness for normal users causing them to put their trust in websites like facebook, google, ect who are prism compliant.
