Title: Re: my first order
Post by: captainjojo on April 24, 2013, 02:06 am
Honestly, you shouldn't be using privnote or some other third party site to send your address to the vendor.  It is quite safe to use the SR order box to enter your address as the address is erased the minute the transaction is completed.  The vendor must cut and paste the address to their system and they are supposed to get rid of it after they have addressed your envelope.

If you are concerned that somehow someone is going to crack SR and someway be able to find your address then you should use the vendors PGP key to encrypt the address and paste the encrypted text into the SR address box.

If you are unfamiliar with PGP go to the SR wiki where you will find information on the what, where and how of PGP.  There is also I believe a thread on the forums explaining how to use PGP as well.

Use liberte linux, the gpg key program is simple to handle. You create a document with a .gpg extension with the public key in it (like a notepad document). Then you import that document using the tool with the button <import>. Then, you open the clipboard, write your address and encrypt it with the key you have imported. If a vendor sends you a message they should be doing it with your public key so that you can decrypt it with your private key, which only you have. The keys work in pairs like this.
