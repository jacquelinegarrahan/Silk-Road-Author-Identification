Post by: trippylizard on January 26, 2013, 06:02 am
If you're worried about them not being acid, there's a couple things you can do to try to test. Taste one... acid should be pretty much tasteless, or have a very mild taste. RC's will have a very noticable bitter and numbing taste. Also, acid works perfectly fine if you swallow it, but nbomes need to be held against your cheek/gum/under tongue for a very long time to work. So if you're concerned, just swallow a tab or two straight down, and if it works it's likely real acid, and if nothing happens, you've avoided taking an nbome. :)

I think DOx's still work if you swallow them straight down, but they'll take forever (like 3 hours) to start working, and will last ages too, so if that happens you'll know what the rest of the batch is.
