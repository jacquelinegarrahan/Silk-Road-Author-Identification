Title: Re: Ephedrine to Amphetamine?
Post by: kilomonster on November 10, 2011, 08:25 am
There is no easy method to convert ephedrine to amphetamine, period.  If you are willing to take the time to learn organic chemistry and at least obtain some basic lab equipment then you have a chance.  You will also need to invest some time and some $$$ into some safety equipment if you are going to do it the right way.   Assuming you are willing to do all this, for amphetamine you are not going to start with ephedrine anyway.  If you are going to start with an ephedra alkaloid, you start with phenylpropanolamine (PPA).  If you are in the US, it's no longer available for humans, but is still used in veterinary medicine.

If you are willing to take the time to learn how to do things right, more power to you.  If you're not, then forget it, and take Halogen's advice and get yourself some Adderall or dexedrine.

it's usually dirty methamphetamine. the process of crystallization is a way to remove impurities from chemicals, etc. you can even crystallize aspirin among other things. so from a technical standpoint crystal meth is not "dirty" although it can still have a lot of impurities. for instance they found some that was orange in color because it contained a huge amount of boron.

cranks different colors is the result of the process used. you can have green, brown, red, yellow and so on.

the blue crystals are from a different process altogether and purple is usually ok too, typically iodine from a phosphorus iodine process.
