Post by: mcguire39 on July 10, 2013, 02:18 pm
Not sure if this is already covered in the thread (and thank you to those who pointed this out), but Pidgin logs all your chat activity by default. And if you happen to be a Windows user, it stores this information (and all your chat nicks and settings) unencrypted in your UserData folder (irregardless of where you actually installed Pidgin).

So it's probably a good idea to disable chat and IM logging, and delete the old logs, and move your settings to an encrypted partition.

For example if J: is your encrypted drive:

1. With Pidgin closed, find the .purple data folder for Pidgin (in Windows this will usually be in a location like this:)
Code: [Select]
C:\Users\mcguire39\AppData\Roaming\.purple
2. Move that folder t your encrypted drive (J: for example)
Code: [Select]
J:\Users\mcguire39\AppData\Roaming\.purple
3. Make a symlink from your C: to J: so that Pidgin still works properly:
(From command prompt running as administrator):
Code: [Select]
cd /d C:\Users\mcguire39\AppData\Roaming
mklink /D .purple J:\Users\mcguire39\AppData\Roaming\.purple
symbolic link created for .purple <<===>> J:\Users\mcguire39\AppData\Roaming\.purple

AUTO-IDENTIFY:

Quote
To set a buddy pounce to auto identify (so you don't have to identify with nickserv each time you log in)

Type /query nickserv
Click conversation
Add buddy to your friends list (if nickserv isn't on your friends list this wont work)
Then add buddy pounce. In the pounce enter "identify yourpassword" (no slash, no quotations)
Checkmark Recurring
Checkmark "Pounce when buddy signs on"

That should do it for auto-identify


AUTOJOIN:

Quote
Click the tab/name of chatroom #cheers
Click Conversation
Click Add..
You'll see a popup dialogue in that window it'll have a small box which says "autorejoin when account connects" CHECKMARK THAT!!
