Ok if you are you need to ask yourself the following questions -

Have you sold IRL?

If yes are you willing to work harder than you would IRL?

Are you willing to learn from other sellers on SR?

Are you the sort of person that does market research?

Do you have a basic idea of how to wash BTC on a basic level?

Are you customer orientated?

Are you good at communicating with people?

If yes to all of those, give it a go. If not, then you either need to learn that shit first or if you don't want to learn it, don't bother because you need them all to make it work on here.
