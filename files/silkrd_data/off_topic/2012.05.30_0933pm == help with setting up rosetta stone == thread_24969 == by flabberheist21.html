Title: Re: help with setting up rosetta stone
Post by: CapnJack on May 31, 2012, 10:26 pm
Install the program.  If on the finish page it has a check box for launching the program, then uncheck it.  Otherwise, don't open the program yet.  Copy the rosettastoneversion3.exe file from the ones you bought by here by right clicking and selecting copy.  Paste it in the program directory (usually something c:\program files (x86)\rosetta stone).

Install daemon tools or something like magic disk (http://www.magiciso.com/tutorials/miso-magicdisc-overview.htm).  I usually use magic disk.  Once you install look in the bottom right where the time is and find the magic disk icon and click it.  Select mount and then browse to the language iso and open it.

Open the program and you should be good to go
