Title: filemaker program I made to help keep track of my Silk Road orders
Post by: username_goes_here on October 30, 2012, 04:40 pm
Hello all,

I am a regular buyer here and created a drop dead simple filemaker program to help me keep track of all my orders, reviews, and vendors.

You can download it here

***CLEARNET WARNING***

http://filebin.ca/KuDk5rtrYD1

You have to have filemaker installed to run this... if there is any interest I can create full runtime programs if you want.

This program does not connect to the internet with the following exceptions.

1. The program will access bitcoinexchangerate.com to learn the current value of bitcoins
2. The program will access blockchain.info and show you a transaction status if you click on the "show status" button

outside of that the program is fully self contained.  You'll be able to store all the follwoing information

order number
order date
btc paid
btc to USD conversion -- so you know how much it cost you down the road
USD unit price - so you know how much each unit is
Arrival Date
Vendor information
Review notes and 1-5 stars
The program also can store your btc address and private keys if you want.
You can also print off a btc paper note or paper wallet
You can store 2 pictures of every item
