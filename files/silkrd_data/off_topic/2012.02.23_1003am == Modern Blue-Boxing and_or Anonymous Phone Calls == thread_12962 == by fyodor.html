Title: Modern Blue-Boxing and/or Anonymous Phone Calls
Post by: fyodor on February 23, 2012, 10:03 am
I've spent the last 5 hours looking this up. Blue-boxing if you don't know, used phone phreaking where hackers war-dialed and found out cell numbers used by tower inspectors, i.e. completely unregulated. This particular method doesn't work anymore but there must be a way! I bet this guy knows (http://www.wired.com/threatlevel/2010/07/intercepting-cell-phone-calls/). I've seen GSM modems for arduino, but you could still be triangulated no matter your spoof skills, right? If anyone has found a way between all the scams out there please let us know. I'd probably pay for the answer too. Also if anyone knows about voip call spoofing(using asterisk?), that'd be amazing info/setup to get my hands on.

So far I've settled on using wifi, TOR/VPN, skype, and goog voice as an alternative. I know that voip/skype does work through TOR, but a recent research paper, which I cannot find atm, said any call through TOR longer than about 90 sec could be traced if they controlled both entry/exit nodes, albeit you would be gone by then. I could use an android but I wouldn't trust it unless I could completely disable the phone when booting linux from the sd card, but current methods are a side-by-side boot, somehow. I could use an old blackberry, but apparently that's impossible. What about a skype wifi phone? Well they're really expensive and may not work with the TOR/VPN. So what about an arduino? Well that has no OS so all the apps would have to be from scratch. One guy is doing it though (http://antipastohw.blogspot.com/2009/03/using-buttonshield-to-make-open-source.html)

What about one of those fancy Raspberry Pi's?! Good excuse to buy one! Fuck Yeah! Throw an alfa 2W wifi adapter on it with a 7db antenna; run Damn Small Linux, TOR script, and skype; salvage screen, keyboard, battery from an old blackberry. More functionality and cheaper than a skype wifi phone.

Let me know if you know an easier/better way or especially if you know how to really blue box with today's towers. Any research you've found I'd be glad to look at.
