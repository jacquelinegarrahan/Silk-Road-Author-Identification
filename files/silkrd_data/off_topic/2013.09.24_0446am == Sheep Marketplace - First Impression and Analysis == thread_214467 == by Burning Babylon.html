Title: Sheep Marketplace - First Impression and Analysis
Post by: Burning Babylon on September 24, 2013, 04:46 am
Yesterday when scouring the Atlantis Forums randomly the topic of where Vendors where planning on going after Atlantis disappearing came up, and one of the answers was "Sheep Marketplace". Now I hadn't heard of this site at all which made me take notice, and this thread will be my perspective of this new site and how they operate. All images linked in this post will have been taken today, namely the date 2013-09-24. Due to the scale of this project I will make one post now and follow up with one later with the rest when I have the time. Now before I get on to the site itself apparently they've done an official thread in the Silk Road Forums called Sheep Marketplace safe alternative SilkRoad: http://dkn255hz262ypmii.onion/index.php?topic=155291

The Original Poster has two posts since their registration which was roughly 144 days ago which does not exactly inspire confidence, unless they forgot their password after five days since they at the time of writing haven't logged in since May 8, 2013. Now I could go into some length to analyze this post and criticize it but I'll just leave it at it's a poorly worded fluff piece - I'm sure it's been improved since and the information might not be up to date any longer.
So, I enter the official URL http://sheep5u64fi457aw.onion and get to their start page which looks like this: http://torimagesbp2vt3u.onion/i/FSBR.png

So I'm not overly impressed by this appearance of this Start Page as it's kind of a cross between Silk Road and Atlantis in it's current state, with the exception it doesn't convey anything in particular. So, unto the Registration Screen which looks like this: http://torimagesbp2vt3u.onion/i/SfEU.png Here one can see a few small differences, the first notable one being the field "(not required) Promo key" being included which is not explained and for some peculiar reason a red reset button is included. Other than that it's pretty much all the same, except it's starting to get noticeable the creators of this site are having some problems with english grammatical structures - from now on I will not refer to any problem of this nature as they can be corrected and I have a feeling there's going to be more of them later, no point in going over all of them.

So there's two types of Accounts which can be registered, "Customer" and "Vendor". Now since I'm going to write this fairly long post I figure I might as well spend the 40 dollars needed for the Vendor Bond/Fee so I can go more into detail on how the interface looks and operates. Said and done, I registered a Vendor Account named "Burning Babylon" along with a Customer Account named "Thalassa" so I could see if there were any noticeable differences between the two types. When logging into my (non-activated) Vendor Account I instantly came to the start screen, but when I logged into my Customer Account I was greeted by the following popup: http://torimagesbp2vt3u.onion/i/kP0j.png - Ignoring the fact it somehow didn't appear for my Vendor Account this is the first big difference Versus other sites like Atlantis and Silk Road, namely keeping the users informed through the site itself what changes have been done recently, a commendable effort if you ask me. Before I commence further I'm running my Vendor Account in my main computer with the resolution 1920x1080 while the Customer Account is run in a VMWare Simulation with the resolution 1152x864. So this is what it looks like when first logging in for my Vendor/Customer Account respectively: http://torimagesbp2vt3u.onion/i/llHS.png / http://torimagesbp2vt3u.onion/i/9sqE.png

First thing that should be noticeable is the upper green line which will always appear whenever there's a new status change of some sort, personally I feel it's very poorly situated design-wise but it disappears after switching page so at least there's that. The next noticeable thing is the Front Page Listings do not have a fixed Image Width, which makes it look kind of bad to be honest. Information-wise it seems to focus on the Title, Seller and curiously Shipment Information. Unfortunately the exact same Listing System as Atlantis and Silk Road is utilized here but I will make an attempt to make them convert to the Upgraded Listing Options: http://dkn255hz262ypmii.onion/index.php?topic=182090 At the sidebar there's a small window detailing the current rates of USD, EUR and GPB against Bitcoin. Now this is actually half the job of applying a solution where one shows the Bitcoin Value versus the Value preferred by the Customer and the value preferred by the Vendor - for example see the thread Show prices both in BTC and in chosen currency: http://dkn255hz262ypmii.onion/index.php?topic=189788 Now unfortunately it appears there's currently only support for showing four different currencies, namely: "$ - USD", "฿ - BTC", "€ - EUR" and "£ - GBP" - this could most likely be changed somewhat easily in the future though I assume.

So at the top there's the following Alternatives showing: "Account", "Orders", "Messages", "Gallery", "Forum" and "Help". Let's start by addressing "Gallery" and "Forum" since these two options are somewhat unique for Sheep Marketplace.

First off Gallery is a Service to upload images for others to see I presume, when attempting to upload one is stopped by the message "Your browser does not support Java Script or it is disabled. You can upload images in bulk." and it stops there as it does not seem capable of uploading images without activating Javascript. Now for the sake of experiment and a relatively small risk ( hopefully ) I've activated Javascript temporarily to see if the service works with it activated - I got a warning from TOR which looked like this: http://torimagesbp2vt3u.onion/i/BxtL.png So there's a total limit of 100 mb and each file can't be larger than 1 mb in size. In conclusion this feature is extremely poorly thought out on so many levels there's no point in going into detail - however the concept itself has some merit if it can be reworked into working as an image-hosting service for Vendors/Buyers which automatically strips EXIF-information, however this would also require vastly better integration and some Advanced new Features and I'm not sure there's the will involved to fix this aspect.

Now for the "Forum" which I mentioned was Unique. This is because the Account on the site itself is combined with the Forum, in other words when you register a Buyer Account it serves as a Forum Account at the same time. An interesting notion but it would require the site itself has been built in such a way there would be no way of identifying Users buying and posting in the Forums, or at the very least give Users the option of staying hidden somehow. I will most likely return on this point at a later time as integration is a bold move and as I said before if done incorrectly it could be a huge liability.

Going right along to the "Account" Setting, which looks like this from a Vendor/Customer Perspective Respectively: http://torimagesbp2vt3u.onion/i/YbET.png / http://torimagesbp2vt3u.onion/i/h85B.png - After I've activated my Vendor Account it changes into this: http://torimagesbp2vt3u.onion/i/nxfp.png - So first thing that draws my attention is "Verified account: no", I click the green option "Check account" and I'm presented with the following screen: http://torimagesbp2vt3u.onion/i/MrmZ.png - after submitting: http://torimagesbp2vt3u.onion/i/l703.png If we go back to Account there's one more peculiar option called "Token", this appears to be the Unique ID for me as a Vendor, it's not explained anywhere so I will get back to this at a later date as it might be a potentially major difference.

Now before I proceed I've switched to another account named "Oneiros Distribution" as some screenshots will reveal a bit too much information about certain Bitcoin Transactions. Back to "Account" with the new Account Oneiros Distribution and the options "Withdraw" and "Deposit" ( after creating a new address ) which look like this: http://torimagesbp2vt3u.onion/i/ZNYo.png / http://torimagesbp2vt3u.onion/i/jFHS.png - Noteworthy is the fact there's no possibility of sending Bitcoin internally and the fact the Tumbler Function might not be as advanced due to this fact, I don't have the required knowledge to investigate that Aspect further though so I will leave it at that. I'll leave the remaining Options for another post as that would be a bit too detailed.

So three Options left to go through, namely "Orders", "Messages" and "Help". "Orders" is simply straightforward as can be seen in this image: http://torimagesbp2vt3u.onion/i/83cd.png - this is unfortunately a very basic type, but seeing as how it's programmed it seems it wouldn't be too hard to add new categories so it could look more like for example Atlantis ( reference image for Atlantis: http://torimagesbp2vt3u.onion/i/LLh6.png ).

The option "Messages" is extremely bare boned as can be seen at: http://torimagesbp2vt3u.onion/i/oynh.png - it probably gets the job done but there's much room for potential innovation on this aspect.

Now there's some other interesting Features here which have the potential to go very far as neither Atlantis or Silk Road has anything resembling an API which apparently Sheep Marketplace offers in a limited way so far. I will have to go into that aspect later on though as this post is starting to become ridiculously long. :D
----------
So in conclusion if Silk Road were to suddenly go down tomorrow Sheep Marketplace would probably not have what it takes to take over the role as it's lacking in too many areas currently, however the fact they're trying to change and innovate constantly doesn't rule out they could potentially fix the current problems and come out on top - eventually. So my plan now is to present my thread Upgraded Listing Options: http://dkn255hz262ypmii.onion/index.php?topic=182090 in an alternate customized form which will be written entirely for Sheep Marketplace. If they start doing things right relatively early and take in the suggestions to rewrite the entire system to work entirely differently I think they could have a chance to compete with Silk Road with a technically more Advanced Website which offers more options for Vendors and takes lower commission. We will see how things turn out. :)
