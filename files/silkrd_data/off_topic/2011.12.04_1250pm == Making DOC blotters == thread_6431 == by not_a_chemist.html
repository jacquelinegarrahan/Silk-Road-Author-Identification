Title: Re: Making DOC blotters
Post by: wowzers on December 04, 2011, 08:05 pm
DOC.HCl has much higher solubility in ethanol than acetone, you'll easily get 60mg in a ml of warm ethanol. Add 0.025 ml of this to each blotter square for 1mg/cm2 loading. Invest in a micropipette if you're doing this on a larger scale for accurate dosing, they're fairly cheap on ebay.
If you're just making them up for personal use, you can afford to spend more time preparing them so you can make up a 10mg/ ml solution and slowly add the solution dropwise to each blotter square, leaving the paper under a lamp between each application to dry. ethanol evaporates very quickly under a lamp. DOX's are stable to light so no worries about degradation. If you saturate your paper, capillary action will draw the solution up to the adjacent blotter squares and you'll get uneven dosing so make sure you wait for the paper to dry between applications. Don't soak the whole sheet, there's too many factors which will lead to wildly uneven doses.

have made 25C-NBOMe blotters at 500ug/cm2 using this method.
