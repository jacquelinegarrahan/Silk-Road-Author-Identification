Title: Re: vaping question
Post by: pulpfictionbro on September 20, 2011, 01:03 am
@theone - I have an extreme Q too and at first i used to go from 190 - 220 C (raising temp between bags by 5 C i.e. about 7-8 bags) but not i dont do more than 2-3 bags.

Btw if you want to get thick hits/vapor from the Extreme Q I would advice you to use the bag, elbow pack (ask me if you dont know what that means), and start at 190 C. Do at max 3 bags at 190, 195 and 200 because above 200 the smoke gets a bit too harsh and there is not much left to vape either. I vape till there is no green left and till the weed is like a sort of coffee brown color.

Pro-tip - If using elbow pack method make sure you clean the screen with Isopropyl alcohol before using it for the next session. I didnt clean the screen initially and wasted a lot of weed due to blockage.

Hope this helps..
