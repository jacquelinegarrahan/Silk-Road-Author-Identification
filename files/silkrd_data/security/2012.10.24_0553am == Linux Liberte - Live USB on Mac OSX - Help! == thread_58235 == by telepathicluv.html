Title: Re: Linux Liberte - Live USB on Mac OSX - Help!
Post by: thesearstower on October 24, 2012, 06:07 pm
If you extract the .zip file directly to the USB, you don't need to run the .bat file. Just make sure that the liberte files that extracted from the .zip are in the top level of the USB (i.e. they're not inside of another folder), and you should be good to go. Hold 'option' when you're starting up, and you should be given the ability to choose the liberte USB. I found that choosing 'framebuffer graphics' was necessary in order for it to load properly, and if my cursor went to the extreme right of the screen, liberte would freeze, but YMMV.

Basically, just follow steps 1 & 2 on the liberte 'install' page, and you're in business.

Title: Re: Linux Liberte - Live USB on Mac OSX - Help!
Post by: Aidoneus on October 25, 2012, 07:34 pm
The USB drive you 'restore' onto also has to be formatted w/ a GUID Partition Table, not Master Boot Record (which is typically default). You can change it by opening up Disk Utility > selecting the drive > choosing the 'Partition' tab > Setup your desired partitions/sizes/etc > hit 'Options..." and make sure it's GUID. :)
.Hades.
