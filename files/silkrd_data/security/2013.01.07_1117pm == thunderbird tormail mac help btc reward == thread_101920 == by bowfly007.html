Post by: ragemyfaceoff on January 09, 2013, 06:40 am
I also struggled with this for three or four days straight and finally got it to work mainly with the help of this post...Here are my settings in case you are still struggling...

1. Open the Vidalia Control Panel --> Settings --> Advanced --> Uncheck "Configure ControlPort Automatically" --> click OK
     It should show "Address: 127.0.0.1 : 9051". Restart Tor. The address will remain 9051, and NOT change to 9050

Now open Thunderbird...

2. In the config editor filter type in network.proxy.socks_remote_dns and change it to true by double-clicking (preferences --> advanced --> general --> config editor...)

3. Go to preferences --> advanced --> network & disk space --> settings...
    Click manual proxy configuration:
    HTTP Proxy: 127.0.0.1 Port 8118
    SSL Proxy: 127.0.0.1 Port 8118
    Socks Proxy: 127.0.0.1 Port 9050
    SOCKS v5!!!!
    Click OK

4. Go to --> "create a new account" and enter your name, email address (username@tormail.net), and password, set the protocol as IMAP

5.  Under the "Mail Account Setup" window for this account:

Server Settings
     Server name: jhiwjjlqpyawmpjx.onion
     Port 143
     Connection security: None
     Authentication method: Password, transmitted insecurely

Outgoing Server (SMTP)
add a new server if you have to with the settings...
     Server name: jhiwjjlqpyawmpjx.onion
     Port: 25
     username: (your username) --> this step probably isnt necessary but just in case
     Connection security: None
     Authentication method: Password, transmitted insecurely

Press OK

Once you go to get mail it will ask you for your password but it should work this time
