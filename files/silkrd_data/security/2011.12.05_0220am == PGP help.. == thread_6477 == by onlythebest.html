Using Kleopatra on windows V. cant figure how to decrypt / encrypt messages ?
Kleopatra, by itself, won't be able to deal with the encrypted messages you'll be working with here.  In fact, Kleopatra isn't required at all.

Gnu Privacy Guard [ gnupg.org ] -- GnuPG or GPG, for short -- is what will be doing the encryption/decryption work for you.  If you haven't already, you might want to check out the forum wiki [ dkn255hz262ypmii.onion/wiki/index.php/PGP/GPG_Encryption ] for more information on using GPG.
