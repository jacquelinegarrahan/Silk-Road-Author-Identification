Title: Uninstall iTunes
Post by: comsec on September 23, 2013, 04:34 am
I don't think anybody here would be careless enough to run proprietary software but if you are at least remove iTunes from your phone and laptop/computer. After going through the Wikileaks 3rd release of Spyfiles (http://wikileaks.org/spyfiles3.html) the vast majority of interception LE spyware is still targeting iTunes updates.

I've gone through 12 brochures now where fake updates for iTunes was the #1 selected attack vector for OS/X, iOS and Windows because it's completely silent. Lot's of remote microphone activation for laptop mics too but so far none of these companies, with exception of VUPEN are targeting Linux, most likely because nobody is using it. I imagine that will change

Interesting FinFly ISP mass spyware device is using Debian 5.0 (lol) with SSH passwords, not keys. I gues the Chaos Computer Club was right those guy's at FinFisher really have no idea what they are doing.
