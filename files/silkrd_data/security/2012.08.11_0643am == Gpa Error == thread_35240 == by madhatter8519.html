Hey dude, I made a topic describing the exact same error.  What you have to do is copy the public PGP key into a Notepad document and save it.  Then, go into GPA and select "Import Key", locate the .txt file that you had save, and import.  You will then have a silver key on your key ring. 

Hope that helps!
