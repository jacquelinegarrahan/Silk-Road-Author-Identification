I'm guessing your running windows.  I dont have any experience with virtual box in windows.  Nor did you give details on how you setup your VB machine.  But they are very easy to get going.
what I do first is
click new, choose they type of VB you plan to install then associate enough ram to it.  next click 'create virtual drive now'  i use the 'VDI'  image.  then i use the fixed size option, allow it enough space and wait......

next go to the settings and storage section,  in the dvd/cd drive area click the empty cd then next to the ide secondary tab, click the cd, from there you can load an .iso image and boot the machine and go through the install process of installang  and operating system.
