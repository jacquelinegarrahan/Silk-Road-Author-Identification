Don't use any email service that is based in the US, even if they claim privacy. Just look at what happened to LavaBit.

Use an email service in a foreign country, that offers web-mail that doesn't require javascript (some email providers use the squirrel mail interface for webmail which doesn't use javascript. Always use TOR to access the email account, and use GPG/PGP to encrypt everything.

Your actually probably better off setting up TorChat, or using the private message features on SR or the forums *WITH* PGP.

The Silk Road chat room is also a good place to meet, it's an IRC server on the onion network so it's anonymous and encrypted, plus you can add OTR for private chats to add an extra layer of encryption/security.
