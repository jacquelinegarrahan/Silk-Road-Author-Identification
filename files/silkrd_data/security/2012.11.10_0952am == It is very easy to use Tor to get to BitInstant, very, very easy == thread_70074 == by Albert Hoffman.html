Title: It is very easy to use Tor to get to BitInstant, very, very easy
Post by: Albert Hoffman on November 10, 2012, 09:52 am
       BitInstant, or any other web site for that matter, that appears to be incompatible with Tor, really is not.  Access is blocked because he web site blocks a list of known Tor Exit Nodes.  So, the problem becomes: How do we use Tor to visit BitInstant, or any other site, without BitInstant, or any other site, knowing that we are using Tor?  In other words: How do we use Tor and cloak it at the same time?

       Honestly, I have no idea.  Just kidding, of course.  We cloak Tor by exiting Tor through a Virtual Private Network (VPN) and then connect to BitInstant, etc.  There are hard ways to do this and there are easy ways to do this, we will use the easy way, to wit:

1. Get Tor up and running and then connect to BitInstant, note that you are blocked as usual.

2. Go to the following web site: https://pierce-traffic.cyberghostvpn.com/go/browse.php?u=http%3A%2F%2Fwww.zipzapinc.om&b=1

3. In the yellow box that comes up, paste the BitInstant URL into the white box in the middle.  Ignore any error message that might be above the white box.

4. Uncheck the boxes except the cookie box.

5. Press GO.

       You are now exiting Tor into a VPN and then to BitInstant,  Bit Instant can no longer see Tor.  That's it.
