Title: Re: How to encrypt a PGP message in Tails .20?
Post by: eustacebagge on August 13, 2013, 04:26 am
from the top left:

Applications -> Accessories ->gedit text editor (this is your clipboard)

System -> Preferences -> Password and Encryption keys (this is the keyholder (Seahorse, mentioned above))

once the key is added, you can copy/paste into gedit, then go to Edit -> Encrypt (or Decrypt), pick a key, and you are done!
