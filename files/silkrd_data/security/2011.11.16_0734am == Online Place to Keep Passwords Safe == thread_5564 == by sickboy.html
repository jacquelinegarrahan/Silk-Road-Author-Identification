Yeah, I'm pretty much an online guy, and I use Tails on main forum, which means I have to save passwords and pgp keys somewhere, and I prefer online, tho I also use a lazyman's flash drive...encrypted is best...but like I said, lazyman ;) ....already getting slack...but anyway:

First you kind of need to see how you want to keep your passwords: see I also need a place to keep my onion urls, and usernames, and I prefer using keepassx for this, then just saving a file in their kdb format...now you could use just regular keepass--it might be keepass2 by this time--but my problem is that I use linux--Tails is a linux distro, but for the forums I use windows, and keepassx is the easiest for me....and then I just save all my pgp keys to an *.asc file....and that's pretty much all I need...some good online storage sites are dropbox, wuala, and swissdisk.  I find swissdisk shows up better in Tor, however...dropbox is also way to 'official' and popular and friendly to LE for my taste, so I don't use them for anything 'interesting'...

I need, on a sidenote--since I"m a tails fan, that unless you update tails, you can't apt-get install keepassx

Now if you use just regular keepass--like I used to--you can save/export the files in an html format, I believe, and then just save them that way: so I open up Tails, then immediately login, make sure I"m online, then I can go to my storage site and download the two files I need...you could get them both in one folder if it suits you....all kinds of ways to do this....hell, you could just save your passwords and such as a text file and just use that...and good advice is to encrypt all this too, then save it...also, if you do use bitcoins on your desktop save your wallet.dat too...me, I just use instawallet....

So whenever I first run Tails, I need to get all my stuff squared away, downloaded and get my pgp keys and install keepassx and then get everything and roll...

I usually open a root terminal, and run apt-get update && apt-get install keepassx .... but gotta make sure you are connected online....this is also handy when tails starts needing updates for security reasons...right now they just put out the latest version, so updates aren't vital for security...but I do it anyway, mostly so I can get keepassx...

good luck :)
