Title: Re: gpg what key to i want to select here?
Post by: CrunchyFrog on January 17, 2012, 03:44 am
Quote from: FreeLife1
When I am in the clipboard and select encrypt a screen comes up having me select a public key and at the bottom asks me if I want to sign...  At this point do I want to click the name of the person im sending the encrypted document to?? or click myself?????...
Select the key(s) of whomever you'd like to be able to decrypt the message.  Normally that would be just the recipient, not you.  However, if you'd like to keep an encrypted copy of your outgoing message (that you could later decrypt, if needed) you can select multiple keys; in this case, the recipient and you.

Quote from: FreeLife1
Also I always select the sign option is this a good idea?...
Signing is fine so long as the recipient has (or can get) your public key on her keyring before decryption.  (Otherwise it might cause confusion when GPG reports that it can't verify the identity of the sender.)
