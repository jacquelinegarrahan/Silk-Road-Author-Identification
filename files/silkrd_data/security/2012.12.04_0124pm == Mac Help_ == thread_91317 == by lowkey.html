It's actually quite simple.. Save the public key of the vendor  in a textfile, under the name of the vendor to make it easy on yourself..

Open GPGtools and press 'import', select the textfile with the public key and import it in your keychain.

In a textfile write your name and address like you normally would and select it all.

Use the key combination: command+shift+e to encrypt it, you'll get a menu where you can select which key to encrypt it with and you select the one of the vendor you're buying from :)

That's it.. you will get an encrypted message and you have to copy that into the address box at the checkout page
