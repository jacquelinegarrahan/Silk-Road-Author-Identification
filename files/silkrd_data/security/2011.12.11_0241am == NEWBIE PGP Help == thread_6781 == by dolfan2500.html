Post by: Asbury on December 11, 2011, 03:18 am
To encrypt:
Put somebody's pgp key into a text file.
Import it into kleopatra via file>import
Make a new text file on your computer with a message in plain text.
File>Encrypt>Choose the text file>choose the profile to encrypt to


You probably can figure out decrypting from this.
