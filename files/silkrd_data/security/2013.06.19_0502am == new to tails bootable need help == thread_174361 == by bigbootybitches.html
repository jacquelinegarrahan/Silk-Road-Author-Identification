Title: Re: new to tails bootable need help
Post by: MrGonzo on June 19, 2013, 11:40 pm
click the black terminal in the top of tails

Then type
Code: [Select]
gpg --import in the window

After that paste the vendors key. Next press
Quote
ctrl+d

That's it you've imported the key!  ;D

When you want to encrypt an address simply type
Code: [Select]
gpg -e -a
You'll be prompted to specify the vendors name, use the name given by the PGP Key.

Next you say yes to the prompt then type your address.

If you couldn't follow this here is a more straight foreward one. http://dkn255hz262ypmii.onion/index.php?topic=168825.msg1204023#msg1204023

It's also possible to save the pgp to a text file and import it by clicking the clipboard in the top right corner.

Post by: PerPETualMOtion on June 20, 2013, 12:31 am
1. Ctrl+C the public pgp.

2. Left click on the clipboard icon for drop down menu: "Manage Keys"

3. Ctrl+V (public key imported... voila)

4. Open Gedit... type message to encrypt

5. Click "Sign/Encrypt Clipboard with Public Keys"

6. Select desired key with which to encrypt

7. Ctrl+V encrypted messeage to desired location
