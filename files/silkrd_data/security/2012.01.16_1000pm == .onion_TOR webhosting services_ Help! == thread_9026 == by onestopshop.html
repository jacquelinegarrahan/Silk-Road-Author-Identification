Quote from: supersecretsquirrel on January 17, 2012, 10:31 am
Why not host the site yourself?

hmm..How can I do this + w/ a .onion url?

Get a VPS using bitcoin.
Engage tor
Ssh into your new VPS, and set up Tor
Set up lighttpd or thttpd or nginx
Bind everything to localhost 127.0.0.1
Enable Tor hidden service in torrc file
Go to location on server where tor shows you what sdkjfldf.onion address it generated for you

Cut and paste that into browser voila you have a hidden service.
This of course the most basic of info. You need to

a) secure the VPS by disabling root logins, scp over your ssh login key for secure passwordless entry, set up iptables to only allow tor access, ect.

Check torservers.net there's a guide there on how to setup a relay just tweak it around for a hidden service. You also need to disable server information so it doesn't leak, like removing lighttpd headers or google how to custom build nginx so you can wipe all the server information. May have to make custom 502/404 error pages as well so they don't show your IP 'Error.. this host is down. Host is: blahblah.. derp.. now everybody can find this server'

Just need to make sure you
a) always ssh through Tor to set it up
b) always pay in anonymous methods
c) always use anonymous email not used anywhere else to buy it

So if you fucked up the configuration there's no way to find you. Make sure you aren't running some crazy PHP/SQL server with bitcoind on it either, or not gpg encrypting customer info put into any of your order forms.
