Changes

Notable user-visible changes include:

    New features
        Install Tor 0.2.4.17-rc.
        Install Linux kernel 3.10-3 (version 3.10.11-1).

    Bugfixes
        Remount persistence devices read-only at shutdown/reboot time (ticket #6228).

        Fix the I2P IRC account.
        Allow installing "Priority: standard" packages that we do not install by default.

    Minor improvements
        Additional software: point to the system log on upgrade failure.

    Localization
        Many translation updates all over the place.
