Title: Re: when seller asks you to put pgp in WITH message
Post by: wretched on March 21, 2012, 02:44 am
export your key to a text file. then open that text file. type your message above your key. save the file as message.txt. then encrypt the text file with the sellers pub key. open the resulting file. copy/paste it to PM or address field. done!
Title: Re: when seller asks you to put pgp in WITH message
Post by: wretched on March 21, 2012, 02:46 am
or export your pub key to txt file. type your message in a seperate txt file, and encrypt it with seller's key copy the message cypher and paste it to the PM/address field. then copy/paste your key txt file underneath the message in the PM/address field. click send. done
