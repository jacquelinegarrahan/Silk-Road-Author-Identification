Title: Re: How to find out the name behind IP?
Post by: jochem on March 20, 2012, 03:19 am
You might get some luck googleing his IP and finding other stuff from that IP, which you can than link to a username or something (which in turn might help you end up on Facebook or something). The physical location of an IP can sometimes but determined, but most the time it cannot (especially if it's a dynamic IP). Only the ISP will know which IP is used where and they will not give you this information. Depending on the procedures of the ISP, a first line support guy will probably not be able to look it up (so social engineering is going to be hard). You can try of course, if you come up with a good story.
