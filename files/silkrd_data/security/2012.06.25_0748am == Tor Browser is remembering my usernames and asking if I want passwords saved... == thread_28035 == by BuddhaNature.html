Title: Re: Tor Browser is remembering my usernames and asking if I want passwords saved...
Post by: k1k1 on June 25, 2012, 10:02 am
Your Tor-Browser shouldn't do that from scratch. (We're talking about "Aurora", yes?)
Get your latest tor-browser from: https://www.torproject.org/download/download-easy.html.en

If you're still have the same issues have a look at: https://www.torproject.org/torbutton/torbutton-options.html.en

If all that solutions don't work for you, just always run your tor browser in private mode (do this by hitting ctrl+shift+p or Aurora->Start Private Browsing)

Regular your tor-browser should start in a mode, which doesn't allow it to write to the disc, doesn't matter if you're using windows or linux.

Hope that could help (:
