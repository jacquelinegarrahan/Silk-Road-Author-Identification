The one hop scenario you postulate would not work with the vast majority of Tor clients, which will only accept 3 hops.  There are clients that can be altered but most of them are not running as exit nodes.  The one hop option is meant to be used as an open proxy, not for relaying inside the Tor network.

Of course, nothing can save you from hubris or stupidity.  Can Tor save you from yourself? No.  If used as intended, will it grant you good anonymitization? Sure.

This is not to you can't mess yourself up with tortunnel or advtor if you really try but the vast majority of users will find the defaults "good enough".

As for your guide, it's well enough, but I would urge any *nix user to consider whole disk encryption with LUKS.
