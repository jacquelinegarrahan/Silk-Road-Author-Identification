Title: Re: APG android pgp - Help please?
Post by: Hungry ghost on July 20, 2013, 07:05 am
You'll need a simple notepad app (I use Jota) and obviously a file manager(I use ES file manager).
         Copy the pgp key block of text to your clipboard. Open Jota and paste the key in a new document. Save the document with a name you can find shortly, it will probably save as .txt  file. (Publickey.txt)
Now, using file manager change the name of document from .txt to .asc. Its as simple as that.
        Now open APG>manage public keys>import key. Use the folder icon to browse to where you have left "publickey.asc" (it will likely be in Jota folder, but you can move it to APG folder if you like to keep things organised. Once imported you can delete this .asc  file anyway). Import the key. It should now be added to your list of public keys.
