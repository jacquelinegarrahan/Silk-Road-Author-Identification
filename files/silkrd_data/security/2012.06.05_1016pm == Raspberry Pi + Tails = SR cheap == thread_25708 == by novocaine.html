If you insist on using Windows then I'd recommend Avast as an antivirus. It has high detection rates, good updates, is minimal and doesn't use up loads of system resources, the sandboxing feature is particularly nice (stop applications interfering with one another enabling viruses etc to spread easier) and it has a neat anti virus scan that works before Windows even loads (which is useful as any anti-virus that only scans from within Windows once it has loaded is essentially useless, can't detect rootkits etc).
