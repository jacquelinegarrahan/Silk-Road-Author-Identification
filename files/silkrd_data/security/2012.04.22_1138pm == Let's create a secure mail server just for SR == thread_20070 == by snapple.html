Title: Let's create a secure mail server just for SR
Post by: snapple on April 22, 2012, 11:38 pm
Tormail is on its death-bed, Hushmail leaks like a sieve, and even if we did find a new secure mail option who's to say it's not a Fed honeypot or will shut itself down in a month?

I recognize the need for a mail service that is: secure, anonymous, torified, reliable.

What would it take for us to set up our own tormail? If it were created internally we could (almost) guarantee it's not run by the Feds. If we charged a reasonable yearly fee, payable in bitcoin, this would ensure it would stand the test of time. One bitcoin a year for reliable anonymous mail? I think this is possible.

I've run many websites, and running a mail server isn't THAT hard, however, there are some unique challenges presented in this hostile environment.

Any maverick entreprenuers out there want to take this on?
