Hi,

I am using gpg on linux.

1. create your own key-pair:
Quote
gpg --gen-key

2. create an ASCII dump of your public key, to give away to anyone who wants to send encrypted messages to you:
Quote
gpg -a --export [key-id] > mypublickkey.txt
key-id is the ID you entered when creating the account, mypublickkey will contain your public key in a format that can
be used in the forum, in emails, etc.

3. open a file with your favorite text-editor and paste some public-key into it (you can use mine, see below). If the file is named "X.txt", do:
Quote
gpg --import X.txt

4. to decrypt a file sent to you:
Quote
gpg --decrypt [encrypted_file] > [decrypted_file]

5. to encrypt a file for copy-paste:
Quote
gpg -a --encrypt [cleartext_file]

You will then be asked for the key-ID(s) to encrypt the file for. The encrypted file will be named like the cleartext file, with ".asc" at the end.


There are of course a lot of other things you can do with gpg, like signing documents, etc., but the above is what you will need to start with.
