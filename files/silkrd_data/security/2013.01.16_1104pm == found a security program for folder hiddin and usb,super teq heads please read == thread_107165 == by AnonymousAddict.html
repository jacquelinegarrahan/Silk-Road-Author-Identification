Title: Re: found a security program for folder hiddin and usb,super teq heads please read
Post by: awhiteknight on January 16, 2013, 11:59 pm
I wouldn't recommend it.

Firstly, it doesn't encrypt the data which means your data isn't really protected, it's just hidden out of the way somewhere. Computer forensics software will find it, as will anyone who is looking for it. This is "security by obscurity" and we quite rightly laugh in its feeble face.

Secondly, it's a freeware application. You really need a free software for this sort of job, or at the very least open source. "Freeware" is software that is without charge, given to you as machine code that only your computer can understand. "open source" is software which can be downloaded as human-readable code (for some humans anyway, programmers) which can be compiled into machine code or audited by a programmer. Any security software which is not open source is worthless because its source code is a secret, you must assume that it's doing evil things behind your back; it cannot be trusted. This application relies on the folder hiding method being and remaining a secret.

If you're going to hide files and folders you ought to be using Truecrypt or some other tried and tested, trustworthy solution, even if it looks ugly. Using an app written by some kid to hide his porn collection from his parents is not a good idea, even if it does have a nice looking user interface.

Unless of course you understand the risks and want to use it anyway, that's your call, but please don't use it for anything serious or recommend it to others.

Post by: Hungry ghost on January 18, 2013, 09:56 am
To create an encrypted USB stick you can either install Truecrypt on your computer and use it to encrypt the drive; or put portable version of Truecrypt on the stick and create a container with a hidden volume. That way there is nothing on your laptop, and you could always claim to have found the USB stick in its current state.
            If you have Truecrypt on your computer just go to Tools>Traveller disk setup and follow instructions. Otherwise just download Truecrypt and instead of installing it just extract it to the drive you want to use. 
      Then run the software and go to Tools>Volume creation wizard. Choose "create an encrypted file container", then "Hidden Truecrypt volume" then "Normal mode"
        It will ask you where to put it. Choose your USB stick, next to the Truecrypt folder.
         First you create an outer volume. I just used default algorithm etc. then choose the size of outer volume. This needs to be big enough for both what you want to hide and something you will use as decoy files to put in outer volume ( porn or something innocuous that you might want hidden)
        Create a password. This is the password you can give away under pressure. It will access the outer volume only. Then format the volume.
     You can now put the decoy files in place.
      Next create the hidden volume. This needs to be big enough for your genuinely secret stuff.
This will have a different password.
       Put your stuff inside it.
You now have a USB stick with Truecrypt and a file container. One password will open outer volume. There is no way of detecting the inner volume. If you want to alter the dummy volume without affecting the hidden volume you must mount it using both passwords. 
         When you open the outer volume with one password, the hidden volume appears identical to empty space, so it's possible to overwrite it by mistake.
         The main benefit of this system is there is no encryption software on your computer.
