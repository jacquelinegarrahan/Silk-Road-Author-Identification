Title: Re: SR not Encrypted?? "Your connection to this website is not encrypted."
Post by: angelkiller on August 27, 2012, 08:06 pm
He's doing it right. If you're using the Tor Browser Bundle, there's a box right to the left of the url. It's a dotted line box. It tells you exactly what he typed in the op.

It's normal behavior because that tool is supposed to check for a https link, as you mentioned. Tor doesn't run on the https protocol, so the tool interprets this as a 'non-https' connection and labels that as unencrypted.

Just for anyone's info, to check if you're using tor you can use the following site: (default homepage of Tor Browser)
https://check.torproject.org

If you're connected to Tor, you have a safe connection regardless of what that Firefox tool reports.
