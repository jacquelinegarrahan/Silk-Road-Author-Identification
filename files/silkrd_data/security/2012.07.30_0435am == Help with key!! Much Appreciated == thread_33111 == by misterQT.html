You should send the messages via whatever method is most likely going to be read. Personally, I PM encrypted messages to people. I wouldn't send the encrypted message to the email in the public key unless I had no other way of contacting that person. I know that I, for a time, used a fake email in my public key, so that wouldn't have worked out if anyone sent messages to that email. Now I use a legitimate one, just for funsies I guess, no one ever emails me. Haha.
