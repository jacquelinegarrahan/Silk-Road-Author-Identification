Title: Re: gpg help
Post by: Ktown99 on September 28, 2012, 04:59 am
Ask for the person's public key, copy it then open a text editor and paste the key into it. Save to a secure location and make sure the file name is identifiable to the public key you are saving.

Next open your encryption programme, find the import option. From here locate the file you want to import and press ok. You should see a new name (that of the person who owns the public key).

Title: Re: gpg help
Post by: Ktown99 on September 28, 2012, 05:12 am
write your message - there should be an option called the "clip board" in the PGP programme. Then press "encrypt" this will bring up a dialogue box where you can select what public key to use (this is the key of the person you want to send the message to). After selecting the persons public key you should sign the message - again there should be an option to tick this.

Then copy or paste the encrypted text into your email message. Be sure to include YOUR public key with your message.
