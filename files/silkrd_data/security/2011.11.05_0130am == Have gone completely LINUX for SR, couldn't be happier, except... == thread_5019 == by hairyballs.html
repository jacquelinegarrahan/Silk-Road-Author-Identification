Post by: QTC on March 20, 2012, 03:14 am
Anything running on Windows should be considered unsafe, plus the feds have historically done application layer attacks (against browsers, chat programs etc.) rather than straight traffic analysis. If you are using an OS like Hardened Gentoo or OpenBSD this risk goes down a lot. Search this forum for "CIPAV" to find some documents about this.
