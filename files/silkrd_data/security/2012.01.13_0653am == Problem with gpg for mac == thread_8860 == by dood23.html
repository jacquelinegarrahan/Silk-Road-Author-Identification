Title: Re: Problem with gpg for mac
Post by: CaptKrunch on January 14, 2012, 03:02 am
OK you probably didn't allow the services for OpenPGP yet.

Go to system preferences> keyboard> services> then allow all the services for OpenPGP (there should be 5 of them.)

Then you should be able to highlight your text file, right click and select encrypt. It should let you choose any address that you have imported.

To import an address is the same thing, paste it into a text file, highlight it, right click and select import to OpenPGP.
