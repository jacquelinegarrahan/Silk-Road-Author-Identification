The advanatge to using a VPN with Tor is it prevents an ISP from seeing a particular user is accessing a Tor entry node. Using a anonymous VPN, the ISP will only see the VPN IP addess and not the Tor entry. The problem with this is, as Astor has pointed out, anonymous VPNs provide anonymity by policy meaning they could be persuaded to allow LE to track incoming IPs if presented with a warrant to do so. The other, probably much better, approach is to use bridges to access Tor. The result is the same (internet traffic appears like any other to the ISP) no Tor node entry detected. But with the added security of anonymity by design, not by policy.
