I've downloaded the latest version of Tor Browser Bundle (2.2.32-4) & have had no problems.  Yeah, it's slow sometimes -- last night I had a hard time getting a connection & even then it was painfully slow.  But that's probably an issue with the Tor relay system being overloaded.

Safest version of Tor portable?  It has to be Liberte Linux, which encrypts all of the browser bundle & whatever documents & files you keep on the encrypted part of the USB flash drive. Connections are made over Tor an the developer has even added in a few other security measures not found in TAILS.
