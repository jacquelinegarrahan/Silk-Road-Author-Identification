Post by: mracid on January 08, 2013, 01:54 pm
I would suggest trying a few things to see what is happening with the package / address,
- wait to see if the package comes in the next 2 weeks
- send a few parcels to the delivery address, make them fit the same profile as your ones filled with drugs, but obviously just have them empty
- pay attention to details of the delivery times and tracking information, to see if anything seems off

If all of those come clean over the next couple of weeks, then personally I would consider using the address again
