Title: Re: Can't update TOR?
Post by: Wadozo on August 13, 2012, 06:11 am
Go to the TOR website,  download and save the TOR Bundle to your computer. Remove the version of Tor your currently using from your system and restart it. Then once it boots up, extract the TOR files to the desired location. Then double click "Start TOR Browser" and you should be able to connect. If that doesn't work, you may have some malware / spyware on your system disrupting TOR starting or maybe a registry error that requires fixing with a cleaner (good free one is CCleaner).

You don't completely install it. You click on "Start TOR Browser" and TOR will start and load the homepage. Ideally, look on the forums for info on creating an encrypted  USB stick.

Title: Re: Can't update TOR?
Post by: HEADSHOP on August 13, 2012, 09:40 am
This happened to me!!!  :)

I found the solution was to look very carefully at where I had my previous version of Tor installed. It was on my desktop in a Folder called “Tor Browser”.
However the new installation had for some reason installed inside this folder in a sub folder named “Tor Browser”.

Whenever I used the “Quick Launch” from windows it launched the old version of my Tor browser – which told me that there was a security update available!!! Very strange I thought but continued using it for a couple of weeks.

So the secret was to erase the old copy of the Tor Broswer. Remove the Quick Launch link etc. Then go to the new installation and right click on the .exe file and make a new “Quick launch” etc.

Now my Tor browser works fine, updated to the latest spec.
Hope this helps.  :)
