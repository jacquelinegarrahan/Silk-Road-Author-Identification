I had this problem as well. Assuming you're using Seahorse, the default PGP client for Tails, just copy the vendors public key, open up Seahorse (the clipboard on the upper-right), go to Manage Keys > Other Keys > and paste your key right in there. Control + V and you're done.

Quote from: weedtender on September 05, 2013, 11:29 pm
when i paste key it says invalid format.

Check the key carefully. It MUST have five dashes both before and after BEGIN PGP PUBLIC KEY BLOCK and the END PGP PUBLIC KEY BLOCK.

If even one of those dashes is missing, the key will fail to import.  This alone is the case of 99.99% of key import failures.
