create a file called whateveryouwant.asc and put my public key in that file, then import that file into your pgp client. then create a file called message.txt (or something similar) type your message into that file, and select encrypt in your pgp client. select that file, and follow the prompts to select my key. once you finish, open the new file that was created (i.e. message.txt.gpg) copy the cyphertext from that file, and post it here.
