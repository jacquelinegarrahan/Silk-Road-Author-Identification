This has been discussed many times before. The single most common cause is improper copying/pasting. If even ONE of the five dashes in the -----BEGIN PGP MESSAGE----- or -----END PGP MESSAGE----- is missing, then you will get that error message.

You can also get this error message if the PGP mesage block happens to be malformed. Doesn't happen often, but it crops up on occasion.

If I am not mistaken, you can also get that error message, if the PGP encrypted message is not encrypted to one of your PGP keys.
