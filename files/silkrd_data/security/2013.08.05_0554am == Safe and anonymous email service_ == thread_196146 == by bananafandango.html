Rent a rack in Iceland, or buy a used one and ship it there to be colocated, install openbsd 5.3, run it as a secure .onion mail server with OpenSMTPD. Incoming mail hits a bullet proof domain and then is forwarded to the hidden service over Tor via a slapped together gentoo VPS.
