Title: Re: IP doesn't appear to be changing when I switch Tor identity
Post by: astor on November 24, 2012, 09:29 pm
Tor will keep active TCP connections even when you request a new identity. This is useful for example if you're downloading a large file. You wouldn't want that connection to die, but you want new connections to come from different IPs. The problem is that you're refreshing too soon, when the current TCP connection to check.torproject.org is still active. If you open a new tab and go to check.torproject.org it should be different.
