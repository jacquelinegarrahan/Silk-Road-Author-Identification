Post by: CrunchyFrog on November 10, 2011, 12:24 am
Quote from: ecstasy
...how do I decrypt the public key of another person ?...
You do not decrypt keys -- public or private -- yours or anyone else's.  You use your private key to decrypt messages that were encrypted using your public key.  You use other people's public key(s) to encrypt messages to them.

Quote from: ecstasy
...how do I decrypt a message encrypted?...
Highlight and copy (Ctrl-C) the encrypted text, open GPG, select Clipboard, paste (Ctrl-V) the encrypted text, select Decrypt.  Note: the message must have been encrypted using your public key or it will not decrypt.

Quote from: ecstasy
...how do I write an encrypted message?
Make certain that the public key of the person you're writing to is in your keyring.  Write a message using a text editor, copy (Ctrl-C) the message text, open GPG, select Clipboard, paste (Ctrl-V) the text, select Encrypt, select the public key of the person you're writing to, select OK, copy (Ctrl-C) the entire contents of the Clipboard, paste (Ctrl-V) into your e-mail (or other) application, send.

That's it in a nutshell.

To add on to what Sam said: when you create a key using GPG, you are actually creating both keys, the public key and the private key. You don't really need to worry about the private key as you won't be giving it out to anyone. GPG will know about it. When you go to decrypt a message that someone has encrypted with your public key, GPG will know about the private key and use it for decryption.

Basically, you are pretty much good to go.
