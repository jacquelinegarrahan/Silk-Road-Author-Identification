Title: Re: need link to tor update
Post by: SorryMario on June 30, 2013, 12:27 am
As Astor said, you've probably already downloaded the update (current version is 2.3.25-10), but have not yet installed it, so you're still using the outdated version. Here's the hidden service page for Tor Browser Bundle downloads (avoids clearnet)
http://idnxcnkne4qt76tg.onion/download/download-easy.html.en

After I download a new Tor update, here's what I do:
1. Download the signature (.sig) file for the update
2. Verify the signature using GPG
3. Exit firefox / tor browser
4. Delete the old "tor browser" folder and all of its contents (might not be necessary, but it's what I do).
5. Install the verified update

Now it's ready to go and you can open tor and browse with the new updated version.
