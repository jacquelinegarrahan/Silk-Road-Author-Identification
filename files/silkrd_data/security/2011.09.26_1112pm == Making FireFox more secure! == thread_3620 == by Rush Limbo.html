Title: Making FireFox more secure!
Post by: Rush Limbo on September 26, 2011, 11:12 pm
In case you guys don't know, there's a FireFox plugin that blocks all scripts / plugins websites try to run. Check it out here: http://noscript.net/ or go to Firefox Menu => Add-ons, search and install it.

Reasons why you want this, is that LE can manufacture a website that runs a script / flash etc that will "call home" and reveal your true IP address even though you've got your tor proxy configured. Some plugins do not respect browser proxy settings and can expose your location. Best to block them all!
