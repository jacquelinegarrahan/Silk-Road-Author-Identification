(I hope you mean "GPG4win). You're using GPA, right?

Did you copy their entire key, including the "-----BEGIN PGP PUBLIC KEY" and "END PGP PUBLIC KEY-----"?

It should work if you just paste it into notepad and save as a txt file, then select that file after you hit IMPORT. But if for some reason it won't import the key in a text file, you can try opening the text file and saving it with an "ASC" extension (be sure change the dropdown menu from "text files" to "ALL FILES", otherwise it will still save it as a TXT file).
