So you're using GPG4Win?...

Open up GPA.  Right click on your key that you want to use, select copy, then paste the public key where ever you want.

If using gpp4usb:  Right click on your key that you want to use, select "Append Selected Key to Text"... and now you should see your public key in the text box on the left.  Copy and paste it if you wish.
