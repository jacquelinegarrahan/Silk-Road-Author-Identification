Title: Re: Windows 8 help
Post by: TA on August 15, 2013, 08:39 pm
I GOT IT! Totally figured out the tails usb issue. Press tab on the first greeting screen to bring up the command line. Delete the "live-media=removable" part of the kernel command line. See if that helps.

Some recent USB sticks pretend they're not removable which is not compatible with tails. This isnt the end all solution but it worked for me!
