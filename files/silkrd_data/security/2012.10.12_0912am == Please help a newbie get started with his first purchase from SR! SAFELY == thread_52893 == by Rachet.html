I agree with j3an, and would +1 him if I could.

If your computer is running as normal you haven't booted from the cd. Liberte does not use the hard drive for a start.
I'm using it on a usb so I'm not too sure about any differences with using an iso...But anyway...

You need change the BIOS settings on your computer so that it boots from removable media.

Do this by tapping one of the 'F' keys at start up (depending on the system it could be F5/F8 etc)

There will be an option similar to: "BOOT FROM REMOVABLE MEDIA". Change this to the appropriately named setting.

You will now be able to boot from cd.
