Try one of these

a.  In Orweb, press the options key, then tick enable javascript
b.  Try using the "Transparent proxying" feature of Orbot to route traffic of your favorite browser through tor
c.   Download firefox, get the addon proxy mobile (not for proxying traditionally, but through tor)
     Click use proxy,
     Socks host = 127.0.0.1
     Socks port = 9050
     Start orbot, firefox should now be routed through tor (plus you get a browser with tabs)
     This isn't good security practice, but I'm fairly sure it's not too much worse than orweb.
d.   Use onion.to (worst solution by far imo)
e.  Use VNC to work on your computer using your tablet.  Fairly inconvenient
