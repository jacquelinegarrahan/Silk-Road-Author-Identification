All you're seeing is a progress log of what the software is doing while it's in the process of connecting to the tor network.  The warning you've highlighted is just a notice that the software had to change the permissions (likely the write bit) relating to a directory that the software has to write to. Personally, I wouldn't worry about it, as long as you have verified that the software you're running is genuine, by verifying the PGP signature on the downloaded tor software.
