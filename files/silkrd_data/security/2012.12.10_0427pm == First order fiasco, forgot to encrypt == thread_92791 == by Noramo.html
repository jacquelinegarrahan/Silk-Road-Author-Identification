If the info was sent during the order via SR, then it was encrypted.  The PGP and privnote.com are extra layers of security.  Don't sweat it.

However, some vendors will not accept orders without this extra layer of protection.  If this is the case, the vendor may cancel.  Otherwise, expect your package to be shipped.  Good luck!
