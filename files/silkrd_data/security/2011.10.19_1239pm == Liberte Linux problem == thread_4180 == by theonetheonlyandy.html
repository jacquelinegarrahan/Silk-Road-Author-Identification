It takes some getting used to: I like the persistence, so if I save bookmarks and stuff, it's all left in the encrypted folder, which is nice, and easy to get to, on the flash drive.  I didn't like the browser, too convoluted for me, but I also was using it on a little netbook, and once I got it on a bigger screened laptop, it seemed to be a lot easier to use, also just on top of a lot more ram and stuff....

It's always between Tails and Privatix and Liberte Linux, and I like'em all for different reasons.  Most of my troubles are just a sort of built--in laziness/forgetfulness or something, where I'll just start using my pc, for pga or keepass, which sort of defeats the purpose...one of the reasons I like Tails is it keeps me honest, I know it'll all disappear, so I keep an html file stashed in a cloud storage space, like SwissDisk or Wuala, that I can get as soon as I boot....with Liberte or Privatix, I don't need this...just need to be careful, if I want to keep it to the flash drive I could get rid of in a heartbeat if I needed...
