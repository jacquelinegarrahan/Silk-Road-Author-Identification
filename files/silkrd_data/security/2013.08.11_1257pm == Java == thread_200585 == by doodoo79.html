Post by: deep987 on August 11, 2013, 01:10 pm
I think you're getting java and javascript confused. They have nothing to do with each other.
and also maybe the functionality of https everywhere with noscript.

Noscript blocks javascript on all sites, so long as you enable this. You can then selectively allow javascript on sites you trust. It's a good idea to not trust anything you don't absolutly need to (freedom host, for example).
https everywhere forces given websites to use https (security) instead of http. The fact that it is "enabled" for "java" means when you go to www.java.com it will connect using a secure connection (a good thing).
