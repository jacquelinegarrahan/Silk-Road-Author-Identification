Maybe they've since updated the FAQ:


Quote
Q: What is the green address feature?

A: By selecting this option, your transaction will only use coins from a specific Instawallet address: 1CDysWzQ5Z4hMLhsj4AKAEFwrgXRC8DqRN. By looking for this "green address", others can verify that this transaction was created by Instawallet. This allows for fast verification in situations where the receiver trusts that Instawallet transactions are legit. This feature is currently free of charge. More details can be found in the announcement thread.

Note: The site is currently experiencing problems with this feature. Transactions are sometimes not executed completely in high load situations. It is therefore disabled on the website until this problem is resolved. It is still available through the API, but I would advice against using it at the moment.
