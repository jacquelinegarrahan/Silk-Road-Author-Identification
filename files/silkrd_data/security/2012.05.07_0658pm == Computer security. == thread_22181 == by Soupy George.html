Nearly correct...

A vendor will have their public key on their page.

You copy this key, and import it into your PGP program.

Then you write your message in Word or whatever, and you encrypt it using the vendor's key.

Send the encrypted message to the vendor, and he will be able to decrypt it.

If you want the vendor to send you an encrypted reply, you will need to send your public key to him.
