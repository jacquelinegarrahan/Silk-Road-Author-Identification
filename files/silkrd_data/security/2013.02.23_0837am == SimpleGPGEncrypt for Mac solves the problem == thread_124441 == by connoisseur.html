Title: SimpleGPGEncrypt for Mac solves the problem
Post by: connoisseur on February 23, 2013, 08:37 am
Here is a solution for Mac users having problems with PGP/GPG.
It is a freeware called

SimpleGPGEncrypt

and it does just that.

1) Write/paste text in top field
2) Paste public key in 2nd field
3) click encrypt
4) paste encrypted message to SR address field etc.

You don't need GPG Keychain Access to encrypt messages but SimpleGPGEncrypt does NOT decrypt messages you receive. But it is a useful crutch to encrypt your sensitive information.
