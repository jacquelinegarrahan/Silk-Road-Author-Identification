Title: Re: PGP Noob Please Help
Post by: radium1911 on February 02, 2012, 06:58 am
You're confused about the terms for different keys, so I'll try to explain:

Everyone's key is in two parts - public and private.

When sending a message to a friend, you ENCRYPT the message using THEIR PUBLIC key. They DECRYPT the message using THEIR PRIVATE key.

When they respond to your message, they encrypt it using YOUR PUBLIC key. You decrypt the it using YOUR PRIVATE key.

The crux of this is that public keys can be shared with anyone and everyone. The private key must be kept secret. A different way to look at it: a lock is closed with a key that anyone can use, but only you have the key to open it.

So when you speak of decrypting your public key, that doesn't really mean anything. Your public key is a cipher - not a message.
