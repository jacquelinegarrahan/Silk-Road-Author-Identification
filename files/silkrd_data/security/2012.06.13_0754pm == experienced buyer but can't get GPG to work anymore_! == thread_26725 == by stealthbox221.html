Post by: someone.somewhere on June 13, 2012, 10:51 pm
What're you doing inserting your private key anyhow?  That's for you and no one else.  If anyone gets your private key they can intercept the encrypted messages meant for your eyes only and read it.

1. Type out your message
2. Insert your public key if you want (you can just send this clear text, it shouldn't matter anyway as it's public)
3. Encrypt it using THEIR public key
4. Send it over
5. They decrypt it using their private key

Once they read the message and want to send you something back they do the same encrypting their message with your public key.

Then you decrypt it using your private key.

Title: Re: experienced buyer but can't get GPG to work anymore?!
Post by: someone.somewhere on June 14, 2012, 12:26 am
I use Windows myself.

You don't have to sign it but you can if you want.  Signing it will just sort of prove that the message came from you and not someone else pretending to be you.  Whether or not you sign it will not affect confidentiality.

As for adding recipients, you probably do.  On Windows, I have to add their public certificate to my list and then add them as the recipient when encrypting (in gpg4win this tells it who to encrypt the message for).

You don't send any keys.  Like said before, you can just cut and paste your public key to them as part of the message (encrypted or unencrypted doesn't matter - although sending it encrypted is a bit more secure for various reasons).
Title: Re: experienced buyer but can't get GPG to work anymore?!
Post by: 4903kmn1d on June 14, 2012, 01:18 am
1. Open GPG KeyChain App (Applications -> GPG Keychain Access) and import the recipient's public key

2. Open a Terminal and do the following:
gpg --encrypt --recipient 'recipient@domain' --armour
(Answer Y if gpg asks you whether you want to use the key above, make sure it matches the recipient)

3. Type your message into the window then press Ctrl-D to let the terminal know you have finished typing.

The PGP message should output to the terminal, use CMD-C and CMD-V to copy pasta

4. Delete text file containing the recipient's public key and delete their key from your GPG Keychain.

HTH
