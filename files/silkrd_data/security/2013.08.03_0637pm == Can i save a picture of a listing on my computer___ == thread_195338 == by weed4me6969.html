are you using Tails? - if so, go to File>Print - under General tab, highlight "Print to file", title window should show up showing it will be titled "mozilla.pdf", make sure pdf is selected as output file (on the right), select where you want to save it, desktop is where i put it, then "print"

i usually move it to another backup usb drive before shutting down - if you have adobe photoshop, you can crop out the web address or any other info that prints on the image, or you can print it onto paper and cut it off yourself
