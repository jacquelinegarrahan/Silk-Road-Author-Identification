The problem is that the vendor is likely using one character set, and you are using another.  Some characters won't display properly in that situation.
Unfortunately, there is really nothing you can do about it.

Pretty much.  The only way around it is for both parties to use a character set which supports all the characters of their existing character sets.  The best candidate for that is UTF-8 (Unicode).
