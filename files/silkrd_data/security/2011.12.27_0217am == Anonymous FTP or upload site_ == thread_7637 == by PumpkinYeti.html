There's a long list of clearnet sites at The Upload Guide [ xqz3u5drneuzhaeo.onion/users/uploadguide/list.html ].

The criteria for inclusion in the list are:
Quote from: The Upload Guide
1.  No java,flash or signup needed for up or download of files.
2. Allow min.50mb files
3. Allow 00x parts.
4. Working with "noscript" and "adblock+" enabled.
5. No advanced captcha requirements.
6. Does not change the file name after the file is uploaded.

You can upload to and download from these sites using Tor.  Files will remain on the servers for only a limited time.  Be certain to encrypt (7z or Rar, strong passphrase) anything that needs to remain private.  See the safety guide [ http://xqz3u5drneuzhaeo.onion/users/uploadguide/safety.html ] for other tips.
