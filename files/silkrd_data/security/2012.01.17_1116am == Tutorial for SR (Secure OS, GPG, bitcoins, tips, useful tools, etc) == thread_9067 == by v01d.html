Title: Re: Tutorial for SR (Secure OS, GPG, bitcoins, tips, useful tools, etc)
Post by: v01d on January 18, 2012, 12:22 am
Quote from: John Keats on January 17, 2012, 10:18 pm
This is a great guide! Could you post on how to set up proper disk encryption on the usb?
Could you be more specific? The ~/persist is already encrypted so anything you save to it is already encrypted.
If you want just a general encrypted USB, grab truecrypt:
http://www.truecrypt.org/

Hit "Create volume"

Select Encrypt a non-system partition/drive (if you want to use the whole thing, which I recommend.)

Make a hidden TrueCrypt container.

Normal mode

Select device

Next.

Select what encryption algorithym and hash you want.

Verify that is the correct size.

Create the outer volume password. Make it something decent easy, but still believable to be a password. l0$t_s3cr3t$ or something like that. Make it easy to remember. You might have to give it up if caught.

Select no for storing large files.

Move your mouse around for about a minute.

Hit format and confirm that's what you want to do.

When it's done hit "open outer container" and fill it with non-critical but data you'd seemingly want to hide. Use creativity. Shemale porn, bank passwords, nudes, whatever.

Hit next to continue on.

Next again.

Select encryption and hash for your actual hidden volume.

Select size. Go for the highest you can.

Create as best a password as you can. Long, random, and easily rememberable by you.

Select yes or no to storing large files.

Move your mouse around for as long as you can. For a few minutes at least to build up the encryption.

Format.

You are done.


--------------Tip
Try not to add to the outside container after it's formed without leaving ample room for it.
If you give up your outer password, seem hesitant and beaten. As if it is your last straw.
