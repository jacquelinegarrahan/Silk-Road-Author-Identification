I had the same problem also with my version of GPG Tools for Mac. So. after perusing the forums here, I came across a very knowledgeable poster with the username "Guru" who suggested that people who have this problem download and install the "nightly build" version of GPG for Mac. (Google "nightly build" and GPG). The"nightly build" version of GPG is apparently a more advanced version of the software which overcomes some of the bugs in the standard version. Downloading and installing the "nightly build" version should solve your problem (just as it did for me).
