Post by: LouisCyphre on July 13, 2012, 04:24 pm
Quote from: psilocybinetic on July 12, 2012, 07:52 pm
I have my PGP key

But when I sent an encrypted message to a vendor, he replied saying: "I can't decrypt your messages it shows only this:
pub 2048R/11393796 2012-07-12 psilocybinetic <psilocybinetic@tormail.org>
sub 2048R/8B721BFB 2012-07-12"

This means that you encrypted the message to your own key only.

Quote from: psilocybinetic on July 12, 2012, 07:52 pm
I use GNU privacy assisted key manager.
I write my message on the clipboard
press encrypt....it encrypts the message
I copy the encrypted message, paste it and send
I also sent the vendor my PGP key. Im not sure what im doing wrong so can someone help me?

After you type your message you need to select the keys to encrypt the message to.  You need to include your own key if you want to be able to decrypt that message at a later date and you need to include the public key of the person you are sending the message to (the vendor's key in this case).  A message encrypted to your public key and the vendor's public key will be able to be decrypted by both of you.
