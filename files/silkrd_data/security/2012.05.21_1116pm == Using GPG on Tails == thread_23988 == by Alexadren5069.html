Title: Re: Using GPG on Tails
Post by: fastcat on June 25, 2012, 08:23 am
You need to send the vendor your public key. He uses that to encrypt his messages to you. You then decrypt these messages using your private key.

For anyone else;
system->preferences->password and encryption keys
file->new  follow above settings to generate key
after making key, to export complete key go to properties->details->export complete key
to export public key go to file->export
