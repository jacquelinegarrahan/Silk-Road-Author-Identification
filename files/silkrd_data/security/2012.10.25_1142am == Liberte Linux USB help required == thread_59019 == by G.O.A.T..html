No this is not normal.  I take it you followed the Liberte Linux installation guide? So when you boot up from the USB you get the red hammer and sickel splash screen and then you have to type in your password (which you can't see!!) followed by a carriage return? You might have to do this three times!
Then you get another splash screen with all the icons at the bottom -?
So for Wireless operation you have to set it up using the wireless setup application. You have to enter the SSID of your router first, then select the type of encrytion - either WPA or WEP , then the password, or wireless key. You also have to setup the wireless Mode, either infrastructure or adhoc. You need  infrastructure (I assume you are using a router - because if you are not then you will have to tell me what you are using).
Once you have entered this info, you save it and exit the application. Now you should be connected to the internet and you should not need to enter the password again.
I am doing all this from memory since at the moment I am not using Liberte Linux, but Tor browser bundle on a windows machine. So if doing the above does not help, get back to me and I'll check it out.
