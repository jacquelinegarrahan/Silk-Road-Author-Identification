Post by: jackthetripper on February 09, 2012, 03:10 pm
Well I don't know any tutorial, but I can give you a quick rundown and try to answer any questions.  First dl gnu4win, my favorite is this one which has the Kleopatra interface, at http://www.gpg4win.org/.  Then install and set it up with a certificate which will create your public and secret keys.  It will be in the top menu under "new certificate".  I picked Open PGP.  It will give you the option to save your public key to file, and you can export your secret key as well (although make sure you label it something obvious so you don't send it out).  If you cannot save a file as .asc (which I could not at first) you may need to google "save as asc" and dl a patch for MS Word.  It will then be part of your save menu under "MS-DOS text with layout".  You will need this to encrypt/decrypt a file because you will select the appropriate option from the menu and then the file you which to encrypt/decrypt (which will be saved as an .asc file on your computer).  Then dl Mozilla Thunderbird and set up your email account with the wizard, it will automatically locate your gnu file and import your certificate.  When you use the program Kleopatra, there is an option when encrypting that says "ASC armored," you need to choose that so it will be in a format that you can copy/paste it into a Silk Road message.  These are the issues I ran into so maybe they might help you.  If you have any questions post back here and I will try to answer.
