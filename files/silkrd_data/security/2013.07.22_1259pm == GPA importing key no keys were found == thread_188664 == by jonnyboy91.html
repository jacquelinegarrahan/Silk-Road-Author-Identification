Not sure if these will help but here are two that I have heard/seen:

1. Delete the 'version' line and try again:
-----BEGIN PGP PUBLIC KEY BLOCK-----
Version: GnuPG v2.0.20 (MingW32)

hQGNBFR9MEwBDAC8QHyuRBLkwEBWsRoXoJhfB8e+E97


2. Near the end of the key, between where those = signs are, it seems the carriage return/linefeed can screw things up. Sometimes deleting the newline and pressing enter can fix it:

HslkHrIPTivi6xy7xPKS+d11UfA=
=oN+/
-----END PGP PUBLIC KEY BLOCK-----
