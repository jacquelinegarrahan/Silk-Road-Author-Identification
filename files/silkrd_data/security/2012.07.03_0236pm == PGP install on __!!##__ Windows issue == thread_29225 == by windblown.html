Post by: Hungry ghost on July 03, 2012, 07:54 pm
Hi, you don't need Kleopatra, I couldn't work out what it was for anyway! Go to control panel/ add remove programs and remove the whole GPG4win package. Then reinstall it but this time wehen you get the tick boxes asking what components uncheck everything except GPG( you can't uncheck it anyway) and GPA. GPA is a nice simple interface, you can import keys from files or clipboard and just copy paste your messages in and out of  the window.
A better alternative is GPG4USB. It is designed to be a self contained unit run from a USB stick ( so wont leave unwanted evidence lurking in your computer) You can put it on a thumb drive with TOR browser bundle and you have a handy set up that is easier to conceal/ destroy than a laptop.

Ideally you will create a Truecrypt hidden volume on the thumb drive along with the truecrypt software. This means the Tor and gpg software will only be accessible if you mount the volume using your password. If forced to give up the password you reveasl the dummy password which unlocks the outer volume containing some gay porn or something. The inner volume is undetectable .

In the meantime I'd go for either GPA or GPG4USB definitely.
