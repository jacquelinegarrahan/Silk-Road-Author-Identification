Title: Re: Can anyone help with the pgp? sending!!!!
Post by: cache on March 21, 2012, 09:50 am
I use the Linux version, others will be similar.

You need to open a txt file and copy/paste your vendor's key into that and save it somewhere.
Open your pgp program and select Import Key.
Navigate to the txt file you created earlier  and import that.
Go into the clipboard and write your message, include your public key if you want the vendor to be able to reply encrypted to you.
Click encrypt and select the vendor key that you just imported.
Click OK to the warning message.
Copy/paste the encrypted text into the message/email to your vendor.
