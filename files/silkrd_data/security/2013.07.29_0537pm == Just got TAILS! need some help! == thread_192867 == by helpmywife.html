Post by: Rastaman Vibration on July 29, 2013, 06:47 pm
To set up PGP keys, look at the top right and you'll see a clipboard icon. Click on it and a menu should pop up. Select Manage PGP Keys from the menu. From there you can import others' public keys or generate your own personal key pair. (Under File menu)
