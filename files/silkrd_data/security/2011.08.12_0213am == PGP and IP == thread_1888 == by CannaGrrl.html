Title: Re: PGP and IP
Post by: CrunchyFrog on August 12, 2011, 05:33 am
There's really no need to publish the key you'll be using for business on SR (or similar sites) to a public keyserver.  And if there's no need, why run the risk?

Many people have chosen to post their SR-related keys here ( http://dkn255hz262ypmii.onion/index.php?topic=174.0 ) for use by others in the community.  Doing so isn't quite as easy as clicking a "publish-to-keyserver" button, but there's also much less concern over anonymity issues.

(Be careful to post only your public key in this fashion, however).
