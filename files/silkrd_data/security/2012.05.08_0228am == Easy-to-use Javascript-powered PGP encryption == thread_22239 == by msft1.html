Title: Easy-to-use Javascript-powered PGP encryption
Post by: msft1 on May 08, 2012, 02:28 am
I thought I'd share what I've recently found when looking for PGP tools:

http://www.hanewin.net/encrypt/PGcrypt.htm

It's a Javascript-powered PGP encryptor. Save this webpage somewhere on your encrypted volume and use it to encrypt your address when ordering. Saves the trouble of having to install PGP tools and import vendors' keys, way less risk.

The guy also provides a decryption algorithm but I haven't tried it yet.
