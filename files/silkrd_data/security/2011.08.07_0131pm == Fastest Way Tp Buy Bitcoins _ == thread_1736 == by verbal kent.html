sending western union to a foreign liberty reserve exchanger is probably the safest and fastest way to do it. there are fees, but the security of it makes it worth it in my opinion. even if they did try to trace the btc, it would be a very long time before they made it all the way back to the western union shop that you used, possibly enough time for the security tapes to be overwritten.

this works both ways. if you have the foreign liberty reserve exchanger sending you cash with western union, a security question and answer will allow you to pick it up with no ID.  there are limits on the amount you can pick up without ID, so find out what they are before you do this with large amounts.
