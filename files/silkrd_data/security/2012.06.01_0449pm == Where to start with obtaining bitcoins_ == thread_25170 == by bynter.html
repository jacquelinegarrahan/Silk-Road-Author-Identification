Title: Re: Where to start with obtaining bitcoins?
Post by: Chernovy on June 01, 2012, 05:45 pm
Paypal to virwox is your best bet.  I have used it before.  You do lose some of it in the transfer because Virwox only allows whole number transfers.  There are also other restrictions with virwox that you can read up on bitcoin's web site.  If you are planning to use any transaction that can be traced back to you make sure you hide your trail to SilkRoad by bouncing between wallets/fogger and SilkRoad.  If you buy coins at Virwox and send them straight to SR it could be red flagged.
