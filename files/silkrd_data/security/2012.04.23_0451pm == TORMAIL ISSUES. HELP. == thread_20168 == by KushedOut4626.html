UPDATE: A Gmail account just got frozen. This happens if the account is usually accessed from the same area (IP range), then suddenly from a different IP, and phone verification is set.
If the IP range or the area I use to access the acc is similar or close by to the usual place, there is no problem. If it's different, or it's a known Tor exit node, then there is a verification process.
In case there was no security questions or phone validation set on the account, you need to tell the name of the area where the account was accessed from in the past.
In case of security questions, those needs to be answered.
But in case of phone validation the acc get frozen, and you need to verify it via phone to open it up again.

Most exit nodes are blocked anyway, I was unable to open an acc without a mandatory phone verification.
I had no problems with my past accounts though.
