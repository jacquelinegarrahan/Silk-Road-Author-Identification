Open a Terminal/shell/DOS prompt and type:

gpg --version

Wherever the "Home:" location is is where the secring.gpg and pubring.gpg files are for your operating system.  If they're all safely backed up then destroy the contents of that directory with the file wiping utility of your choice (options vary according to operating system type).
