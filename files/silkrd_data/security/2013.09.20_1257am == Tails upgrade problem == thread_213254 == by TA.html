I don't know if it is the fastest method with 2 USB sticks. But it worked :D

-I copied tails.iso on fresh usb(1) stick (format it if it was tails-stick otherwise windows doesn`t read it)
-copied tails.iso on persistent drive on old tails (usb(2))
-installed tails by upgrade from iso to usb(1)
-run tails from usb(1)
-Clone&upgrade on usb(2)
perfectly running usb(2) with tails and untouched persistent
