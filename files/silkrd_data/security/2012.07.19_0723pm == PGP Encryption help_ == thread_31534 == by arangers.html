Title: Re: PGP Encryption help?
Post by: TurboSti on July 19, 2012, 07:31 pm
I use gpg4usb as it seems most people do. but someone will post their public key in which you copy and paste it on the blank document in the program. Up in the header is a button that says copy which it'll move it to a clipboard and then you can select IMPORT which will show that person's public key on your key ring on the right hand side of the screen. If both you and that person have each other's public key, you can then type a message and press ENCRYPT. Post this somewhere or message them with whatever you say and then they can use their private key to decode it. Hope this helps and makes sense : )
