Title: Re: How do I make my macbook safer?
Post by: QTC on March 23, 2012, 12:03 am
Using a hardened live Linux distro geared towards anonymity is on balance better than trying to harden a Mac. There are packages like Liberte and amnesia that make this relatively easy to do. And if you are attempting to disguise Tor use from your ISP you should look into using obfsproxy. There is a Snow Leopard/Lion obfsproxy bundle here: https://www.torproject.org/dist/obfsproxy/tor-obfsproxy-browser-bundle-2.3.12-2-osx.zip
