Dear all,

As those who have am currently working with another seller (who is based in the US) on writing a guide for devices running Android 4.0 to use SR.

Specifically our goal is to write a step by step guide on how to install the necessary applications such as Orbot, Orweb, Bitcoin wallet software (I'm thinking BitCoinSpinner) and of course securing the device through using the built in encryption option. I would like to avoid having to "root" the device if possible as from what I've seen this would be difficult for people without strong IT skills but if you think there's a strong case for it please let me know.

We're confident we can write the basic instructions ourselves but we would like to run these past more experienced users in our community to make sure they represent the best way to secure an Android device. We also believe an open and honest approach like this will

The reason we are doing this is that we hope to start selling Android Tablets via SR in the coming weeks and want to encourage users to secure them themselves. This prevents any accusations of tampering and encourages SR users to take responsibility for their own safety.

I would be more than willing to pay you a consultancy fee (I'm thinking around 5 Bitcoins, please let me know if you think this is fair) - naturally if you agree the guide is suitable for the purpose for which it is intended then all you need do is say so and we will list your endorsement in the "Acknowledgements" section.

If (as I suspect) you have further thoughts of your own then I'd ask you to edit the draft as you see fit and return it with your comments.

 I am aware there are other ways to access SR safely and I also know that an Android device properly secured would be a convenient way for many newer users to do so, especially as the OS and the Apps we plan to use are Open Source.

The first draft of the guide itself should be ready in around ten days. We plan to start offering Android devices for sale in the next 2 - 3 weeks.

If any member of the SR Geeks interested in being paid for their expertise and their endorsement, please send me a message or e-mail me at vlad1m1r@tormail.org.

Many thanks and I look forward to hearing from you.
