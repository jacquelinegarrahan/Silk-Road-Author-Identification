Post by: Raoul Duke on June 13, 2012, 08:25 am
it's the same as windows - it keeps the history somewhere hidden in the user directory. the only ways to clear it out properly is either find exactly where the history is kept (https://en.bitcoin.it/wiki/Data_directory), stick it in the trash and do a 'secure empty' trash, or erase and install. TBH if you are that worried i'd do an erase an install, you can pretty much guarantee it will have put another give-away file somewhere else on the computer
