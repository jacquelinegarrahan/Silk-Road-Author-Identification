Title: Re: GPG Tools Encryption Issues with Mac OS X
Post by: Diamond on March 28, 2012, 12:37 am
When you are encrypting a message, a dialog window pops up and asks you to choose a key public key for encryption.

There is also a menu on the bottom with two options "sign" and "add to recipients", this is looking for a private key. If you have not generated a key pair then this will not be available to you, so make sure those options aren't selected.
Title: Re: GPG Tools Encryption Issues with Mac OS X
Post by: klaaat21 on March 28, 2012, 12:51 am
If you have installed gpg tools then it should work within text edit.
First you copy someones key into text edit then you highlight the key and right click and there should be a menu option to "import key"

After you have imported somones key, you go into text edit and type them a plain text message.  You can then insert your key at the bottom of the message.  Then highlight the whole thing, right click and there is an option to encrypt.  Encrypt using the person's key that you are sending it to.  After it is encrypted just copy and paste it into a message

If you want to try a test, here is my key. Copy and paste my key into your text editor, then right click and import my key.  Feel free to send me a message along with your key and I'll send you one back.
