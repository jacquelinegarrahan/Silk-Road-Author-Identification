Title: interesting development for android tablets.
Post by: Hungry ghost on June 21, 2013, 12:57 am
     I have been experimenting with custom ROMs on an android tablet, and now I have discovered it is possible to have a dual/multi boot using software called "Multi boot" and a modified TWRP recovery.
         You can boot into various versions of the android ROM, and also Ubuntu and other distros of Linux.
         The interesting thing is that you can also boot from a USB drive; this obviously set me thinking that it might be possible to get Tails or Liberte working on an android device. I imagine there would be particular issues, that are probably beyond my competence. Setting up the multi boot with android ROMs is straightforward, but getting the Linux distros working is a bit more difficult, requiring patching of kernels and whatnot and obviously you need more than my fairly superficial knowledge.
       However this does raise the possibility of a USB based impermanent ROM for higher spec android devices.
         Please note before the avalanche of " well I personally would never use an android device to browse SR " replies, I AM NOT ADVOCATING THIS, just saying, its interesting.
