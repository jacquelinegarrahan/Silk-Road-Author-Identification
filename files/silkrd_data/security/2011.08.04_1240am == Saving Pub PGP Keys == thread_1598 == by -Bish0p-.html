Title: Re: Saving Pub PGP Keys
Post by: Kind Bud on August 04, 2011, 02:56 pm
Getting a hold of your PC should not be helpful to LE at all.

Whatever computer you use for Silk Road should be secure. At the very least get truecrypt but also look at Tails Linux or a linux boot with encrypted USB drive for bookmark storage. Passwords use keepassX (linux) or keepasss2 (windows) on the truecrypt encrypted drive.

Basic scheme is boot off static CD or USB, and use truecrypt for any data you save. There are other options, but this will give you great security and is easy to set up.
