Title: Re: Liberte USB setup on MAC- can I use it on my MacBook PRO after setting up on PC?
Post by: thesearstower on October 20, 2012, 08:09 pm
The most recent version of Liberte will boot on a Mac without any additional tweaks. No need to set up on a PC. In fact, that might even cause it NOT to work. Follow the instructions here:

CLEARNET: http://dee.su/liberte-install
(Pay attention to Instruction #3.)

After following the directions, just plug in the USB, hold the 'option' key as your Mac boots up, and select the Liberte USB.

When getting it working on a friend's Mac, I found that I had to select 'Framebuffer Graphics' at the boot screen, otherwise the boot sequence would hang. Also, after it booted,i found that if I moved my cursor to the extreme right of the screen, the system would freeze. My keyboard backlight also cycled on and off strangely, but Liberte worked fine.

I doubt that the same stick will work on a windows machine, unless that hardware uses (U)EFI to boot, instead of the more common BIOS, which is very uncommon.
