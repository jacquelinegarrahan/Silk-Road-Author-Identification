Title: Re: No will answer this question?
Post by: Bob Arctor on January 23, 2012, 08:07 am
hello, Heyenezz

delete your key files in C:\Documents and Settings\...\App\.silc\ if you are using pidgin, and setup everything again. Be sure the password matches the password you originally set before replacing the default keypair. If you left it blank, you'll need to leave it blank in the log in as well otherwise it'll give you this error.

hope you can follow this:)
