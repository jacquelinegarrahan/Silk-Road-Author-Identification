Title: Re: PGP..... Lots of help needed
Post by: Moldybread on November 14, 2012, 04:08 pm
It really isn't that hard.

first, you have to import the vendors public key. Copy it from his vendor page and save as .txt file. Then, in Kleopatra, click on file/import certificate (or simply press Strg+I) and select the .txt file.

Now you have to encrypt your message. To do this, save the message you want to encrypt as a .txt file, and in Kleopatra click on file/sign or encrypt file and choose the .txt you want to encrypt. In the next window Kleopatra asks you what you want to do. Encrypt the file only, do not sign it. Also it is important that you select the field where it says output as text (!).
In the next window, choose your vendors certificate you have imported before.
The encrypted message is saved as a .txt.asc file, open this file with text editor or notepad, it contains your PGP encrypted message.


(Its possible that there is a faster way to do this, but this has always worked for me.)
