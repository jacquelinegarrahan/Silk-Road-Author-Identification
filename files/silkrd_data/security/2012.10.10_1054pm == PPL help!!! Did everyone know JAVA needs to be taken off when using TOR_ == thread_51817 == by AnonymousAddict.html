Post by: pine on October 10, 2012, 11:10 pm
Two things.

1. Java is not Javascript. They are both languages used for various applications/services on the web, which makes things confusing.
2. Your friend is correct.

You can disable Javascript in the Tor Browser Bundle which I imagine you're using, by going to the TorBrowser button -> Options -> Untick "Enable Javascript".

Note: some websites will become non-functional as a result of this. But not SR or SRF, which is what you want.

Java, which is inside many web browsers, is disabled, as is Flash, with the TBB anyway, so you don't need to worry about that so much. And never use Privnote! Horrible idea that.
