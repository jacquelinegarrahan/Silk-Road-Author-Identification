if you use linux then exiftool is great.  tails also has it already installed.
in terminal type  exiftool -all= *.jpg       and it will strip all the data out of every .jpg in that folder.  nice for multiple images.  it will also make a copy of each image.  secure erease is another neat tool to over write your data to help hide anything was ever there.   srm *.jpg*     will erase them all :)     (i should note secure erasing works differently on usb flash and ssd)

Title: Re: Uploading pictures
Post by: astronomicat on May 19, 2013, 07:18 pm
Like someone said earlier, if you use tails there is an application included in the bundle that helps you with this. Go to Applications -> Accessories -> Metadata Anonymisation Toolkit. This is very easy to use as you just add files into the list, click "Clean", and let the application do the rest. New files will be created but you can simply remove the old files. This is especially important if you use a smartphone camera as most of the current ones tend to include geolocation, which is probably among the worst giveaways.
