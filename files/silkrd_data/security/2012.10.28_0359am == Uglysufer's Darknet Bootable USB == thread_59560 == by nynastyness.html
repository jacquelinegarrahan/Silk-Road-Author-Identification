Title: Re: Uglysufer's Darknet Bootable USB
Post by: Nakorx on October 31, 2012, 01:22 am
There is more to one's choice of implementation or "platform " than just security.  For example, some people value plausible deny-ability, and if using TrueCrypt, they set up two containers, one of which has their SR stuff and another that has something like banking documents and other private papers. Then if they are forced to reveal the TrueCrypt password (which they can be in some countries by LE etc)  they can offer up the password of the latter and  so keep the SR container hidden. Other people like the Liberte Linux route, which has an encrypted container already set up. I don't know about TAILS, other than many people swear by it (bloody TAILS!)
As far as I know neither are hard to setup, you can find all the information on the clearnet using Google (or whatever search engine you prefer).
