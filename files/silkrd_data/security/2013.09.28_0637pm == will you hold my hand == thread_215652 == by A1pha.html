Post by: Rastaman Vibration on September 28, 2013, 07:42 pm
Its realy easy to use PGP with Tails. If you look at the top right, you'll see several icons. You'l see a clipboard icon. This is you PGP app. Click the clipboard icon and select Manage Keys. From there you can import keys, like your own key pair or a vendors public key you want to use to encryot a message.

To actually encrypt a message, open up the text editor (Applications>Accessories>gedit) and type up a message. Then look under the edit menu and you'll see an option to encrypt. Select it, and you'll be prompted to choose a key from the ones you have already imported. Select it, hit OK and you're good to go.

It's really not that hard. Forget everything I've said so far.

In TAILS, press alt-F2. Type Seahorse. Click file (at the top) and select New. Select create PGP key. Put fake information in all of the fields. Click create and choose a password that you can remember. Now you have a public and private keypair.

When you want to buy from a new vendor, copy their entire pgp key (including the parts where it says begin/end key). Then do exactly what you did above to open Seahorse. This time, however, click import and select the file you've just saved. Now you've imported the vendors key to your keyring.

At the top right of your desktop, click applications, accessories, and select gedit text editor.

Type the text in gedit. From the menu at the top of the gedit window, select edit followed by encrypt. Select your recipient and you're done.

Sorry didn't realize you were that new to this. You don't need to install anything to do this. Everything in these steps is preinstalled. And I just tested this in a VM, so I know it works.

If you have more questions, I can walk you through it. It really is very easy once you learn it!
