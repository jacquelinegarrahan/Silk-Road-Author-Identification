Title: Re: How safe is using a non-Tor browser while using tor
Post by: theotherotherside on May 24, 2012, 06:45 am
Routing a non-tor browser through Tor only protects you in some ways. Sure, for the most part, you'll appear anonymous to your destination, but you're open to more attacks. For example, vulnerable plugins which may expose your identity such as flash or java will still be active. Also, the browser may leave more traces of your activity on your computer.

Basically, why take the risk? Take the time and set up a Virtual Machine or preferably USB/CD based OS such as Tails or Liberte.

Title: Re: How safe is using a non-Tor browser while using tor
Post by: QwertAnon on May 24, 2012, 10:04 pm
I'm doing it too.

Well, a possible vulnerability is accessing the same server from both instances (e.g. being logged in to youtube in normal browser and googleing in tor browser).
The Tor devs warn that if you do it and your internet breaks down both connections stop at the same time and the server can correlate both users. But that attack is probably not likely to uncover the average SR user.

In case you're - for whatever reason - not using tormail you might accidently log in to your account over clearnet because you're in a "i'm protected nothing can happen to me - oh shit wrong browser window" mood.
