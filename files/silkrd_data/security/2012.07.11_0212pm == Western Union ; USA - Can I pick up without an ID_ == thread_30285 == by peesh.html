Title: Re: Western Union ; USA - Can I pick up without an ID?
Post by: RxKing on July 11, 2012, 04:19 pm
YES-- But you need more then just the MTCN. You need to have the sender make sure and tell the agent you have NO ID and to mark a box that no ID is needed(the box is on their computer screen, not on the paper the sender fills out). That is important... Just because the sender puts down the question and answer does not automatically mean pick up with NO ID is ok, the agent has to check the box, I would say 50% of agents don't do it. The receiver MUST check after it is sent to make sure the agent did it right. Then the sender has to put down a question and an answer.

So you need a question,  answer, MTCN, City and State the money was sent from, and name of sender and receiver.
