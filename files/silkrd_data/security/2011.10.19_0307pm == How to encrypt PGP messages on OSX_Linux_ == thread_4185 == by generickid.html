To import my public key copy paste it into a text file and type:
gpg --import /link/to/your/textfile.txt

than you can type
gpg --list-keys
This will list all the keys that you have including yours.

Now, to send me a message make another .txt file and type your message into it then go to the console and type:

gpg -u [YOUREMAIL] -r sun@sun.com -a -s -e [FILE]

This will encrypt the .txt file you wrote and armour it so that its in a format that is copy-pastable text.
