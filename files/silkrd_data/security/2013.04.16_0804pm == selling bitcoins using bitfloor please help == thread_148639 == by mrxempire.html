Title: Re: selling bitcoins using bitfloor please help
Post by: cantharidin on April 16, 2013, 08:32 pm
The only option I know of to get cash is to have Bitfloor do a direct deposit USD to your bank (perhaps this only works in the US??). This requires that you set up ACH with Bitfloor and send them a copy of a picture ID. The option to do this is on the 'Withdrawal' page.  When I set up with Bitfloor, I could not find this option, but then discovered that there are two buttons toward the top of the Withdrawal page. The default is BTC, which is why you don't see any $ option. Just click the USD button, and all of the process is laid out.

This will take a while while Bitfloor sets up your automatic bank transfer but once you have it set, I'd expect that it would be pretty efficient. I'm curious abotu what fees Bitfloor charges. One of my issues with the whole bitcoin system is that, right now, eveyrone grabs a piece of your coin everytime you transfer anywhere.  Bitfloor would at least be only one step each direction. This assumes you don't mind sending ID - if you are just speculating, then why not? Even for Silf Road use, a pass or two through Bitfog should suffice as long as you're just a casual user.
