Correct. The easiest way is to first burn to DVD, then once u start TAIL, Application > Tails > Tails USB Installer. This will create the USB boot for you. From that point on, you can destroy and discard the DVD and boot from USB.

For upgrades, you have a few options. The easiest is having a spare flash drive to download the new TAILs on. Using the same "Tails USB Installer' as mentioned up, you can Clone & Install, Clone & Upgrade or Upgrade from file.

USB isn't as safe as just a DVD. USB drive can be write to where DVD can not. The USB seem much fast than CD. Tradeoff.
