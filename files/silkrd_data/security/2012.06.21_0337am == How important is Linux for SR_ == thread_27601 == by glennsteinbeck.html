It's not that Linux is more secure than Windows or OSX, it's more about the ability to boot a lightweight and encrypted OS from a thumb drive. If you wanted a more secure OS then there are plenty of other options, I would recommend FreeBSD. As much as I despise Macs and Apple in general, I will say that OSX is more secure than mainstream Linux distros, but time will tell if OSX will still be as secure in the future as it's replacing much of it's BSD roots with every release.
