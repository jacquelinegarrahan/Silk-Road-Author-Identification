Title: Re: can I transfer my PGP keys from windows to Linux?
Post by: eJ3k1 on July 22, 2012, 02:13 pm
Definitely. You need to export your private keys. Then on the other computer, you need to import those private keys. Then you can use them, providing you know the passphrase.

Note that the private key is considered sensitive information. Don't just email it to yourself. Either put it on an encrypted volume and transfer that. Or create a new key pair on the computer where you want to use the keys, then email the public key to yourself and import it on the other pc (the one that contains the keychain). Then email it to yourself encrypted with your public key.

And yes, always encrypt your address using GPG. If there is a leak in TOR security (not likely, but not impossible either), then you're address won't be out in the open.
