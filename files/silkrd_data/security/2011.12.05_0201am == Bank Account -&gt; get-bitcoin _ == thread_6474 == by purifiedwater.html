With your bank, yes.  With GetBitcoin, not necessarily.

Using your real name might help if there were a problem with the transaction.  Buying BTC is not illegal (yet), so using a false name might look "odd" if the transaction were to come under scrutiny for some reason.  If the amount is over $US1000 -- which may make it more likely to attract attention -- I'd think a real name would be best.

Tumbling in an insta-wallet before moving to SR would seem a wise precaution as well.
