Title: Re: Key not showing up in recipient list (Mac GPG Keychain Access)
Post by: Chopperos on July 19, 2012, 11:34 am
Hi,

I find it easier to use TextEdit.

1.) Copy and paste your Vendors public key into TextEdit.
2.) Save the TextEdit file as Unicode UTF-8 format
3.) Goto GPG Key Chain access
4.) Click import.
5.) Select the file from Step 2.
6.) Type your address in TextEdit and highlight it.  Right click, select Services, Encrypt and then select the Vendors public key.
7.) Text will become ciphertext and then paste into SR.

I find this really easy.  Hope that helps.
