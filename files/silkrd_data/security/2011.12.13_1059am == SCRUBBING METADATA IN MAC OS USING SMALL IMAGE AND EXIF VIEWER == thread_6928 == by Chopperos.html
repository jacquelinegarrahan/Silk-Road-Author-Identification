Title: SCRUBBING METADATA IN MAC OS USING SMALL IMAGE AND EXIF VIEWER
Post by: Chopperos on December 13, 2011, 10:59 am
Found 2 great new programs for viewing and erasing meta data using images in Mac OS.

The first program called EXIF Viewer allows you to view what EXIF data is stored within the metadata attached to your image

The 2nd program called Smallimage allows you to clean all the metadata in your image and resize


Steps:

1.) Install the 2 pieces of software above
2.) Drag the .jpg image you want over the EXIF camera icon in your Dock
3.) Drag the the .jpg image you wish to clean into the open box of the SmallImage software
4.) Select the necessary metadata to clean
5.) Click process
6.) Duplicate copies with a new suffix will appear to identify which images have been cleaned
7.) Double check and for peace of mind.  Drag the new cleansed image into EXIF Viewer and confirm all meta data has been erased.
8.) Upload your clean image into Silk Road.
