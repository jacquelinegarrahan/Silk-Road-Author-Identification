Title: Re: Tails Uses Iceweasel To Connect To The Tor Network, Is this Safe?
Post by: Tessellated on June 22, 2013, 04:47 pm
Tor browser is just a modified version of firefox. Iceweasel is just firefox without the trademarked name.

TAILS implements the same modifications as the tor browser bundle but also provides special firewall rules that prevent any attempt to connect directly to the internet.
Title: Re: Tails Uses Iceweasel To Connect To The Tor Network, Is this Safe?
Post by: leaf on June 22, 2013, 04:52 pm
TAILS uses firewall/network level restrictions as opposed to user application restrictions.  It forces Everything in the application layer through Tor, minimizing the room for error.   This makes the TAILS environment much safer then most other options.  Another good option, if you like keeping your own OS is installing Tor as a middlebox.
