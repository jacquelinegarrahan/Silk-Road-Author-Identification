Title: Re: Buyers - Using multiple throwaway user accounts?
Post by: zenvoboy on November 25, 2012, 09:05 pm
Probably overly paranoid. As long as you make sure your method of getting bitcoins into your SR account is safe enough, you should be fine. Vendors I'm sure are a bit weary of first time buyers; they never know if you're LE or not. Keeping the same account would build trust between you and the vendors, and if you're buying from somebody you've not done business with before, I'm sure they'd feel a lot happier to see you've been around for a little while and are clearly a legit buyer.

Some vendors also demand for first time buyers to finalise early, or only refund 50% if a package goes missing if you have less than 50 previous transactions and things like that.

You could still do it if you wanted to though. Being paranoid isn't necessarily a bad thing.
