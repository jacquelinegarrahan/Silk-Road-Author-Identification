Title: Re: Britcoin and Wallet help needed
Post by: ogre86 on June 19, 2011, 04:11 am
If you're looking at the transaction on blockexplorer, check the number of the block it appears in, then check the current block number in the bottom right of the bitcoin client. When the current block number is 5 more than the transaction's block, that's 6 confirmations (the original block + 5 later blocks).

If you use the bitcoin client to send money, it also shows the confirmation count next to each transaction.
