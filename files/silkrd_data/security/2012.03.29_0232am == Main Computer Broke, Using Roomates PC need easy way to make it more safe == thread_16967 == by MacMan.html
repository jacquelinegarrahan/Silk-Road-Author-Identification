Title: Re: Main Computer Broke, Using Roomates PC need easy way to make it more safe
Post by: cacoethes on March 29, 2012, 06:50 pm
Download Liberte from here:

http://dee.su/liberte

Use the .zip file if using Windows, extract to a USB drive.  Set your BIOS to boot from USB (Probably FDD in BIOS)

Liberte is pre-configured to do in stealth mode exactly what you intend to do.

Liberte creates an encrypted virtual disk on the USB drive.

Liberte will not write anything to the computer on which it's run unless you tell it to.

Liberte comes with PGP and other tools you will find beneficial.

Liberte has a good support forum, and the developer of Liberte will usually respond to questions in short order.

Liberte clears the computer's RAM/reconfigures a new kernel when you log off, or when you jerk the USB drive out.

Liberte, and the drive it's contained on, can easily be hidden.

Liberte, and the drive it's contained on, can be destroyed completely in seconds via a microwave oven (for the truly paranoid).
