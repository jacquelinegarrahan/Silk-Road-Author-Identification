Title: Re: Nessecary to install tails and make a USB?
Post by: fatoldsun on October 20, 2012, 03:25 am
Using a "live" system makes sure that your computer never bears records of your Tor use, account info, etc. Your GPG keys, Tor account info, bitcoin accounts, etc -- all of that never hits disk. It does sit on your USB, but even then it's encrypted before being stored.

You don't have to use a "live" system if you can set up the same level of security on a regular system, but it's all done for you already in TAILS/Liberte.
