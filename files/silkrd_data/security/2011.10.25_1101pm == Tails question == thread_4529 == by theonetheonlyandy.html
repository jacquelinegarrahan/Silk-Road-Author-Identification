Title: Re: Tails question
Post by: phubaiblues on October 25, 2011, 11:14 pm
Yep, no persistence with Tails, and, yeah, the browser anyway on liberte can be a pain...

What I do, is just have an online account for this purpose...like Dropbox, tho I prefer someone less popular: swissdisk comes to mind, wuala is another good one...and then open Tails, and open online storage at same time, download the file, plain text, or html, depending, with your passwords and stuff, and you're good to go...

A handy way I've found, is to use keypass, and make an account, putting all the passwords/usernames/pgp pass phrases/onion url sites...and them export them in an html file ...

All kinds of ways to get around the 'persistence' problem ... another avenue is privatix, which uses a regular FF browser and has persistence...you can google, but like liberte, not a lot of english info.....

good luck, lots of fun studying up on all this....but Paypal does keep you honest with not accidentally storing a bunch of stuff that you don't want around...just store all the junk you need on one file in wuala/swissdisk and it only takes a minute to download it all when u need to get rolling....

I'm working from memory, so I could be wrong (and waiting on the *late* mailman ha ha)  But anyway: on windows, I actually had keepass, not keypassx, and I believe the windows keypass had the option of exporting the file in html mode, so I saved it on desktop, then uploaded and stashed it in swissdisk. 

Keepassx is handy because I can use it on linux and windows, but if I remember right, they don't give u the html option, which is just an easy way to save and use, download to desktop.

The other way, as you've discovered, is just to save as a plaintext doc or something like that, and yeah, upload...there are several ways to do it.  Mostly I just save it in whatever mode keypassx has *.kdb I believe, then just quickly install keypassx on all my machines...if I'm using tails, for example, with no persistence, I either save it as just a text file, or, in my case, since I had it saved html, I download that whenever I open tails..

All of it kind of interesting, kind of a pain...and partly trying both keypass, and then keypassx, then you'll see the difference.  http://keepass.info/index.html   and http://www.keepassx.org/

EDIT: To keep Tails tight, as soon as it boots I open root terminal then run:
apt-get update && apt-get install tor && service tor start

and:

apt-get install keepassx

then go to online storage spot: my favorite for working good in tor, and anonymity is swissdisk.com

I set iceweasal preferences to download to Desktop, then download my keepassx files, and my pgp keys ... and I'm good to go...tried'em all: like tails the best....works good, browser is one I'm used to, and it loads with everything I want except keypassx...
