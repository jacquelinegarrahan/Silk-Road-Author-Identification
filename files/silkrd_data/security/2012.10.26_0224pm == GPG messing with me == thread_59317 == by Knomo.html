Post by: StrangeHands on October 26, 2012, 06:27 pm
Sounds like a glitch, a small mistake in the software that causes a minor problem that can be worked around.

Often this sort of state confusion can be fixed by closing the program and opening it again. If not, then consider filing a bug report with the authors of the software doing your best to explain what you did to make it happen so they can reproduce it.

Understand though that the problem is not with GPG, but rather with the tool that presents you with a fancy menu and uses GPG for you. You may want to look into using the command line version, it is not that hard and is what your tool is using under the hood.
