Title: Re: Transfering bookmarks from TOR?
Post by: sp0reman on October 17, 2012, 09:44 pm
1. click orange torbrowser button at top right of windoe
2. hover over the bookmarks section
3. select "show all bookmarks"
4. in the new window at the top select "import and backup"
5. to export hit the backup button it will save all your bookmarks as a.json file
    B. to import select restore > chose file
