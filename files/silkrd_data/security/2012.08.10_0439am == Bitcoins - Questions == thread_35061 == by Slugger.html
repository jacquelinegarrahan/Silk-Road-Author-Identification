Title: Re: Bitcoins - Questions
Post by: OneOfMany on August 10, 2012, 05:23 am
I used get-bitcoin twice a while ago, and have no complaints. The amounts were fairly small, so I actually just mailed cash. A money order is probably smarter, though again I didn't have any problem. I don't see any reason to go the pre-paid card route. They're great for other kinds of anonymous use online, but since you have to actually mail it anyway in this case, it seems like a waste of money. If you do get a card, make sure it's the non-reloading kind, they won't ask your name or anything. The re-loadable ones are almost like a bank account so they want all kinds of info.

I don't see any glaring flaws in this plan, but I'm not a huge Bitcoin expert so others may have refinements to suggest.
