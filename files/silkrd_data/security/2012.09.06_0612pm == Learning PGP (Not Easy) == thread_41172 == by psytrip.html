The above message was encrypted with only your own PGP key. 

In order to  encrypt messages to other people, you have to add the recipients' keys to your PGP keyring. Think of someone's public key as an envelope that you can put a message into, that only the recipient can open.

In order to do this, you have to save the key to a file, and then use GPA --> Key Manager --> Keys --> Import Key.

You will then be prompted for a file which contains the key to be imported. Click OK, and the key will be imported into your keyring.

When you want to encrypt, click on the Clipboard button in GPA, and either compose or paste-in your message. Click the Encrypt button, and then select the keys to encrypt to. Use Control-Click to highlight (choose) which keys to use, then click OK. You'll see a confirmation dialog asking you if you really want to use these keys; click OK.  Your encrypted message will then be found on the Clipboard, and you can copy and paste it into wherever you want it to go.
