As far as I know, TAILS doesn't have daily downloadable security updates like Ubuntu and the other mainstream distros do.

I think you just have to wait for the next release and download the ISO. Then you can use Applications -> SystemTools -> TailsUSBInstaller to update your stick. New releases come every month-and-a-half or so.

You might need two sticks to do the update, or one CD/DVD and one stick, or whatever, so that you can boot from one while you update the other.
