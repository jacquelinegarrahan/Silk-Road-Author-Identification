For android you can use something like Gibberbot to enable off-the-record IM conversations with other OTR compatible Jabber clients. Best part is it's free.
https://play.google.com/store/apps/details?id=info.guardianproject.otr.app.im&feature=nav_result

Or you can use IM+ Pro, which has support for off-the-record for multiple chats protocols, only problem is it's $5 for the OTR feature.. IM+ Pro is on Android and iPhone, so if you're looking for a cross-platform secure IM, that's a good choice.
https://play.google.com/store/apps/details?id=de.shapeservices.implusfull&feature=search_result#?t=W251bGwsMSwxLDEsImRlLnNoYXBlc2VydmljZXMuaW1wbHVzZnVsbCJd

For added security, if you're using android, I would recommend rooting your phone and installing Orbot, then set it up to transparently proxy the traffic for your IM client through Tor.
https://play.google.com/store/apps/details?id=org.torproject.android&feature=nav_other

If you can't root it, orbot will work directly with gibberbot to proxy traffic for the client to Tor. Good luck.
