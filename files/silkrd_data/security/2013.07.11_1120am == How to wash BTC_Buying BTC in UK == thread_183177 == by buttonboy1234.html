Post by: PrincessButtercup on July 11, 2013, 11:50 am
Unless you buy face-to-face with cash the transaction won't be anonymous - not that this matters unless one is making very large purchases of btc. As for 'washing' them, another option is to move the coins to a Blockchain wallet and then send from there using 'shared send'. This costs around 0.5% as I recall, but makes it close to impossible to trace the origin from the destination address.

Post by: yokes101 on July 12, 2013, 02:17 am
On local bitcoins you are given a four digit code when you place an order and you have to memorize it.
When the seller has sent the coin to you the code is then released to him. So when your seller tells you the code you know your coin has been sent.
No need for a smartphone. :)

Bitcoin fog sends your coin though a load of dummy transactions to make it very hard to track.
