Title: Re: BitInstant (Didn't get my coins)
Post by: scout on February 14, 2013, 10:48 pm
Yeah, when you place two orders with them within a 24 hour period, the second transaction usually gets "stuck" due to their fraud filter.  You need to make sure you're using different amounts, different btc deposit addresses, etc. each time to try and avoid this.  It's because their system sees two very similar transactions for the same amount, or going to the same wallet, and the system thinks the second one is an accidental duplicate.

Also be aware that when this happens, they state that they will refund you their 4% fee as well .... so make sure you ask them to do this.
