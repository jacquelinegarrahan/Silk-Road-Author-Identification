If you have a Windows machine:

Download the DBAN iso image.  This is a file that when you burn it to a disc in a specific way, it un packages the file and will allow you to boot from it.
Link [CLEARNET] http://www.dban.org/download
Burn the image file to disc using ImgBurn [Image to Disc function] or the built in windows burning software.

Download TAILS from [CLEARNET] https://tails.boum.org/download/
There are quite simple instructions on that page
Burn the image file to disc using ImgBurn or the built in windows burning software

Boot your PC with the DBAN disc in the CD drive
Follow the instructions and wipe the drive

Once complete, boot the PC with the TAILS disc in the drive

Insert a USB storage device
There is a menu function within TAILS after it boots to install TAILS to a USB storage device
This will install TAILS onto the USB storage device you have
If you wish, set up persistent storage on the rest of the USB storage device (menu option under the install tails to usb function)

Sorry it is a bit rushed but hope this helps
