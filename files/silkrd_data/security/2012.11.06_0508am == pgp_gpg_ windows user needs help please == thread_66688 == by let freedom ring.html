There are plenty of PGP tutorials and I could tell you to go read those, but I know what it was like feeling clueless, so I will try my best to help. Ok here we go.
1. Download PGP. There are tons of links out there to download it. Just google PGP. Go ahead and install the program and get it up and running.

2. You must obtain the "public key" of the person with whom you wish to exchange encrypted messages. This is almost always in their profile. Click on their name and "copy" the ENTIRE section of text, beginning with "start PGP signed message", and ending with " end PGP signed message".

3. Once you have their "key" copied, open a new text document with word processor, etc. and paste the user's key into the new document. TEMPORARILY save the document to your desktop.

4. Now, using your pgp program, click the button at the top of the screen that says "import". A menu will pop up and allow you to select the source of the file you want to import. Find and select the text document you just created, which contains the user's key. A message may pop up asking if you want to use the key, answer yes. Now a list of keys (called your "keychain") will be created within your PGP program, containing all public keys of users you select.

5. Ok, now click the button that says "clipboard". A blank window will pop up, and you can now type your message. Just type a few generic words(hey, how are you?) for a test message. Once you have done this, click "encrypt" at the top of your PGP program.

6. Another window will pop up(your "keychain" again), prompting you to select for whom you would like the message encrypted. Select the person you are messaging, and be sure to select the appropriate key, as only the owner of the key you select can "read" the encrypted message.

7. Once again, you will be asked if you want to use the key you just selected. Select yes. Now your generic "test" message will transform into a scrambled series of characters. The message is now encrypted, and can only be decrypted by the person you selected from your list of keys(keychain).

8. Highlight and "copy" the ENTIRE "scrambled" message.

9. Now simply "paste" the scrambled message into an SR message, email, or other form of transmitting, and you're done. Once the recipient opens it, they can copy and paste the "scrambled" message into their PGP program, and decrypt it using their keys, and subsequently read the message you sent them. Easy right? Trust me, it will be once you do it a couple times.

PGP simply encrypts and decrypts the text, it does not send/transmit the messages. You use whatever method you want to send it, just encrypt it with PGP first. Hope this helped.
