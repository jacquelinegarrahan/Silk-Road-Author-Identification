You can start by reading the stickied thread http://dkn255hz262ypmii.onion/index.php?topic=131.0

To generate a key in gpg4usb, Keys/Key Management/Key/Generate New.

Select your key, right click, "Append selected key to the text".

Your public key will appear in the editor.
