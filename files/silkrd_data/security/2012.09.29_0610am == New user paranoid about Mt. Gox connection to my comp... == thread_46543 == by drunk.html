Title: Re: New user paranoid about Mt. Gox connection to my comp...
Post by: Rowsdower on September 29, 2012, 11:18 am
I don't know about the legal side of things but I'd get new accounts immediately with different usernames and passwords.  Then hope none of the orders you already made attract any attention in transit.  You definitely compromised yourself more than you should by using the same usernames in both places, also by trying to log into MtGox using TOR, they're known to get suspicious of that.  It will make it a lot easier for law enforcement to build a case against you and convict you if you make it so easy for them.  I don't know if I remember correctly but I think usernames from SR have leaked out before, so it's not unreasonable to think that law enforcement might have lists of them too.  So don't use your real name as your username or use the SR username anywhere else, especially not at MtGox or similar services.
