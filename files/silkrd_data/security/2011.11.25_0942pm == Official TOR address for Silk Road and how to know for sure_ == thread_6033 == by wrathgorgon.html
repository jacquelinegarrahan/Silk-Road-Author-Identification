Memorize it man... I used to just keep it encrypted in PGP on my computer and decrypted it each time I wanted to use it, but that got tiring.

Memorizing it is easier than it seems

break up the phrase

ianxz   6zefk   72ulzz   .onion

Very easy, 3 parts, each rather basic.

First part is like a name, with xz. IANxz.

the next part can be pronounced six-zefk. Fairly straight forward.

Last part is just... 72ulzz. Just remember the sounds of each part.

After about recalling this from memory with this method I had it memorized in about 2-4 tries. Same concept can be applied to any TOR address, just break it up.

This way, it's from your memory, you'll never be phised.
