Title: Re: Liberte linux backing up GPA info please help
Post by: bp on June 13, 2013, 03:04 am
Actually, if you are booting the stick on a computer with Windows on the hard drive you can mount the NtFS parttions using a gui by opening the file manager going to /media.
When you open an NTFS partion it mounts read/write and can be copied to.
Select all of your keys in GPA, or just ones you want, and export to the NTFS partition you mounted using the file manager.
