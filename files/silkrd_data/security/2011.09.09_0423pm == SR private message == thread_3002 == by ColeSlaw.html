Post by: Oracle of Delphi on September 09, 2011, 08:47 pm
Manually encrypting messages is a way to make sure nobody but you and the recipient can read the messages exchanged. Doing so has many advantages, which may come to use lest something else goes wrong. Some users do not encrypt anything and rely on the security of the system as it is. My advice would be to never feel secure enough, and make use of simple tools like encryption. Ask your vendor to put up a public key in their profile.
