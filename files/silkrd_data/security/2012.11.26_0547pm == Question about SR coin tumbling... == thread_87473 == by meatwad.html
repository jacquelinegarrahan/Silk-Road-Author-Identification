Title: Re: Question about SR coin tumbling...
Post by: deep987 on November 26, 2012, 06:20 pm
You don't tumble "transactions". Transactions are simply point A to point B. AKA BTC address A to BTC address B.

Tumbling is done on coins in a wallet. I'm not sure why you would need to perform any of the steps you've described. If anything, you would want to send coins to a non-silk road BTC address or tumbling service (optimal you would have done this before sending coins to SR in the first place) and back again. Use the SR function that gives you a new BTC address after every transaction you make completes.
