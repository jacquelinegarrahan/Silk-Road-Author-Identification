For wifi cracking, look into Reaver.  It's a little temperamental sometimes with the wifi card you use but works well when it does.  It can run from a live cd (or even better, bootable usb).

As long as you take Tor and VPN precautions, using SR at work should be fine.  I get on SR at work daily without worrying because amount of traffic is low.  As Astor said, unless you're running a lot of traffic you should still be fine.

Just don't neglect the encryption either.
