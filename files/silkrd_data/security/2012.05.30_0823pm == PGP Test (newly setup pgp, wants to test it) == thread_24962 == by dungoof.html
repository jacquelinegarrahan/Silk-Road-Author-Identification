Post by: CapnJack on May 31, 2012, 01:51 am
Wrote this for another user and it is for GPG4Win but should help.  Sorry that there are a few extra steps added in but I figured it might help other people reading.

Click start --> All Programs --> GPG4win
The first things that comes up when the program does is "Generate Key Now"
Type either your SR name or a FAKE name on the next page (Needs 2 Words or 1 and a space)
Enter a tormail.org or a FAKE email on the next page.
Create a backup you store somewhere safe on the next page.
Make a hard to crack encryption password on the next page.
And you are done with you key. If you right click they key and select "copy" it will copy your Public key. Post this in your profile.

If someone sends you a key, open notepad, save the key, change .txt in the dropdown box to "all files" and call the key (something).ASC

If you need to use their key, import it by clicking the "Import" Button

To encrypt a message:
Click Clipboard and type your message
Click encrypt and select THEIR key. Copy and paste the results in your message.

To decrypt a message:
Click Clipboard and paste their encrypted message.
Click decrypt and then your password.
