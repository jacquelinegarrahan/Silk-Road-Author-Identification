Title: Re: GPG 4 Windows8-HELP
Post by: Nakorx on October 31, 2012, 02:23 am
What program have you downloaded? Because it sounds like you are trying to use a command line version of GPG, that is why you get the black screen. If you need to run the command line version you must run it from a command prompt. You can get to this by clicking on:  Start - Accessories - Command Prompt.
But perhaps you would be better off using the Windows GUI version called Kleopatra. This can be found in the program called GPG4WIN and you can download for free from:
www.gpg4win.org
Hope this helps.
