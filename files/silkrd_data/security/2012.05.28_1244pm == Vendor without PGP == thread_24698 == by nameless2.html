Title: Re: Vendor without PGP
Post by: UKMJ on May 28, 2012, 01:50 pm
How is it possible? Easy, SR does not make using PGP by vendors a requirement. Your selected vendor may use tormail or rely on the SR PM function to encrypt your messages.

Either way I would be reticent about sending my address in clear text, check with the vendor to see if they have a PGP key and ask them to cancel the order (if that's what you want) if the response is negative.
