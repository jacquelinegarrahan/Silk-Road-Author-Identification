Title: Re: Wiping a computer to use
Post by: Oldtoker on February 12, 2012, 05:42 pm
I would remove the hard disks from the machine and run a Linux System with Security Protocols from a USB Thumb Drive.  Keep the thumb drive in a special hidden place and just stick it in that machine when you want to access the TOR Network.  When done, shut the machine down and hide the thumb drive.  Make sure you digitally secure the thumb drive.  This way if LE for some reason confiscates your computers you still have all your info on the thumb drive.  If they find the thumb drive, if you have properly secured it, they're shit out of luck.

Title: Re: Wiping a computer to use
Post by: Oldtoker on February 12, 2012, 10:17 pm
Quote from: newhorizen on February 12, 2012, 08:35 pm
Thank you both.  Is it "ok" to use ubuntu as the linux op system?

Yeah, all you have to do is download the iso to the hard drive.  Then do an "apt-get -i unetbootin".  That'll install unebootin and use that to install the Ubuntu iso to the thumb drive.  It's really fairly simple.

I would remove the hard drives afterward and let them think that it's an old computer that doesn't work.  They may not walk off with it that way.  But, even it they do you still have the thumb drive.  I use Knoppix and an 8 GB thumb drive.  Works well.  If your really paranoid you can use one of these:  http://www.pcmag.com/article2/0,2817,2381269,00.asp
