Title: Re: The absolute security in Tor net?
Post by: Leapfrogger on November 18, 2012, 06:45 am
Tails ist ziemlich benutzerfreundlich. Mit Liberte hab ich noch keine Erfahrung, aber es ist wahrscheinlich so gut wie Tails.

Truecrypt ist fuer die meisten Benutzer ausreichend, aber mit einem Linux USB Live-Betriebssystem hat man einen grossen Vorteil: man kann sicher sein, dass alle belastenden Daten nur auf einem Geraet liegen. Wenn dieses Geraet verschluesselt ist, was beim Tails ziemlich leicht zu erreichen ist, sind die Behoerde nicht in der Lage, sogenannte "Data-Leaks" zu gewinnen bzw. untersuchen, was beim Windows/Truecrypt wahrscheinlicher ist. Deswegen empfehle ich es dir, Tails oder Liberte auf einem USB-Stick zu installieren.

Ich hoffe, dass meine Nachricht verstaendlich ist! :)
