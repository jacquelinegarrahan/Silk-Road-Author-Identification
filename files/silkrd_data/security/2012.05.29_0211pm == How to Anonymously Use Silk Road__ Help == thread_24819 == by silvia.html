Silvia,
  Obviously you have gotten this far so i'm assuming you used the Tor bundle. That provides some anonyminity (sp?) but not the greatest, as the rest of your PC is still using the clearnet. I would suggest not searching the normal internet when you are using the Tor browser.
To keep you a little more secure I would recommend:
Liberte (USB bootable version of Linux thats been secured)
Tails  (a version of ubuntu linux I believe that adds security and can be loaded on a PC)
VM   (setup a Virtual Machine on a hidden partition on your Hard Drive)

I think the top two are easier to setup, and there is definately a Liberte tutorial on here that I used to get started if you search.  8)
