Post by: OuterLimits on February 06, 2012, 10:02 pm
Are you using the Tor Browser bundle? If so, I have noticed that the browser bundle randomly uses a different port each time you start it, but of course the bundled browser is also configured to the right port for the socks v5 proxy too, except this makes it a bit of a pain because each time you start TOR you have to re-configure any other applications besides the browser to the new port that its using.

Try this:

1. In the Tor browser (aurora) go to the network settings and see which port is listed under the socks v5 proxy, then check to make sure that you have the same port listed in Thunderbird as well.

Hope that helps!
