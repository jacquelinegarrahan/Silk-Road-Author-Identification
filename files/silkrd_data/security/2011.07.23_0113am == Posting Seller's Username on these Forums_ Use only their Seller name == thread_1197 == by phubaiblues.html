Title: Posting Seller's Username on these Forums: Use only their Seller name
Post by: phubaiblues on July 23, 2011, 01:13 am
Sellers that are also buyers, have wisely used different usernames on these forums.  And sometimes I've seen buyers post both names, in an effort to help know 'who is who...'  It's ok for the seller himself to do that, if he wishes, but totally *wrong* in my opinion for us buyers to do that.  Many sellers wish to protect themselves, so if they haven't done it themselves, please don't post both names for the seller, when you start threads on them.  We buyers welcome the threads, but we want to protect them any way we can, and some of them try to use a different name when buying, and selling, and even when posting.  Sometimes it can be confusing, but I try to just use the username they have posted when selling, if I am starting a 'feedback' thread on them.

Often, of course, it's the same, but some of them use one name for forums and buying, and another different name just for selling, and I try, when referring to their sales, to just use the seller name, even if I am reasonably certain I know who is who...
