No, this is absolutely nothing to worry about. This is PGP & GPG's SOP (standard operating procedure).  It has to do with the Web of Trust that PGP/GPG uses. 

If you want an explanation of what the Web of Trust is, please see: http://www.rubin.ch/pgp/weboftrust.en.html
