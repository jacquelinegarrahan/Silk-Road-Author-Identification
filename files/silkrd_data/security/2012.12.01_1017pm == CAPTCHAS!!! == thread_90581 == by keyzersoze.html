Title: Re: CAPTCHAS!!!
Post by: ShardInspector on December 02, 2012, 06:31 am
bottom right hand corner of screen... click on "options" button, choose 'temporarily allow from (same domain as in the address bar)", reload page.

Sometimes the captcha is run by Google and you need to allow '(something)Google.com', I forget what the (something) is but you'll see it there if it is run by Google.

I do not guarantee there would not be any security implications with doing that as there are some pretty obscure attack vectors out there, but off the top of my head I cant think of any right now and in my opinion it should be fairly safe if you only allow the relevant domain temporarily.
