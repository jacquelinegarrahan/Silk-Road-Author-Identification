Title: Re: Time to update TOR again my friends!
Post by: Wadozo on June 27, 2013, 12:55 pm
Also guys, save your bookmarks to use in the latest version of Tor. Click on the BOOKMARK icon (second one in from the RHS top of the page), select SHOW ALL BOOKMARKS and then click on IMPORT AND BACKUP. Then select BACKUP and save the JSON file to your desired location.
Once the updated version of Tor is up and running, click on the BOOKMARK icon again and select SHOW ALL BOOKMARKS. Then select IMPORT AND BACKUP, click on RESTORE and then select CHOOSE A FILE. Double click the JSON file you previously saved and your saved bookmarks will appear as per normal.
