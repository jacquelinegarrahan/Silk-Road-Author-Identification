Title: Re: Deposit not shown up yet on SR
Post by: Variety Jones on December 15, 2011, 07:01 pm
No, not normal at all.

Check Tradehill first and make sure the transaction was even initiated yet.

If it was, get the transaction info and confirm on blockchain explorer that it was a valid transaction.

If it was valid and still hasn't shown up, go to the main site and PM Silk Road Support using the link on the bottom of the page. Include all pertinent info, dates and times and issuing and receiving addresses. When I've had an error happen before, they fixed it within minutes of my PM.
