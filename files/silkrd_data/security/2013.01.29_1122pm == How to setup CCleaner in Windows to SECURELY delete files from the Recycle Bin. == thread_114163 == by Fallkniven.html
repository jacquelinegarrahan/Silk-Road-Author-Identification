Title: How to setup CCleaner in Windows to SECURELY delete files from the Recycle Bin.
Post by: Fallkniven on January 29, 2013, 11:22 pm
Furthering what i was rambling about in another thread about 'How to setup CCleaner in Windows to SECURELY delete files from the Recycle Bin."
 
 
 
Download and install the latest version of CCleaner from - " http://www.piriform.com/ccleaner/download/standard " If you feel like purchasing it, go right ahead, otherwise just get the free version.
 
After installing it, open CCleaner, on the left hand side, make sure the big "Cleaner" tab is selected, on the right-hand list under the "Windows" tab, under "System" sub-heading, place a check mark on "Empty Recycle Bin" and "Temporary Files".
 
Now click the big "Options" tab on the left hand side, click the "Settings" button to the right, place a check mark on  'Add "Run CCleaner" option to Recycle Bin context menu'.  Next, under the 'Secure Deletion' sub-heading select "Secure file deletion". In the drop-down menu under that set it to whatever security level your paranoia desires, i just use 'Simple Overwrite (1 pass)', it is generally the quickest.
 
That's it, close the CCleaner program, and whenever you dump something in the regular Windows Recycle Bin, you can SECURELY erase it using the bins right-click menu and selecting "Run CCleaner". If you use the regular windows "Empty Recycle Bin" option, it WILL NOT securely erase the contents of the bin.
 
Note - The CCleaner icon will popup in the windows system tray and dissapear once it has finished deleting. It canan take a few minutes if you have a lot of files to delete!
