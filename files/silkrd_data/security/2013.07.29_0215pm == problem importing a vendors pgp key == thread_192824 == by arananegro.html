I am unable to import a particular vendors public key.  Any suggestions?  I am using gpg4usb.  When I click to import it says " no keys found to import".  The vendor says others have used it successfully, so I dont understand why it wont work for me?  Any idea how I can get it to work?
Title: Re: problem importing a vendors pgp key
Post by: mcguire39 on July 29, 2013, 03:02 pm
Make sure to copy the entire ASCII armored key including the lines

-----BEGIN PGP PUBLIC KEY BLOCK-----
-----END PGP PUBLIC KEY BLOCK-----

Try deleting the 'version' line such as:
Version: GnuPG v2.0.20 (MingW32)

Near the end, try deleting the carriage return/linefeed between the = and then pressing enter and try again:

GDy3ajk8UMCIRpcT7wdLjSV/Fn4b3sRj2lM+vm02sFnwoqA=
=SugC
