Post by: Torn on November 24, 2011, 08:25 am
The browser itself is craptacular given what we're used to from just about any browser offered on the iphone. I don't think this version of the browser will ever work for an SR site since the browser doesn't appear to support cookies. Which is ok really, since you wouldn't be able to read any of the SR site text because of how it renders pages and zooming doesn't actually result in a better rendering of the page, just a larger version of the same rendered image.

If you're serious about SR from an iPhone, you currently have two choices... well one choice since web-based tor proxies are a bad idea. Search for and install the .deb for the sid77.slackware.it repository (I didn't have any luck entering repo data manually, just download the deb to your phone and use dpkg in a terminal to install it). That repo will let you use Cydia to put tor, a lightweight proxy and a couple other useful goodies onto your phone (like tor-toggle). Once its installed, getting onto the darknet is a two step process that requires a wifi connection:
1) Use the tor-toggle to turn on tor and the proxy on
2) Settings -> WiFi -> (whatever-wifi-you're-on) -> HTTP Proxy -> Manual
     For the proxy, set it to host: 127.0.0.1 port: 8118
Fire up Safari and profit! (Note: you can use other browsers if you want, just not Opera)

I think there may be sneakier ways to do this as well, I'm still digging around out of curiosity, but it will all essentially be some version of the above.

When I do this, I wish I could have the phone in airplane mode + wifi, so I use the SB settings to turn off 3G, Data. Also, I only ever actually access the SR site for emergencies like this (I did it once to finalize an order that would have otherwise sat around for 4-5 days). I have not done an actual security review to ensure the water-tightness of this set up. Things like push notifications might still be in play, I'm not sure.

I wanted to add another step to my earlier recipe for the technologically gifted, it's possible to tweak the phone data connection settings by hand to enable "auto" proxy and point it at your own proxy.pac on your phone. Then it just takes a couple rules in it to correctly channel clearnet and proxy onion traffic based on the URL. Though, this makes onion surfing almost too easy.
