Welcome to SR!

An excellent page for beginners http://www.mccune.cc/PGPpage2.htm

There is no real difference between PGP and GPG for our purposes - GPG is simply a royalty free version of PGP which is unrestricted by patents and can be used in any software.

The easiest version I use is Symantec's Portable PGP (http://ppgp.sourceforge.net/usb.html) - which as the name suggests can be run from a USB stick. I would suggest you encrypt the USB first using a program like Truecrypt.

After reading the guide above you'll probably be familiar with the basic principles and since there are only about five buttons to press with this version of PGP I don't think you'll have any problems generating your own key pair and getting going but any questions please feel free to ask.
