Quote
You'll need at least 20gb.. Whonix gateway + workstation + virtualbox will bring you a little under 20gigs

I successfully installed Whonix on a bootable 8GB USB stick. 16GB should be more than enough, even for future Whonix versions. Unlike my previous Whonix tutorial this USB stick can be used on all computers with modern AMD/Intel CPU and 4GB RAM. Should work on Macs with Intel/AMD too, but there may be a problem with the UEFI crap in the BIOS.

Partitioned the USB stick with EaseUS Partition Master, first partition ~1350MB FAT, second partition the rest (EXT3). Downloaded Xubuntu 12.04 Desktop and created a live system with persistent storage (~500MB) on the first partition by using UNetbootin (this won't work properly with PendriveLinux USB Installer). Then I booted Xubuntu and installed VirtualBox and TrueCrypt, creating a TrueCrypt container which fills the whole 2nd partition of the USB stick. Then I imported the Whonix applianaces to the TrueCrypt container.

It works well, but you should use a quality USB stick with decent speed (e.g. OCZ Rally2). If you only have some cheap CNMemory USB stick, forget about it. There will constantly be errors and the system hangs.
I'll post a step-by-step tutorial in future.
