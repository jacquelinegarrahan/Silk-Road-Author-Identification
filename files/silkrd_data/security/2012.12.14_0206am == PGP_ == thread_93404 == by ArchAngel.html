Are you pasting someones PGP key into the window and asking the software to decrypt it? If so, that won't work, because there's nothing to decrypt. Someone has to encrypt a message using YOUR public key, send it to you, and then you decrypt using your private key. You use OTHER peoples keys to encrypt messages to THEM.
