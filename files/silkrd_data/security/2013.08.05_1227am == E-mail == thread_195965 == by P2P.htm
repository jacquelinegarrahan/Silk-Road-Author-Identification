Bitmessages or I2p-Bote mail. Or Privatdemail.net (no .onion server though, also Egypt server.. can't email Israel or receive emails from Israel)

If you want to be ballin' anon then use a nym server with a newsgroup in the reply block http://security.resist.ca/create-nym.shtml for decentralized anonymous retrieval. Rent a cheap VPS with bitcoins to download alt.anonymous.messages and ssh into it over Tor using passwordless keys for retrieval. Can even run your own tor hidden service to retrieve messages.

If you read the excellent book "SSH Mastery" by Michael Lucas you can automate this
