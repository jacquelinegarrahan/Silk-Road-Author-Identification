Title: Re: Need help with PGP and Encryption
Post by: takashikovacs on March 10, 2012, 01:51 am
When you're in the clipboard, and you have your message typed up, click Encrypt. Select the seller's Public Key, Check the Sign box, and select your key. Click Ok.
(If you don't have your seller's public key listed, you need to get it. Copy-paste it to a new text document, save it, click Import and browse to the text document. That should input the seller's key if it's copied correctly.)

Your seller will need your public key as well to be able to decrypt the message. Export your public key by clicking on your key in the Key Manager, then clicking Export. Open the exported file with notepad and copy-paste it to your seller.
