Hi mate,

Are you trying to create a bootable USB? If so I don't think you can store Liberte on a hidden partition as the bootloader needs to be visible.

Since the entire OS is encrypted anyway you may just want to install it to USB and hope for the best.

Otherwise I suggest you create your hidden partition and then install a Virtual machine image of Liberte onto it (download the "open virtualization bundle" from the Liberte website). Once the virtual machine is set up, move the hard disk image file to your hidden partition. You'll need to point your Virtual Machine software to it of course but it would be a way to keep it hidden.

Really though it seems a lot of trouble you're going to, as Liberte is already encrypted!

V.
