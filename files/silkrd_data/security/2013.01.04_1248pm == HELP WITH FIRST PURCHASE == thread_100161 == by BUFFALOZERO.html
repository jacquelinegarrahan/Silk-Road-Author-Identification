Post by: SorryMario on January 05, 2013, 12:59 am
1. Download GPG4Win from www.gnupg.org

2. Install GPG4Win. As part of this installation you will get the "GNU Privacy Assistant" or "GPA" which is what you will use to encrypt and decrypt PGP messages.

3. On SR, highlight and copy the seller's public PGP block (the whole thing that begins with "------BEGIN PGP PUBLIC KEY BLOCK" and ends with "-----END PGP PUBLIC KEY BLOCK-----")

4. Open Windows Notepad and paste the copied PGP block, then save the text file (save it as any name you want - it doesn't matter).

5. Open GPA and click on "IMPORT", then locate the text file you saved the public key as, select it and hit enter.

Voila! You've just imported their public key! To encrypt something for that person:

1. in the GPA program just click on Windows menu item, then Clipboard. This brings up a clipboard/text editor that you can type or paste your message text into.

2. Once you've got the message you want to encrypt, just press the "ENCRYPT" button, then select the person's public key (it will be listed by the name/email they used to create the key, which will probably be almost identical to their SR vendor name), then press "OK" and click "YES" when it asks you if you really want to use that key.

Now the message in the clipboard is encrypted. Copy the encrypted text (all of it!) and paste it into the email or PM you want to send to that person. Only they will be able to decrypt and read the message.
