Post by: Nakorx on November 21, 2012, 01:20 am
Yes browsing using Liberte is slower than using the Tor Browser Bundle. And using multiple tabs will slow it down even more. You can edit the configuration files but they are already optimized and unless you really know what you are doing you may easily make it worse. Best leave it as it is. You have to remember that there is an awful lot that has been put on Liberte, it has a fully working linux sytem, albeit stripped down but that is the price you are paying for all the anonymity, encrypted volume, a GUI PGP etc and all on a removable USB memory stick.
