Title: Re: Necessary to use USB? Ok to use personal computer?
Post by: Sloan on November 10, 2012, 08:35 am
When you run your computer from a hard drive, various bits of information are stored on that drive, that can be recovered by LE.
When your run a bootable USB, all this information is stored in RAM, instead of the hard drive, which is completely erased when it is powered off.
If your not going to use a bootable USB, at least use Truecrypt to encrypt your system partition with a long password
