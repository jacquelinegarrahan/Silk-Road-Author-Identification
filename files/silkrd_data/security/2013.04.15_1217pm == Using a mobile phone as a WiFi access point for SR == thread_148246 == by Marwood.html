Title: Re: Using a mobile phone as a WiFi access point for SR
Post by: shinzon76 on April 15, 2013, 12:44 pm
I see this potential problem: you'd be using a cell phone, so all the inherent risks that comes with that applies, such as your location being traceable via cell-phone tower triangulation, geolocation, etc. To get around this, you'd need to never, ever let this phone connect to the cell network while: a. home, work, or some other place that you could be easily identified (coffee shop within view of a security camera, etc), and b. your personal cell phone is active, as you don't want the feds to see records of both in the same location at the same time.

That said, this is an interesting idea if you used it to drive out to the middle of nowhere, put the battery in, then turn the phone on, tether, and do your business. Done right, this could allow for a great deal of flexibility in maintaining anonymity.

edit: it'd also have to be, obviously, a phone that has never had been associated with you previously, as each phone that is connected to the network is individually identifiable to the carrier; no using an old phone, it'd have to be a purpose bought burner that couldn't be traced back through purchase records.
