Post by: Joeyjojojr on June 25, 2012, 08:30 am
If you're using a macbook i cant help you, so sorry. If its windows its really easy once you do it once.

Download GPA.

When you first open the program it will prompt you to create a passphrase. ONce you have all that done you just need to copy a vendors public key and paste it in notepad than save it to your desktop.

Click on import. Then the desktop folder, than the file with the users public key. Now you can send that person an encrypted message.

Next is getting your public key which is as simple as clicking your name on the key manager page it should be at the top of the list. Then click export make up a file name and save it to your desktop. Open it and when asked chose notepad to view it.

Now copy and paste that into your clipboard. (clipboard is located next to import and export on the toolbar of gpa). Type your message under your public key in the clipboard (your address, or whatever you want encrypted).

Next click encrypt. Choose the person who you want to be able to read the message. Then click ok.

Copy and paste the encrypted message into the address box on SR or anywhere (message box, forums, etc.) Only the person who the message is encrypted to will be able to read the message.

If you receive a message you simply copy and paste it into your clipboard, click decrypt, and enter your passphrase. Too easy. Good luck.
