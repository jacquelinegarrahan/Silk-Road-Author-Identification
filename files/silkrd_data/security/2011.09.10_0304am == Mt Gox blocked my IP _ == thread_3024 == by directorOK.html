Title: Re: Mt Gox blocked my IP ?
Post by: CaptainSensible on September 10, 2011, 04:31 am
The error message "IP address has been blocked because of too many failed login attempts" is just some sort of default error message -- it doesn't have any bearing on what actually happened.  MtGox is blocking Tor exit nodes, so you can't log in via Tor.  I know, you may have actually created your account via Tor, but you can't get back to your account. 

Accessing MtGox via clearnet is the only way to log back in, that's why many people on SR have stopped using MtGox.

Title: Re: Mt Gox blocked my IP ?
Post by: CrunchyFrog on September 10, 2011, 04:41 am
Mt. Gox has blocked the IP address of the Tor exit relay you're using.  You can try another exit using the Vidalia interface by clicking "Use a new identity".

***BUT***

Before you waste too much time doing that, know that it is widely believed that Mt. Gox blocks *ALL* Tor exit relay addresses.  (These addresses are public knowledge; the Tor Project generates a list of them which is available to all.)

If that is the case, you'll have no option but to connect using your real IP address.  (Yes, this could be a security concern, BTW.)
