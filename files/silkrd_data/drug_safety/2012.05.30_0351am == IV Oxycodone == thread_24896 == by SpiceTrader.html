Title: Re: IV Oxycodone
Post by: Banjo on May 30, 2012, 04:14 am
First of all, don't crush and IV pills, it's bad for you. Secondly, oxycodone doesn't give the same rush that you get from IVing other opiates. It still feels nice, but you're not going to get a huge rush. Oxycodone has a bioavailability of about 86%, which is much higher than most other opiates. So eating the pill will make you feel nearly as good (but not quite) as injecting it.

In my opinion, 40mg IV is way too much for a first time, unless you're experienced with heroin (IV). I'd start off with 10-15mg. Remember, you can always do more after a few minutes, but you can NEVER take back what you've done. Trying for a huge rush/high the first time just isn't worth the risk. Start off small, see how it feels, and then work up from there. Especially since you have two pills, I'd really encourage you to start off with 10mg.

With that said, make sure the OC's are the old formula that can be crushed for instant release. (From what I understand, anyway) the new formula shouldn't be IV'd at all.

Step 1.)
Get some sterile, distilled water. Don't just use water out of your tap. Roughly crush up the pill on something clean like aluminum foil. You don't need to get it super fine.

Step 2.)
Put the crushed pill in about 2ml of the sterile water, and let it set for about an hour. Shake it up (I find using a syringe is handy, if you have some larger ones) so that it mixes up nicely.

Step 3.)
Run the solution through a .2 micron filter. Then run it through a new, clean .2 micron filter again.

Step 4.)
You should now have a clear, or slightly blue tinted clear solution that you can inject.

Please be careful. I don't know how easy wheel filters are to get in Australia, but here in the US I just order them off Amazon, and they're at my door in a couple days. Just using cotton is better than nothing, but IT ISN'T SAFE. I don't crush pills anymore, but I do use heroin IV, and I like to filter that as well, so that's why I keep them on hand. As far as I know, this is about the safest way to inject a pill, if you're set on doing it. Feel free to add to/correct anything I've said.
