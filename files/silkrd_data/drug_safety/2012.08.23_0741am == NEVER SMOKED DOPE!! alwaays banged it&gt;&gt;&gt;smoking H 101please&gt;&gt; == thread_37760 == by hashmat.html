No don't use a 2-litre bottle. That wont work. You gotta make sure you follow the blob with your tooter, you won't miss any if you get it right. It's very hard to explain the proper technique without showing it to you unfortunately.

top tip: make a sort of a ditch in the foil you're about to run the gear on, this will keep it running straight and the walls of the ditch will help the smoke to go up the toot. Looking from the side the foil should look like this:

_________        _________
                   \__/

Run the heroin up and down the ditch, as the blob moves on, keep the lighter *slightly* behind it (underneath ofc) this give the blob heat to keep it moving but it also heats up the blob's trail and this is where the smoke comes from. If it starts to boil or burn, it's too hot. Only run the H up and down the ditch a few times before running it to the side, making a new ditch and going up and down that. If you stay in the one ditch too long it will start to stick and burn.
