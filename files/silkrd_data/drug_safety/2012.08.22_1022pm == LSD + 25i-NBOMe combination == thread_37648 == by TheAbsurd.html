Title: Re: LSD + 25i-NBOMe combination
Post by: NFHC on August 22, 2012, 10:31 pm
25i-NBOMe is highly active on the 5-HTA2 receptors ( hxxps://en.wikipedia.org/wiki/5-HT_receptor ) (apparently more than LSD) which makes it such a quality psychedelic. The problem at high doses is vasoconstriction. 500ug _should_ be safe, it's really higher doses were you really need to start worrying.

Someone on bluelight.ru has probably tried this combination if you'd like to find more comments / experiences
