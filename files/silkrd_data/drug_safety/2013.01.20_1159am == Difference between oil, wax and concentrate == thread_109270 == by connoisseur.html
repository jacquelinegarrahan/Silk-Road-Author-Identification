Title: Re: Difference between oil, wax and concentrate
Post by: Ballzinator on January 23, 2013, 07:38 pm
BHO = Butane Hash Oil
BHO is a type of hash oil that is made via butane extraction.

Hash oils are concentrates that are made via different methods. There is BHO, QWISO, CO2 extracted and probably some lesser-known oils.

Concentrates are pretty much all the products that are not weed. Even regular hash is a type of concentrate.

All of these can be used to treat cancer.
