Title: Re: can you differentiate real LSD from an RC using a reagent test?
Post by: Ben on July 28, 2013, 01:49 am
These quick tests should be taken off the market where they are used as evidence.

Police often treat them as definitive proof some sample contains the active substance the test was developed for. Realistically it is possible to create substances that these tests will show positive for, while the substances in them are not banned nor psychoactive.

The point of doing so would be to demonstrate that the tests used are actually not reliable to the point where a positive test result would be evidence enough to give a search warrant.
