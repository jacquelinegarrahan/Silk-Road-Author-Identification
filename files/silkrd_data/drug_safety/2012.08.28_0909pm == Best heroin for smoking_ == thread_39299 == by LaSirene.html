Title: Re: Best heroin for smoking?
Post by: jackintherox. on August 28, 2012, 11:52 pm
Number 3 is heroin base, used solely for smoking. Number 4 is the salt form, heroin HCL which is extremely inefficient when smoked. I don't think BTH fits into the numbering system since it is made so differently, but it smokes really well. Thus, as lefthandspinner said, if you are in EU you will find #3, or base, but in the USA, you will find BTH. Those are the ones for smoking.
