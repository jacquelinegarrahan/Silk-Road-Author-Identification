Title: Re: NBOMe & MDMA Tolerance question?
Post by: Ballzinator on May 16, 2013, 10:07 pm
NBOMes have extremely high selectivity for 5-HT2A receptors so those are the only receptors that downregulate themselves, causing tolerance. As Naloxone pointed out correctly, MDMA causes a release of serotonin, which then stimulates all 5-HT receptors. If you have a tolerance to NBOMes, there will theoretically be SOME cross-tolerance but I think that should be barely noticeable, if at all. MDMA also causes release of dopamine, which is completely unaffected by NBOMes.

TL;DR
MDMA's dopaminergic component will not be affected.
MDMA's serotonergic component is slightly affected, which will probably not be noticeable though.
