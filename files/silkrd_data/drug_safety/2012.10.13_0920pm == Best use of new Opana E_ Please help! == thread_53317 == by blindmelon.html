at least with the OPs just dissolve the chunks in an acidic beverage for a few hours to release the oxycodone. i assume it works with opana too, but havent tried myself

make sure you start with tiny doses, some even find a few mg of oxymorphone too much with a low tolerance

also just so you are aware, oxymorphone has horrid oral bio-availability - about ten percent if i remember correctly - so dont think because you can take a whole pill orally that you should even snort a 1/4 of the pill. more like an 1/8 to 1/16th to start and work up

there is a drug interaction between alcohol and OM that can easily result in a FATAL overdose as well, so please be careful - never drink on OM because some people absorb much more than usual, and some others absorb even less... no way to know how it will effect you
