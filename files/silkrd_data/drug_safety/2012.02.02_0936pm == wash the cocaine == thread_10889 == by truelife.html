Title: Re: wash the cocaine
Post by: tinytim on February 03, 2012, 03:37 am
This is a more complicated procedure that it first sounds.  I have been studying this in depth recently.

An acetone wash is often used but that will only remove non-polar compounds and since acetone has water in it you will lose a portion of your cocaine while doing it.

Worse, the active cuts (amphetamines, ephedrine) aren't removed by this process.    The way i'm looking to remove these is to rebase, and while in base form filter these out using a solvent which dissolves these but not base cocaine (water), then to reformultate back into the hydrochloride derivitive.

I'm sorry if that sounds like greek, it was to me at first.  I have done a lot of studying and I've had to have some professional tutoring to understand.  Unfortunately it is more complicated than it sounds.
