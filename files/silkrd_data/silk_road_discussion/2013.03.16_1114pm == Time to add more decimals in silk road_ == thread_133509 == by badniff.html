Title: Time to add more decimals in silk road?
Post by: badniff on March 16, 2013, 11:14 pm
The last month the bitcoin has almost doubled in value. If memory serves me right it has more than tripled since the year began! Still, in silk road I can only see two decimals even though bitcoins are infinitely divisible. This means at most that we can deal in centibitcoins. this is enough by most standards, but if the value of bitcoins keep rising, then smaller amounts might become relevant. Adding another decimal, thus dealing in millibitcoins, seems like a necessary upgrade sooner or later.

Vendors might use this to their advantage by milli-adjusting prices to make some extra dollars, or lowering prices just enough to be cheaper to the closest competitor. How it might affect the buyers, I can only begin to speculate.

Honestly, I'd love to see microbitcoins soon, but those are faaar off ;)
