Stealth mode is when a vendor goes into hiding by using that option. Then he won`t appear on SR listing, but his vendor profile will be accessible if you remember the url to it. Vendors usually go into stealth mode to avoid detection or when they have run out of items to sell. The fees that goes to SR is what will be charged while the rest will be returned. It depends on the value set by the vendor. There is a list in the 'state of the road thread' that explains it.
