Vendor prespective;
Order can be cancelled at any time by vendor. When order has been placed in the transit(confirmed shipping), vendor can still cancel it with two options selections(never shipped or never arrived). Trough resolution there is full refund button and ordinary x percentage refund. For all that mentioned, for whatever action took place, same automated sr message is sent with notification about order cancelation e.g. pm does not separate were refund issued or simple cancellation.
