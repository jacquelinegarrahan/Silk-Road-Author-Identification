Post by: Wazup7 on January 14, 2013, 03:17 pm
@wasta This thread was originally from 2011, when the URL was indeed the ian... version, before they got the vanity silkroadvb5piz3r address.  Not sure why this thread was resurrected, referring to an outage in 2011, but the URL wasn't always silkroadvb5piz3r--they changed it to that to help prevent phishing, since I imagine it's difficult for a phishing site to get a similar URL that also has the phrase silkroad in it.

Also, Tor2Web is a horrible idea if you are connecting from an IP that you own or are associated with.  But I sometimes use Tor2Web when I'm on public wifi (not for SR).  The thing is if Silk Road is working on Tor2Web, then it's not down, it's something with your connection.  Try closing and restarting Tor.  If SR were down, no one would be able to access it, not even the Tor2Web proxy.
