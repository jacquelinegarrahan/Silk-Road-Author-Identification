Title: User has no active listings?

My vendors page suddenly has no active listings...He only sold 3 items, each were the same weed but different products but just different weights, does this just mean he ran out?
Title: Re: User has no active listings?
Post by: OldGuard on February 01, 2012, 06:04 pm
Yes and no, a vendor can list a quantity for each listing and when that number is met then the listing disappears. Many vendors list a certain amount and when it is reached they package up and send the orders then relist the same items again if they have more. Sometimes it means they are sold out other times it can be used to control the number of orders coming in and being shipped each day.
