Title: Re: Need help encypting address - finalising order. (GPG)
Post by: sdesu on July 03, 2012, 08:59 pm
Save the vendor's key into your keychain.
Type your info into textedit (notepad).
Highlight all text and right click > Services > Encrypt Selection.
From there, OpenPGP should pop up and ask what key you want to use to encrypt. Select the vendor's key and hit ok.
