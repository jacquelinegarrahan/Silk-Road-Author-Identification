Title: Re: Weighting DMT
Post by: Asylum on July 10, 2012, 02:16 pm
As two posters above said, use scales - some can be obtained pretty cheaply.

If you've already smoked DMT before, you might be able to eyeball the required amount pretty easily, it actually really equals to a tip of a pencil eraser.

Stay safe, an OD on DMT can be quite overwhelming and scary if not dangerous to your body, so if unsure, rather use less than more.
