First, before you post a new thread, make sure you are posting in the right section of the forum. Silk Road discussion is strictly for discussion pertaining to the Road itself. Not, for instance, discussion of vendors on the Road (that's what the Rumor Mill is for). Everything else has a small description underneath to help you find where you need to post.

Next, before you post that thread, make sure there isn't already a thread for the same topic. You can check by looking through the first few pages of threads in the section of the forum you want to post in, or you can use the search tool at the upper-right hand side of the page.


Please report spam to a moderator using the "Report to moderator" link on the spam post, with a clear explanation of why you reported it, so it can be taken care of more efficiently.
