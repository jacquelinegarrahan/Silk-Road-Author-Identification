Interesting notion but are you referring to a Key System here or buyers manually signing their reviews with a PGP Key? I'm assuming you're referring to the first option as there's no way buyers would take the time to sign with PGP keys - anyway so I'm just thinking openly here about how that could possibly work:

* If the feedback is stored on a third party site how long would it be valid and could it be accessed by anyone?
* Certain parameters could be added to the Feedback which would only make it possible for the Vendor the feedback was intended for to use it on their profile.
* The Feedback System could work both ways for a way for Vendors to quickly warn other Venders of troublesome Customers.
* If a key is used to unlock the Feedback some system would have to be in place so two Vendors or more can't utilize the "same" Feedback at the same time.
* As some Sites have different rules the "Type of Transaction" could be included in the Feedback, i.e conventional/escrow, FE, Lottery and so on.

Sure it could be refined significantly with some thought but definitely interesting idea. :)
