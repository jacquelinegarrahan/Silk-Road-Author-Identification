Quote from: Nuggets on December 03, 2012, 06:24 am

BTW why doesn't the vendor need my public key when I encrypt my address in the allotted address space?

You can read lots on public key cryptography. But the short answer is that your keypair is not used to encrypt the message to the vendor. In fact you don't even need a key. The public key from the vendor encrypts the message which then can only be decoded with the vendors private key. So he does not need your public key unless he is going to be sending an encrypted message back which is usually not the case.
