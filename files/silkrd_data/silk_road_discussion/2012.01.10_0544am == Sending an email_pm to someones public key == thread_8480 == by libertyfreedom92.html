Title: Re: Sending an email/pm to someones public key
Post by: freddurst1000 on January 10, 2012, 09:45 am
Their public key isn't an address, it's like a password that you encrypt the message with, before you send it by other means (email, SR messaging system). If you're using windows get a program called gpg4win, and use GPA instead of Kleopatra when it asks during install, it's a lot easier to work with for beginners.

You'll need to import vendor public keys into GPA, first copy+paste them into notepad or something, then save them as "filename.asc". Then click import in GPA and locate the file you just made. Open up the clipboard in GPA then type your message, press encrypt at the top then choose the person you're sending the message too. You'll get something that says -----BEGIN PGP MESSAGE-----

Copy absolutely everything and send that on to the vendor. They'll be able to read it.
