i wouldnt recommend dealing with BP, he just scammed a member

that begin..end thing is called the vendors public key. you need to read more in the security forum like i said - it will answer your questions, if you dont understand what public and private keys are you are far from being able to use a pgp program.

id recommend you look into gpg4usb which is an easier program to use -you can get it (and a tutorial) from this site: gpg4usb.cpunk.de

just unzip it onto your bootable usb (look into tails or liberte) - no i cant explain it all to you, read the forum

then you import the vendors public key (copy the entire public key and paste it in the text box then click import) and then you type your addy, select the key and encrypt using it - can be done in as little as ten minutes once you understand the concept of encryption
