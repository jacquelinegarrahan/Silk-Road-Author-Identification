Post by: Spunkaroo on January 08, 2013, 12:34 pm
Oddly enough I've been doing a lot of research into drug testing recently.

My understanding is that saliva tests detect THC in your saliva (duh...) but urine tests detect THC metabolites. What this means is saliva tests will pick up recent usage, so a few hours to a couple of days (at most) since using. Urine tests checking for the metabolites pick up usage over a longer term as the THC that is stored in your body fat takes longer to be metabolised and then excreted.

Point is, urine test will detect longer term usage of THC, but saliva tests won't. I've read anywhere from 5 to 48 hours before a saliva test will come up clean, however this is still much shorter than the weeks or months of a urine test, so a good thing in detecting actually impaired drivers (to use one example.)

Just as a disclaimer, I am from Australia where we have roadside saliva tests already and it's these I've been looking into mostly. And in case it helps fellow aussies out, I've been breath tested more times than I can remember but not once have I been subjected to a drug test on the roads, so I don't think they're as common as the traffic ads would have you believe. This is only my experience though, and maybe it's not typical.
