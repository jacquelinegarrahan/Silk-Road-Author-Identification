Title: external programs...
Post by: loginathome on July 29, 2011, 05:50 am
It really depends on what program you are using to display the file.

If its a simple text file, then you are probably fine...  Using an external program to show Tor files isnt necessarily bad.  You just have to know what type of file it is, and what risks that entails.  NEVER open PDF files in this manner, they can 'phone home' using your normal connection and tell whomever that this ip downloaded this file on this site via tor.

Thats a BIG risk, Just be careful and if you don't know what you are doing, then dont do it. 
If you are like me and you THINK you know what you are doing, its still best to not do it.
