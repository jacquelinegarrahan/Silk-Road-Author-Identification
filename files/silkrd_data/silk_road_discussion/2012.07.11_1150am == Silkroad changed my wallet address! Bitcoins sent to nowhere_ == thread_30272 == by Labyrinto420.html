Title: Re: Silkroad changed my wallet address! Bitcoins sent to nowhere?
Post by: thesearstower on July 11, 2012, 06:30 pm
Bitcoin works by writing all of the transactions into the block chain. To avoid double spending, the standard is to wait for 6 blocks to get written to make sure that the transaction is valid and the coins have only been spent once.  SR waits for these 6 confirmations. How quickly blocks are written depends on a lot of things. Blocks that include a transaction fee (some fraction of a BTC, usually around .01) are picked up quicker than those without, and get written faster. If you're using a bitcoin client, try including a transaction fee to have your block picked up faster. I'm not sure about the transaction fees with online wallets. It usually takes about an hour for my transactions to get picked up, and i include a fee of .001BTC

tl;dr SR waits from BTC transaction confirmation from the network. Include a transaction fee to speed things up.
