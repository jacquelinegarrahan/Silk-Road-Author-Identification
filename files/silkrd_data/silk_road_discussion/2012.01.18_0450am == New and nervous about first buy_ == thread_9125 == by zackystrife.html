Title: Re: New and nervous about first buy?
Post by: CaptainSensible on January 20, 2012, 11:40 pm
And don't forget how you connect to Silk Road.  You can use the Tor bundle for Windows, Mac, or Linux, but a safer way is to use Tails or Liberte Linux from a USB drive.  It's not that hard to do. 

You'll also need to learn how to encrypt files.  Once you create a PGP key you'll have two parts to it: a public key which you send to others so they can use it to send you an encrypted message only you can read, and a private key which is used to decrypt these messages. NEVER give out your private key, and keep it stored in an encrypted file. 

Finally, select a very long and complex password for your PGP key, and a another long and complex password for the file you've encrypted.  All this will help you stay a little safer.
