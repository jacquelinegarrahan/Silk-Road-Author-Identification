Post by: MarijuanaIsMyMuse on November 16, 2012, 05:35 pm
Juicy

You're probably asleep by now. If you haven't yet marked the order in transit, you can safely cancel it at no loss to yourself. If you use the hedging service, the buyer will get back 2-4% less due to hedging fees. If you have already marked the order in transit, but have not yet shipped (you shouldn't be doing this), you can cancel the order and specify never sent and again the buyer will dinged a 2-4% hedging fee.

Also, if you have not yet marked the order in transit and cancel it, the buyer will not be able to leave feedback. If you have marked it in transit and then cancel it, the buyer will be able to leave feedback. Be careful, I have had buyers in the past leave a 1/5 for feedback on cancelled orders.

MJMuse
