Post by: jameslink2 on September 15, 2012, 09:21 pm
Once you have created your key and imported the vendors public key you open the clipboard.

In there you type your address and click the encrypt button.

You will be presented with a menu of keys that it can use to encrypt the message. You want to select the vendors key from the list and click "OK"

It will then display the encrypted message. This is your address encrypted to the vendors key.

Copy and paste that into the message box when ordering from the vendor. s/he will be the only one that can view your address.

If you want to test it, grab my key from my vendor page and encrypt a message to it. Then send me the message on SR. Ill let you know if I can read the message.
