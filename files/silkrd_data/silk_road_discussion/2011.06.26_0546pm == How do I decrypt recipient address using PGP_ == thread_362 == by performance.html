Post by: dance4life on June 26, 2011, 07:39 pm
You should get GPG4Win.  Is that what you have, also says Kleopatra?

If this is the case, find the Kleopatra icon ( while the program is running ) located where your clock is for windows ( usually bottom right ) - > right click that icon -> clipboard -> then choose your operations

You need to most likely first import their public key.

Once you have done this you need to decrypt their message by doing

clipboard -> decrypt

If you want to write a message YOU MUST SELECT BOTH KEYS WHEN MAKING THE MESSAGE.  YOURS AND THE OTHER PERSONS ( BY ADD RECIPIENT -> OTHER CERTIFICATES ( FAR RIGHT TAB ) )

Hope that makes sense
