Post by: Duffman on April 22, 2013, 10:25 am
Litecoin creates its blocks faster, it takes around 2.5min instead of the 10min that it takes for Bitcoin.
Which doesn't really matter for the speed of transactions as you just need to wait for more confirmations instead.
They used a different hashing algorithm so that it would only be feasible to mine with a CPU and to decentralize the mining.
However this has already been proven incorrect as you can already use a GPU for mining and apparently ASICs would be a possibility as well.
There will also be more litecoins in circulation than bitcoins, 84 million instead of the 21 million bitcoins.
I don't really see the value in it, it's just Bitcoin with some tweaks which don't seem to make much sense.
If you still want to buy litecoins though, then you can do so at https://www.btc-e.com.
