Title: Re: bit instant only one transaction showed in acct
Post by: scout on February 14, 2013, 11:01 pm
It's because you used the same amount twice -- the system sees your second transaction as an accidental duplicate, or a fraudulent one, so you'll need to contact them to have them manually push it.  In the future, make sure you're using two different amounts, and two different bitcoin deposit addresses -- otherwise, this will happen again.
