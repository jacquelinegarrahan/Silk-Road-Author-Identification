Post by: wowzers on January 15, 2012, 09:58 pm
1. Use tor, if you are more paranoid, run it on a virtual machine and through a proxy.
2. Buy bitcoins through an exchange, mtgox, intersango etc. Store them in a bitcoin wallet or just in your SR wallet if you don't have too many of them.
3. Yes, use SR PM but you can just add your item to cart and check out. We don't use hotmail or gmail, we never use paypal. If you are security conscious then use PGP for PMs. There are free PGP programs. Practice using PGP by sending messages to yourself before doing it IRL.
3'. Buy Bitcoins from exchange > wallet > SR. Buy item. Have item delivered.

(4). Read as much as you can about Bitcoin, PGP, VM's etc before you take the plunge. Buy a few cheapo items first like a couple of ebooks to get comfortable using the interface. Newbie questions can sometimes sound a bit LE'esque.

Hi iheartpills, and welcome to the Silk Road.

In the future, it's far more efficient to ask ONE question in a thread, and put that as best you can in the title - 'please read' doesn't tell anyone what to expect. 'How do I find a PGP program that doesn't cost $100' would be a great title, and limit the thread to that one subject.
