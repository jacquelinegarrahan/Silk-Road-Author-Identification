Title: Script to improve the functionality of browsing SilkRoad
Post by: fl0gg3r on August 27, 2012, 12:48 pm
Hey guys.
I've whipped up a small scriptish(greasemonkey) userscript to improve the functionality of browsing SilkRoad.
Made it for myself, but maybe someone will find it useful.
Features:
1. Filter items by target country (i.e. it will show only items that are sent to specific country AND worldwide. domestic filter on SR works differently..)
2. Show USD value for all BTC prices (in main screen, category browse and search results). You can change the URL to MTGOX api and pick an currency.
3. Filter search results by category

------- Update - 29/8/2012, Version 1.1 ------
I've uploaded a new version, you can get it at http://xqz3u5drneuzhaeo.onion/users/qicpic/files/n.js , just save that file instead of the old srgrease.user.js
Changes: Easier support for EUR and GBP, Sorting of category filter, minor fixes
I've also added some comments to help you configure the script:
Code: [Select]
//****************************************
//The values below can be used to configure the script
//****************************************
var selectedCurrency = "USD"; // you change this to one of the supported currencies
var country = "Israel"; // your country - will be used for filtering out items that don't ship to that country
var filterCountry = true; // enable filtering search results to removed items that don't ship to your country
var enablefilter = true; // enable adding a set of checkboxes above the search to filter by category
//****************************************
-----------------------------------------------------

Instructions:
0. Get the ZIP from: http://xqz3u5drneuzhaeo.onion/users/qicpic/files/q.zip
1. Unzip the zip into your firefox profile directory (if in torbrowser: FirefoxPortable/data/profile)
2. Edit the srgrease.user.js to match your contry and optionally disable features
3. Restart the browser
4. Enjoy!

Have fun and let me know if there are questions/suggestions.
