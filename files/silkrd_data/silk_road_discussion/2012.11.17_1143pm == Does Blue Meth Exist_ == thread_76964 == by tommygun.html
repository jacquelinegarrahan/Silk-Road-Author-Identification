Title: Re: Does Blue Meth Exist?
Post by: cacoethes on November 18, 2012, 01:52 am
It CAN exist, but in Breaking Bad it's a plot device, especially as it's used within the context of Walt's ultra-pure crystal.

Methamphetamine HCl is an amine hydrochloride salt, and should be colorless, odorless crystal.  Any coloration is an impurity, and a chemist of Walt's caliber would never have allowed it in the final product.  Using methylamine does not yield blue meth, but in BB there is a need to identify Walt's product from all others, so it is allowed that a methylamine cook will yield the infamous "Blue Sky Meth".
