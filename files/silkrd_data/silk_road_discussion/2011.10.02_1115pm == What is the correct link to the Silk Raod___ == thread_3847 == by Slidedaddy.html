as MrM said thats the one.

The warning is because a number of people have discovered how easy it is to edit the wiki and set up alternative (presumably hosted by FH) login screens to gather sr user credentials :)

But some simple tests you can apply to say safe: reload the page does the capcha change? many scammers use a fixed jpg.

Also, check the register link, ive not seen a scammer change this yet so they still point to the original sr address (which should be what is displayed in your browser address bar).

Some other indicators, sometimes the css is not copied so the bolden'ed warning text at the top of the page looks a little odd, and for the little more technically minded take a look at the html for clues, i.e. the post address being /whatever.php.
