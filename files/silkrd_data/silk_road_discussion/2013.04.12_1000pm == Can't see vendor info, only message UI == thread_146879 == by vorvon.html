Title: Re: Can't see vendor info, only message UI
Post by: grahamgreene on April 13, 2013, 06:58 pm
Quote from: nitpi950 on April 13, 2013, 04:54 pm
I've definitely seen it for reputable vendors who just have nothing to sell. Granted I'm not a vendor but I'm quite positive on this. Maybe things have changed

This isn't possible as the Silk Road user link is static; when you purchase vending privileges the page at your static link is 'unlocked' for lack of a better word to enable a different template which will ALWAYS have your username, vendor rankings and 'report this vendor' link on it regardless of whether you're making any sales or not. If a vendor has nothing to sell, their profile will show the 'no listings' notice; however, the actual page will always stay the same.

The ONLY way to have a page which is just a user message interface is if you no longer have vending privileges and your account has been demoted to a regular user account.
