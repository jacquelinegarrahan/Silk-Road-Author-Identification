Title: Re: First time buyer/experience, need questions answered :)
Post by: PlutoPete on July 01, 2012, 11:08 pm
Signing for a package doesn't mean you know what's inside, but some people don't like it anyway.
In the UK the likely scenario of having 5g cubensis intercepted by customs is that it would be destroyed and you probably won't get a letter because they usually save them for taxable contraband or larger amounts. It normally takes a few interceptions before you get a knock on the door and a stern talking to.
The security thread has a good tutorial.
Delivery times vary a lot, 3-5 working days is normal for EU to UK, but remember that some vendors mark In Transit before actually shipping so allow a bit longer.
