Title: Why the new forum is good
Post by: envious on June 18, 2011, 10:35 pm
1. Distributes load off the market server. I'm sure everyone noticed how slow the market server has been lately.
2. If this server is hacked (which so many are worrying about) the worst that can happen is they will be able to post under your name. Oh noes.
3. No hidden service is logging IPs as everyone is worried about. Your IP shows up as 127.0.0.1 which is local loopback.
4. If market server goes down, we aren't left out in the dark.
5. The forum software is open source. This is a good thing because anyone can review the code for holes, therefore it is less likely to have them.
6. The forum software has the advanced features that were needed.
7. Developers can now focus on improving the marketplace instead of reinventing the wheel (forum).
